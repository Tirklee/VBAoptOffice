DisplayUnitLabel 对象成员

代表指定图表中坐标轴上的单位标签。

方法

 名称说明
Delete删除对象。
Select选择对象。

属性

 名称说明
Application如果不与对象识别符一起使用，则该属性返回一个代表 Microsoft Word 应用程序的 Application 对象。如果与对象识别符一起使用，则该属性返回代表指定对象的创建者的 Application 对象（可对自动操作对象使用该属性来返回该对象的应用程序）。只读。
Caption返回或设置显示单位标签文本。可读/写 String 类型。
Characters返回一个 ChartCharacters 对象，该对象代表对象文本中的一系列字符。使用 ChartCharacters 对象可为文本字符串中的字符设置格式。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Format返回对象的线条、填充和效果格式。只读 ChartFormat。
Formula获取或设置一个 String 值，该值以英语使用 A1 样式表示法来表示对象的公式。可读/写。
FormulaLocal返回或设置对象的公式，使用用户语言 A1 格式引用。可读/写 String 类型。
FormulaR1C1返回或设置对象的公式，使用宏语言 R1C1 格式符号表示。可读/写 String 类型。
FormulaR1C1Local返回或设置对象的公式，使用用户语言 R1C1 格式符号表示。可读/写 String 类型。
Height返回对象的高度（以磅为单位）。只读。
HorizontalAlignment返回或设置指定对象的水平对齐方式。可读/写 Variant 类型。
IncludeInLayout如果在确定图表布局时显示单位标签将占用图表布局空间，则为 True。默认值为 True。可读/写 Boolean 类型。
Left以磅为单位返回或设置对象左边缘至图表区左边缘的距离。可读/写 Double 类型。
Name返回对象的名称。只读 String 类型。
Orientation返回或设置文本方向。可读/写 Long 类型。
Parent返回指定对象的父对象。只读 Object 对象。
Position返回或设置单位标签在图表坐标轴上的位置。可读/写 XlChartElementPosition。
ReadingOrder返回或设置一个 XlReadingOrder 常量，该常量代表指定对象的阅读顺序。可读写 Long 类型。
Shadow返回或设置一个表示对象是否有阴影的值。可读/写 Boolean 类型。
Text返回或设置指定对象中的文本。可读/写 String 类型。
Top返回或设置一个以磅为单位的距离，它代表从对象的上边缘到第一行顶部（在工作表上）或图表区顶部（在图表上）的距离。可读/写 Double 类型。
VerticalAlignment返回或设置指定对象的垂直对齐方式。可读/写 Variant 类型。
Width返回对象的宽度（以磅为单位）。只读。
