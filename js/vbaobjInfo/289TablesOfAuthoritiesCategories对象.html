TablesOfAuthoritiesCategories 对象成员

TableOfAuthoritiesCategory 对象的集合，这些对象代表引文目录类别，如“事例”和“法规”。TablesOfAuthoritiesCategories 集合包括“索引和目录”对话框的“引文目录”选项卡上的“类别”框中列出的所有类别（共 16 项）。

方法

 名称说明
Item返回集合中的单个 TablesOfAuthoritiesCategory 对象。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Count返回一个 Long 类型的值，该值代表集合中类别的数目。只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Parent返回一个 Object 类型的值，该值代表指定 TablesOfAuthoritiesCategories 对象的父对象。
