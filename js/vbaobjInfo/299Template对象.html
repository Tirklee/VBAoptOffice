Template 对象成员

代表文档模板。Template 对象是 Templates 集合的一个成员。Templates 集合包含所有可用的 Template 对象。

方法

 名称说明
OpenAsDocument将指定模板作为文档打开并返回一个 Document 对象。
Save保存指定的模板。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
BuildingBlockEntries返回一个 BuildingBlockEntries 集合，该集合代表模板中构建基块条目的集合。只读。
BuildingBlockTypes返回一个 BuildingBlockTypes 集合，该集合代表模板中包含的构建基块类型的集合。只读。
BuiltInDocumentProperties返回一个 DocumentProperties 集合，该集合代表指定文档的所有内置的文档属性。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
CustomDocumentProperties返回一个 DocumentProperties 集合，该集合代表指定文档的所有自定义文档属性。
FarEastLineBreakLanguage返回或设置在指定的文档或模板中对文本进行换行时使用的东亚语言。WdFarEastLineBreakLanguageID 类型，可读写。
FarEastLineBreakLevel返回或设置指定文档的换行控制级别。WdFarEastLineBreakLevel 类型，可读写。
FullName指定模板的名称，其中包含驱动器或 Web 路径。String 类型，只读。
JustificationMode返回或设置指定模板的字符间距调整。可读写 WdJustificationMode 类型。
KerningByAlgorithm 如果 Microsoft Word 调整指定文档中的半角西文字符与标点符号的字距，则该属性值为 True。可读写 Boolean 类型。
LanguageID返回或设置一个 WdLanguageID 常量，该常量代表指定范围的语言。可读写。
LanguageIDFarEast为指定的对象返回或设置东亚语言。WdLanguageID 类型，可读写。
ListTemplates返回一个 ListTemplates 集合，该集合代表指定文档、模板或列表库的所有列表格式。只读。
Name返回指定对象的名称。只读 String 类型。
NoLineBreakAfter返回或设置 Microsoft Word 不在其后面进行换行的首尾字符。可读写 String 类型。
NoLineBreakBefore返回或设置 Microsoft Word 不在其前面进行换行的首尾字符。可读写 String 类型。
NoProofing 如果拼写和语法检查程序忽略基于此模板的文档，则该属性值为 True。可读写 Long 类型。
Parent返回一个 Object 类型的值，该值代表指定 Template 对象的父对象。
Path返回指定文档模板的路径。只读 String 类型。
Saved 如果自上次保存之后指定的模板未更改，则该属性值为 True。如果关闭文档时 Microsoft Word 提示保存对文档所做的更改，则该属性值为 False。Boolean 类型，可读写。
Type返回模板的类型。只读 WdTemplateType 类型。
VBProject返回指定模板的 VBProject 对象。
