Trendline 对象成员

代表图表上的趋势线。

方法

 名称说明
ClearFormats清除对象的格式设置。
Delete删除对象。
Select选择对象。

属性

 名称说明
Application如果不与对象识别符一起使用，则该属性返回一个代表 Microsoft Word 应用程序的 Application 对象。如果与对象识别符一起使用，则该属性返回代表指定对象的创建者的 Application 对象（可对自动操作对象使用该属性来返回该对象的应用程序）。只读。
Backward2返回或设置趋势线向后延伸的周期数目（或散点图中的单位个数）。可读/写 Double 类型。
Border返回对象的边框。只读 ChartBorder 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
DataLabel返回与趋势线相关联的数据标签。只读 DataLabel 对象。
DisplayEquation 如果图表中显示趋势线公式（与 R 平方值位于同一数据标签中），则为 True。可读/写 Boolean 类型。
DisplayRSquared 如果图表上显示趋势线的 R 平方值，则为 True。可读/写 Boolean 类型。
Format返回对象的线条、填充和效果格式。只读 ChartFormat。
Forward2返回或设置趋势线向前延伸的周期数目（或散点图中的单位个数）。可读/写 Double 类型。
Index返回对象在其同类对象所组成的集合内的索引号。只读 Long 类型。
Intercept返回或设置趋势线与数值轴的交叉点。可读/写 Double 类型。
InterceptIsAuto 如果趋势线与数值轴的交叉点由回归分析自动确定，则该属性值为 True。可读/写 Boolean 类型。
Name返回或设置对象的名称。可读/写 String 类型。
NameIsAuto 如果 Microsoft Word 自动确定趋势线的名称，则为 True。可读/写 Boolean 类型。
Order当趋势线类型为 xlPolynomial 时，返回或设置趋势线的顺序号（大于 1 的整数）。可读/写 Long 类型。
Parent返回指定对象的父对象。只读 Object 对象。
Period返回或设置移动平均趋势线的周期。可读/写 Long 类型。
Type返回或设置趋势线类型。可读/写 XlTrendlineType。
