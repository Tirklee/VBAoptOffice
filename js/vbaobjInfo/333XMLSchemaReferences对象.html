XMLSchemaReferences 对象成员

XMLSchemaReference 对象的集合，这些对象代表附加到文档的唯一命名空间。

方法

 名称说明
Add返回一个 XMLSchemaReference 对象，该对象代表应用于文档的架构。
Item返回集合中的单个 XMLSchemaReference 对象。
Validate 验证所有附加到文档的 XML 架构。

属性

 名称说明
AllowSaveAsXMLWithoutValidation返回一个 Boolean 类型的值，该值代表 Microsoft Word 保存文档之前是否验证文档中的 XML。可读写。
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
AutomaticValidation返回一个 Boolean 类型的值，该值代表 Microsoft Word 在用户键入时是否验证文档中的 XML。可读写。
Count返回一个 Long 类型的值，该值代表集合中 XML 架构的数量。只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
HideValidationErrors返回一个 Boolean 类型的值，该值指示 Word 是否显示当前 XML 文档的架构验证错误。可读写。
IgnoreMixedContent返回一个 Boolean 类型的值，该值代表 Microsoft Word 在具有同类元素的文本节点上是否进行验证，并指定当 XMLSaveDataOnly 属性值为 True 时，是否将这些文本节点保存为 XML。可读写。
Parent返回一个 Object 类型的值，该值代表指定 XMLSchemaReferences 对象的父对象。
ShowPlaceholderText返回一个 Boolean 类型的值，该值代表在文档中是否显示 XML 元素的自动占位符文本。可读写。
