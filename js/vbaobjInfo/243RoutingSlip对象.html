RoutingSlip 对象成员

代表文档的传送名单。可用传送名单通过电子邮件系统发送文档。

方法

 名称说明
AddRecipient向指定的传送名单添加收件人姓名。
Reset重新设置传送清单，以便新的传送能够使用同样的收件人清单和传送信息启动。完成该传送过程后才能使用该方法。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Delivery返回或设置传递文档时的传送方式。WdRoutingSlipDelivery 类型，可读写。在传送开始以前为 Long 类型，可读写。；在进行传送时为 Long 类型，只读。
Message返回或设置指定传送名单的邮件文本。String 类型，可读写。
Parent返回一个 Object 类型的值，该值代表指定 RoutingSlip 对象的父对象。
Protect返回或设置与指定传送名单相关联的文档的保护类型。WdProtectionType 类型，可读写。
Recipients返回指定传送名单中某位收件人的姓名。只读 Variant 类型。
ReturnWhenDone 如果为 True，则当传送结束时，将指定传送名单的相关文档传送给原来的发件人。在传送之前是 Boolean 类型，可读写。在传送过程中，是 Boolean 类型，只读。
Status返回指定传送名单的传送状态。WdRoutingSlipStatus，只读。
Subject返回或设置用于传送文档的邮件的主题文本。可读写 String 类型。
TrackStatus 如果为 True，则每次转发传送的文档时，都向原来的发件人返回一条邮件。传送前该属性为 Boolean 类型，可读写；传送过程中，为 Boolean 类型，只读。
