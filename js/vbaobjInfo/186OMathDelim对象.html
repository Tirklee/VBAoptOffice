OMathDelim 对象成员

代表定界符对象，包括左右定界符（如括号、大括号、中括号或竖线）以及定界符中所包含的一个或多个元素。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
BegChar返回或设置一个 Integer 类型的值，该值代表定界符对象中的定界符开始字符。可读写。
Creator返回一个 32 位的整数，该整数表示在其中创建加载项的应用程序。只读 Long 类型。
E返回一个 OMathArgs 对象，该对象代表指定公式对象的参数列表。只读。
EndChar返回或设置一个 Integer 类型的值，该值代表定界符对象中的定界符结束字符。可读写。
Grow返回或设置一个 Boolean 类型的值，该值代表定界符字符是否增长到所包含参数的整体高度。可读写。
NoLeftChar返回或设置一个 Boolean 类型的值，该值代表是否隐藏定界符对象中的左定界符。可读写。
NoRightChar返回或设置一个 Boolean 类型的值，该值代表是否隐藏定界符对象中的右定界符。可读写。
Parent返回一个 Object 类型的值，该值代表指定 OMathDelim 对象的父对象。
SepChar返回或设置一个 Integer 类型的值，该值代表当定界符对象中包含两个或更多参数时，定界符对象中的分隔符字符。可读写。
Shape返回或设置一个 WdOMathShapeType 常量，该常量代表与其中内容有关的定界符（例如括号、大括号和方括号）的外观。可读写。
