CalloutFormat 对象成员

包含应用于线形标注的属性和方法。

方法

 名称说明
CustomDrop设置文本范围限制框边到文本框标注线的垂直距离（磅值）。
CustomLength指明在移动标注时标注线的第一条线段（即附加于文字标注框的线段）保持固定长度。
PresetDrop指定是将标注线条连接到标注文本框的顶部、底部或中心，还是将它连接到距文本框的顶部或底部为指定距离的某个点上。

属性

 名称说明
Accent 如果垂直强调线分开标注文字和标注线，则该属性值为 True。MsoTriState 类型，可读写。
Angle返回或设置标注线角度。MsoCalloutAngleType 类型，可读写。
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
AutoLength MsoTrue 自动设置标注线的长度。MsoTriState 类型，只读。
Border返回或设置指定标注中的文本是否具有边框。MsoTriState 类型，可读写。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Drop返回文本范围限制框边到文本框标注线的垂直距离（磅值）。Single 类型，只读。
DropType返回一个代表位置的值，标注线在该位置连到标注文本框。MsoCalloutDropType 类型，只读。
Gap以磅为单位，返回或设置标注线终点与标注文本框边框之间的水平距离。Single 类型，可读写。
Length返回标注的第一条线段（附加到文本标注框的线段）的长度（以磅为单位）。Single 类型，只读。
Parent返回一个 Object 类型的值，该值代表指定 CalloutFormat 对象的父对象。
Type返回或设置标注类型。MsoCalloutType 类型，可读写。
