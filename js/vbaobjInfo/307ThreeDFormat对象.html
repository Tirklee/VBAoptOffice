ThreeDFormat 对象成员

代表形状的三维格式。

方法

 名称说明
IncrementRotationHorizontal按照指定的增量值在 x 轴上水平旋转一个形状。
IncrementRotationVertical 按照指定的增量值在 y 轴上垂直旋转一个形状。
IncrementRotationX按指定度数值，改变指定图形绕 X 轴的旋转度。
IncrementRotationY按指定的度数值，改变指定图形绕 Y 轴的旋转量。
IncrementRotationZ按照指定的增量在 z 轴上旋转一个形状。
ResetRotation将延伸绕 x 轴和 y 轴的旋转重设为 0（零），使延伸的正面向前。
SetExtrusionDirection设置延伸部分经过的路径偏离延伸形状的方向。
SetPresetCamera设置形状的照相机预设。
SetThreeDFormat设置预设的延伸格式。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
BevelBottomDepth返回或设置一个 Single 类型的值，该值代表底部棱台的深度。可读/写。
BevelBottomInset返回或设置一个 Single 类型的值，该值代表底部棱台的嵌入尺寸。可读/写。
BevelBottomType返回或设置一个代表底部棱台的棱台类型的 MsoPresetCamera 常量。可读/写。
BevelTopDepth返回或设置一个 Single 类型的值，该值代表顶部棱台的深度。可读/写。
BevelTopInset返回或设置一个 Single 类型的值，该值代表顶部棱台的嵌入尺寸。可读/写。
BevelTopType返回或设置一个代表顶部棱台的棱台类型的 MsoPresetCamera 常量。可读/写。
ContourColor返回或设置一个代表形状的轮廓线颜色的 ColorFormat 对象。可读/写。
ContourWidth返回或设置一个 Single 类型的值，该值代表形状的轮廓线宽度。可读/写。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Depth返回或设置图形的延伸深度。Single 类型，可读写。
ExtrusionColor返回一个 ColorFormat 对象，该对象代表形状延伸区域的颜色。只读。
ExtrusionColorType返回或设置一个属性值，该值表明延伸区域的颜色是否基于原图形的填充（延伸区域的正面）色，并随着图形填充色的改变而自动变化，或者延伸区域的颜色是否与图形填充色的无关。MsoExtrusionColorType，可读写。
FieldOfView返回或设置一个 Single 类型的值，该值代表形状的透视度。可读/写。
LightAngle返回或设置一个 Single 类型的值，该值代表照明的角度。可读/写。
Parent返回一个 Object 类型的值，该值代表指定 ThreeDFormat 对象的父对象。
Perspective 如果以透视效果显示延伸，即延伸部分的外壁向消失点收敛，则该属性值为 MsoTrue。如果延伸采用平行、正交或投影方式，即外壁不向消失点收敛，则该属性值为 MsoFalse。MsoTriState 类型，可读写。
PresetCamera返回一个代表照相机预设的 MsoPresetCamera 常量。只读。
PresetExtrusionDirection返回形状（指延伸部分的正面）延伸的方向。MsoPresetExtrusionDirection 类型，可读写。
PresetLighting返回或设置一个代表照明预设的 MsoBevelType 常量。可读/写。
PresetLightingDirection返回或设置光源相对于延伸的位置。MsoPresetLightingDirection 类型，可读写。
PresetLightingSoftness返回或设置延伸光源的亮度。MsoPresetLightingSoftness 类型，可读写。
PresetMaterial返回或设置延伸的表面材料。MsoPresetMaterial 类型，可读写。
PresetThreeDFormat返回预设延伸格式。MsoPresetThreeDFormat 类型，只读。
ProjectText返回或设置一个表示形状上的文本是否随同形状一起旋转的 MsoTriState 常量。msoTrue 表示旋转文本。可读/写。
RotationX返回或设置延伸图形绕 X 轴旋转的度数。Single 类型，可读写。
RotationY返回或设置图形延伸部分绕 Y 轴旋转的度数。Single 类型，可读写。
RotationZ返回或设置一个 Single 类型的值，该值代表照相机的 z 轴旋转。可读/写。
Visible 如果指定对象或应用于该对象的格式是可见的，则该属性值为 True。MsoTriState 类型，可读写。
Z返回或设置一个 Single 类型的值，该值代表形状的 z 轴。可读/写。
