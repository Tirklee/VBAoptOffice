SmartTagType 对象成员

代表智能标记的类型。智能标记类型是智能标记列表中的单个项目。智能标记列表可以包含多个智能标记类型。例如，安装在英文系统上的 Address 智能标记列表默认情况下包含 name 智能标记类型、street 智能标记类型和 city 智能标记类型等等。

属性

  名称 说明
Application 返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator 返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
FriendlyName 返回一个 String 类型的值，该值代表智能标记类型的显示名称。
Name 返回或设置智能标记类型的名称。String 类型，只读。
Parent 返回一个 Object 类型值，该值代表指定 SmartTagType 对象的父对象。
SmartTagActions 返回一个 SmartTagActions 集合，该集合代表智能标记上可用操作的集合。
SmartTagRecognizers 返回应用程序的一个 SmartTagRecognizers 集合。
