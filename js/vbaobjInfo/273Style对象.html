Style 对象成员

代表单个内置样式或用户定义的样式。Style 对象将样式属性（如字体、字形、段落间距）添加为 Style 对象的属性。Style 对象是 Styles 集合的成员。Styles 集合包含指定文档中的所有样式。

方法

 名称说明
Delete删除指定的样式。
LinkToListTemplate将指定的样式与一个列表模板链接，如此该样式中的格式就可以应用于列表。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
AutomaticallyUpdate 如果样式根据选定内容重新定义，则该属性值为 True。Boolean 类型，可读写。
BaseStyle返回或设置一种现有样式，根据此样式可以设置其他样式。Variant 类型，可读写。
Borders返回一个 Borders 集合，该集合代表指定样式的所有边框。
BuiltIn 如果指定对象是 Word 的内置样式或题注标签之一，则该属性值为 True。Boolean 类型，只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Description返回指定样式的说明。只读 String 类型。
Font返回或设置一个 Font 对象，该对象代表指定样式的字符格式。可读写 Font。
Frame返回一个 Frame 对象，该对象代表指定样式的图文框格式。只读。
InUse 如果为 True，则指定的样式是一个已在文档中修改或应用的内置样式，或者是在文档中新建的样式。Boolean 类型，只读。
LanguageID返回或设置一个 WdLanguageID 常量，该常量代表指定范围的语言。可读写。
LanguageIDFarEast返回或设置指定对象的东亚语言。可读写 WdLanguageID 类型。
Linked返回或设置 Boolean 值，该值表示某样式是否是段落和字符格式都可以使用的链接样式。只读。
LinkStyle设置或返回一个 Variant 类型的值，该值代表段落和字符样式之间的链接。可读写。
ListLevelNumber返回指定样式的列表级别。只读 Long 类型。
ListTemplate返回一个 ListTemplate 对象，该对象代表指定 Style 对象的列表格式。
Locked如果不能更改或编辑样式，则该属性值为 True。可读写 Boolean 类型。
NameLocal返回由用户语言表示的内置样式的名称。可读写 String 类型。
NextParagraphStyle返回或设置自动应用于在格式设置为指定样式的段落之后插入的新段落的样式。Variant 类型，可读写。
NoProofing 如果拼写和语法检查程序忽略设置为此样式格式的文本，则该属性值为 True。可读写 Long 类型。
NoSpaceBetweenParagraphsOfSameStyle 如果为 True，则 Microsoft Word 会删除设置为使用相同样式的段落的间距。Boolean 类型，可读写。
ParagraphFormat返回或设置一个 ParagraphFormat 对象，该对象代表指定样式的段落设置。可读写。
Parent返回一个 Object 类型的值，该值代表指定 Style 对象的父对象。
Priority返回或设置一个 Long 类型的值，该值代表“样式”任务窗格中排序样式的优先级。可读写。
QuickStyle返回或设置一个 Boolean 类型的值，该值代表样式是否对应于可用的快速样式。可读写。
Shading返回一个 Shading 对象，该对象代表指定对象的底纹格式。
Table返回一个 TableStyle 对象，该对象代表可通过使用表格样式应用于表格的属性。
Type返回样式的类型。只读 WdStyleType 类型。
UnhideWhenUsed如果在文档中使用指定样式后，该样式在 Word 2010 的“样式”库和“样式”任务窗格中显示为建议样式，则该属性值为 True。可读写。
Visibility如果指定样式在“样式”库和“样式”任务窗格中显示为建议样式，则该属性值为 True。可读写。
