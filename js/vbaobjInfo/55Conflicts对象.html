Conflicts 对象成员

Conflict 对象的集合，这些对象代表文档中的冲突。Conflict 对象的类型由 WdRevisionType 枚举指定。

方法

 名称说明
AcceptAll接受所有冲突更改，删除冲突，并将更改合并到文档的服务器副本中。
Item返回集合中的单个 Conflicts 对象。
RejectAll拒绝所有用户的更改并保留文档的服务器副本。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。只读。
Count返回 Conflicts 集合中的项目数。只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Parent返回一个 Object 类型的值，该值代表指定 Conflicts 对象的父对象。
