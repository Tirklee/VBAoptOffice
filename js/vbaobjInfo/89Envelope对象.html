Envelope 对象成员

代表附加到文档的信封。

方法

 名称说明
Insert在指定文档的开头以单独一节的形式插入一个信封。
Options显示“信封选项”对话框。
PrintOut打印信封但不将信封添至活动文档。
UpdateDocument用当前的信封设置更新文档中的信封。

属性

 名称说明
Address以 Range 对象形式返回信件收件人地址。只读。
AddressFromLeft返回或设置信封左边距与收信人地址之间的距离（以磅为单位）。Single 类型，可读写。
AddressFromTop返回或设置信封上边距与收信人地址之间的距离（以磅为单位）。Single 类型，可读写。
AddressStyle返回一个 Style 对象，该对象代表信封的收信人地址样式。只读。
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
DefaultFaceUp 如果按默认值将送纸方式设置为正面向上，则该属性值为 True。Boolean 类型，可读写。
DefaultHeight返回或设置默认的信封高度（以磅为单位）。Single 类型，可读写。
DefaultOmitReturnAddress 如果默认省略信封上的寄信人地址，则该属性值为 True。Boolean 类型，可读写。
DefaultOrientation返回或设置信封的默认送纸方向。WdEnvelopeOrientation 类型，可读写。
DefaultPrintFIMA 如果为 True，则默认向信封添加“饰面标识标记”(FIM-A)。Boolean 类型，可读写。
DefaultSize返回或设置默认信封尺寸。String 类型，可读写。
DefaultWidth返回或设置默认信封宽度（以磅为单位）。Single 类型，可读写。
FeedSource返回或设置打印信封时的纸盒。WdPaperTray 类型，可读写。
Parent返回一个 Object 类型的值，该值代表指定 Envelope 对象的父对象。
RecipientNamefromLeft返回或设置一个 Single 类型的值，该值代表收件人姓名距信封左边距的位置（以磅为单位）。可读写。
RecipientNamefromTop返回或设置一个 Single 类型的值，该值代表收件人姓名距信封上边距的位置（以磅为单位）。可读写。
RecipientPostalfromLeft返回或设置一个 Single 类型的值，该值代表收件人邮政编码距信封左边距的位置（以磅为单位）。可读写。
RecipientPostalfromTop返回或设置一个 Single 类型的值，该值代表收件人邮政编码距信封上边距的位置（以磅为单位）。可读写。
ReturnAddress返回一个 Range 对象，该对象代表寄信人地址。
ReturnAddressFromLeft返回或设置信封左边距和寄信人地址之间的距离（以磅为单位）。Single 类型，可读写。
ReturnAddressFromTop返回或设置信封上边距和寄信人地址之间的距离（以磅为单位）。Single 类型，可读写。
ReturnAddressStyle返回一个 Style 对象，该对象代表信封的寄信人地址样式。
SenderNamefromLeft返回或设置一个 Single 类型的值，该值代表发件人姓名距信封左边距的位置（以磅为单位）。可读写。
SenderNamefromTop返回或设置一个 Single 类型的值，该值代表发件人姓名距信封上边距的位置（以磅为单位）。可读写。
SenderPostalfromLeft返回或设置一个 Single 类型的值，该值代表发件人邮政编码距信封左边距的位置（以磅为单位）。可读写。
SenderPostalfromTop返回或设置一个 Single 类型的值，该值代表发件人邮政编码距信封上边距的位置（以磅为单位）。可读写。
Vertical 如果该属性值为 True，则在中文信封上垂直排列文本。Boolean 类型，可读写。
