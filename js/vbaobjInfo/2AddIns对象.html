AddIns 对象成员

AddIn 对象的集合，代表所有可供 Word 使用的加载项，不论它们当前是否加载。AddIns 集合包含在“模板和加载项”对话框中显示的全局模板或 Word 加载项库 (WLL)。

方法

 名称说明
Add返回一个 AddIn 对象，该对象代表一个添加到可用加载项列表中的加载项。
Item返回集合中的单个对象。
Unload卸载所有已加载的加载项，并根据 RemoveFromList 参数值删除 AddIns 集合中相应的加载项。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Count返回 AddIns 集合中 AddIn 对象的数目。只读 Long 类型。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Parent返回一个 Object 类型的值，该值代表 Addins 集合中的父对象。通常为 Application 对象。
