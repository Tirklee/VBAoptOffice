Columns 对象成员

Column 对象的集合，这些对象代表表格中的列。

方法

 名称说明
Add返回一个 Column 对象，该对象代表添加到表中的列。
AutoFit改变表格列宽，使之在单元格文本换行方式不变的情况下，适应文本宽度。
Delete删除指定列。
DistributeWidth将指定列调整为等宽。
Item返回集合中的单个 Column 对象。
Select选择指定表格列。
SetWidth设置表格列的宽度。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Borders返回一个 Borders 集合，该集合代表指定列的所有边框。
Count返回一个 Long 类型的值，该值代表集合中列的数目。只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
First返回一个 Column 对象，该对象代表 Columns 集合中的第一个项目。
Last返回表示表格中最后一列的 Column 对象。
NestingLevel返回指定列的嵌套层。只读 Long 类型。
Parent返回一个 Object 类型的值，该值代表指定 Columns 对象的父对象。
PreferredWidth返回或设置指定列的首选宽度（以磅为单位或表示为窗口宽度的百分比）。Single 类型，可读写。
PreferredWidthType返回或设置用于指定单元格、列或表格的首选度量单位。可读/写 WdPreferredWidthType 类型。
Shading返回一个表示指定表格列的底纹格式的 Shading 对象。
Width返回或设置指定列的宽度，以磅为单位。可读/写 Long 类型。
