ProtectedViewWindow 对象成员

代表受保护的视图窗口。

方法

 名称说明
Activate激活指定的受保护的视图窗口。
Close关闭指定的受保护的视图窗口。
Edit以后的版本中将提供关于此项目的说明。
ToggleRibbon显示或隐藏功能区。

属性

 名称说明
Active如果指定的受保护的视图窗口处于活动状态，则该属性值为 True。只读 Boolean 类型。
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。只读。
Caption返回或设置显示在文档或受保护的视图窗口标题栏中的标题文字。可读写 String。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Document返回一个 Document 对象，该对象与受保护的视图窗口相关。只读。
Height返回或设置受保护的视图窗口的高度。可读写 Long。
Index返回 Long 值，该值表示项目在集合中的位置。只读。
Left返回或设置一个 Long 值（以磅为单位），该值代表所指定受保护的视图窗口的水平位置。可读写。
Parent返回一个 Object 类型的值，该值代表指定 ProtectedViewWindow 对象的父对象。
SourceName返回所指定受保护的视图窗口的源文件名称。只读 String。
SourcePath返回所指定受保护的视图窗口的源文件路径。只读 String。
Top返回或设置所指定受保护的视图窗口的垂直位置（以磅为单位）。可读写 Long。
Visible如果指定的受保护的视图窗口可见，则该属性值为 True。可读写 Boolean。
Width返回或设置所指定受保护的视图窗口的宽度（以磅为单位）。可读写 Long。
WindowState返回或设置所指定受保护的视图窗口的状态。可读写 WdWindowState。
