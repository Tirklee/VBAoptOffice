MailMessage 对象成员

代表在将 Microsoft Word 用作电子邮件编辑器时的活动电子邮件。

方法

 名称说明
CheckName验证活动电子邮件中“收件人”、“抄送”、“密件抄送”行中显示的电子邮件地址。
Delete删除指定的邮件。
DisplayMoveDialog显示“移动”对话框，用户可在该对话框中为可用的邮件存储区中的活动电子邮件指定一个新位置。
DisplayProperties为活动的电子邮件显示“属性”对话框。
DisplaySelectNamesDialog显示“选择姓名”对话框，用户可在该对话框向活动、未发送电子邮件的“收件人”、“抄送”和“密件抄送”行添加地址。
Forward打开一个新的“收件人”行为空的电子邮件以转发当前邮件。
GoToNext如果将 Word 作为邮件编辑器，则显示下一条邮件信息。
GoToPrevious如果将 Word 作为邮件编辑器，将显示下一条邮件的信息。
Reply打开一个新的电子邮件，并在“收件人”行添加发件人地址，以答复当前邮件。
ReplyAll打开一个新的电子邮件，并根据需要在“收件人”行和“抄送”行添加发件人和所有其他收件人的地址，以答复当前邮件。
ToggleHeader切换活动电子邮件中页眉的显示。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Parent返回一个 Object 类型的值，该值代表指定 MailMessage 对象的父对象。
