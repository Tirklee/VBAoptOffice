OMathBorderBox 对象成员

代表整个公式或部分公式周围不可见的框，您可以向该公式分配属性，以改变整个框的布局或数学格式。例如，框可以作为带对齐点或不带对齐点的运算符模拟器，可以作为类似的断点，可以具有相关的参数大小 (argSz) 或进行分组以不允许换行。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回一个 32 位的整数，该整数表示在其中创建加载项的应用程序。只读 Long 类型。
E返回一个 OMath 对象，该对象代表指定公式对象的基对象。只读。
HideBot返回或设置一个 Boolean 类型的值，该值代表是否隐藏公式边界框的下框线。可读写。
HideLeft返回或设置一个 Boolean 类型的值，该值代表是否隐藏公式边界框的左侧框线。可读写。
HideRight返回或设置一个 Boolean 类型的值，该值代表是否隐藏公式边界框的右侧框线。可读写。
HideTop返回或设置一个 Boolean 类型的值，该值代表是否隐藏公式边界框的上框线。可读写。
Parent返回一个 Object 类型的值，该值代表指定 OMathBorderBox 对象的父对象。
StrikeBLTR返回或设置一个 Boolean 类型的值，该值代表从左下角到右上角的斜删除线。可读写。
StrikeH返回或设置一个 Boolean 类型的值，该值代表水平删除线。可读写。
StrikeTLBR返回或设置一个 Boolean 类型的值，该值代表从左上角到右下角的斜删除线。可读写。
StrikeV返回或设置一个 Boolean 类型的值，该值代表垂直删除线。可读写。
