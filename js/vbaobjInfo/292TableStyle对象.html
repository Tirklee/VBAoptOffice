TableStyle 对象成员

代表可应用于表格的单个样式。

方法

 名称说明
Condition返回一个 ConditionalStyle 对象，该对象代表表格某个部分的特殊样式格式设置。

属性

 名称说明
Alignment返回或设置一个 WdRowAlignment 常量，该常量代表指定表格样式的对齐方式，可读写。
AllowBreakAcrossPage设置或返回一个 Long 类型的值，表明设置为特定样式的表格行中的文本行是否可以跨页断行。可读写。
AllowPageBreaks允许 Microsoft Word 对指定表格进行跨页断行。Boolean 类型，可读写。
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Borders返回一个 Borders 集合，该集合代表指定对象的所有边框。
BottomPadding返回或设置在表格中单独的单元格或所有单元格内容的下方添加的间距（以磅为单位）。Single 类型，可读写。
ColumnStripe返回或设置一个 Long 类型的值，当样式指定奇数或偶数列成组时，该属性代表组中的列数。可读写。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
LeftIndent返回或设置一个 Single 类型的值，该值代表采用指定表格样式的行的左缩进值（以磅为单位）。可读写。
LeftPadding返回或设置在表格中所有单元格的内容左侧要增加的间距（以磅为单位）。可读写 Single 类型。
Parent返回一个 Object 类型的值，该值代表指定 TableStyle 对象的父对象。
RightPadding返回或设置在表格的所有单元格的内容右侧要增加的间距（以磅为单位）。可读写 Single 类型。
RowStripe返回或设置一个 Long 类型的值，当样式指定奇数或偶数行成组时，代表组中包含的行数。可读/写。
Shading返回一个 Shading 对象，该对象代表指定表格样式的底纹格式。
Spacing返回或设置采用表格样式的单元格的间距（以磅为单位）。可读写 Single 类型。
TableDirection返回或设置 Microsoft Word 对采用指定表格样式的单元格进行排序的方向。可读写 WdTableDirection 类型。
TopPadding返回或设置表格中所有单元格的内容上方要增加的间距（以磅为单位）。可读写 Single 类型。
