Cells 对象成员

由表格列、表格行、选定内容或区域中的 Cell 对象组成的集合。

方法

 名称说明
Add返回一个 Cell 对象，该对象代表添加到表格中的单元格。
AutoFit改变表格列宽，使之在单元格文本换行方式不变的情况下，适应文本宽度。
Delete删除一个或多个表格单元格并可选择控制如何移动剩余的单元格。
DistributeHeight将指定单元格调整为等高。
DistributeWidth将指定单元格调整为等宽。
Item返回集合中的单个 Cell 对象。
Merge将指定的多个表格单元格相互合并。合并后的结果是一个表格单元格。
SetHeight设定表格单元格的高度。
SetWidth设置表格的列或单元格的宽度。
Split拆分表格单元格范围。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Borders返回一个 Borders 集合，该集合代表指定对象的所有边框。
Count返回 Cells 集合中的项目数。只读 Long 类型。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Height返回或设置指定表格单元格的高度。可读/写 Single 类型。
HeightRule返回或设置一个 WdRowHeightRule 常量，该常量代表确定指定单元格高度的规则。可读写。
NestingLevel返回指定单元格的嵌套层。Long 类型，只读。
Parent返回一个 Object 类型的值，该值代表指定 Cells 对象的父对象。
PreferredWidth返回或设置指定单元格的首选宽度（以磅为单位或表示为窗口宽度的百分比）。Single 类型，可读写。
PreferredWidthType返回或设置用于指定单元格的宽度的首选度量单位。WdPreferredWidthType 类型，只读。
Shading返回一个 Shading 对象，该对象代表指定对象的底纹格式。
VerticalAlignment返回或设置表格中一个或多个单元格中文字的垂直对齐方式。WdCellVerticalAlignment 类型，可读写。
Width返回或设置表格单元格的宽度（以磅为单位）。Long 类型，可读写。
