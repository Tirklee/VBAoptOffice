FileConverter 对象成员

代表用于打开或保存文件的文件转换器。FileConverter 对象是 FileConverters 集合的一个成员。FileConverters 集合包含所有已安装的用于打开和保存文件的文件转换器。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
CanOpen 如果指定的文件转换器可用来打开文件，则该属性值为 True。只读 Boolean 类型。
CanSave 如果指定的文件转换器可用来保存文件，则该属性值为 True。只读 Boolean 类型。
ClassName返回用来标识文件转换器的唯一名称。只读 String 类型。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Extensions返回与指定 FileConverter 对象关联的文件扩展名。只读 String 类型。
FormatName返回指定文件转换器的名称。只读 String 类型。
Name返回指定对象的名称。String 类型，只读。
OpenFormat返回指定文件转换器的文件格式。只读 Long 类型。
Parent返回一个 Object 类型的值，该值代表指定 FileConverter 对象的父对象。
Path返回指定对象的磁盘或 Web 路径。只读 String 类型。
SaveFormat返回指定文档或文件转换器的文件格式。只读 Long 类型。
