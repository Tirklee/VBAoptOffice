Rectangle 对象成员

代表页面中的文本或图形部分。使用 Rectangle 对象及相关方法和属性可通过编程方式定义文档的页面版式。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Height返回一个 Long 类型的值，该值代表矩形的高度（以像素为单位）。
Left返回一个 Long 类型的值，该值代表从页面左边缘到矩形左边缘的像素数。
Lines返回一个 Lines 集合，该集合代表页面上指定文字部分中的行数。
Parent返回一个 Object 类型的值，该值代表指定 Rectangle 对象的父对象。
Range返回一个 Range 对象，该对象代表矩形内包含的文档部分。
RectangleType返回一个 WdRectangleType 常量，该常量代表指定矩形的类型。
Top返回一个 Long 类型的值，该值代表从页面顶部到矩形顶部的像素数。
Width返回或设置一个 Long 类型的值，该值代表矩形的宽度（以磅为单位）。可读写 Long 类型。
