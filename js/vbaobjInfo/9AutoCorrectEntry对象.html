AutoCorrectEntry 对象成员

代表单个“自动更正”词条。AutoCorrectEntry 对象是 AutoCorrectEntries 集合的一个成员。AutoCorrectEntries 集合包括“自动更正”对话框（“工具”菜单）中的词条。

方法

 名称说明
Apply用指定的“自动更正”词条的值替换某一区域。
Delete删除指定的“自动更正”词条。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Index返回 Long 值，该值表示项目在集合中的位置。只读。
Name返回或设置指定对象的名称。String 类型，可读写。
Parent返回一个 Object 类型的值，该值代表指定 AutoCorrectEntry 对象的父对象。
RichText 如果将替换文本连同格式保存为“自动更正”词条，则该属性值为 True。Boolean 类型，只读。
Value返回或设置自动更正词条的值。String 类型，可读写。
