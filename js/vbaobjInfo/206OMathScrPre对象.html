OMathScrPre 对象成员

代表一个在基线左侧包含上标或下标的公式。

方法

 名称说明
ToScrSubSup 将在基线左侧具有上标或下标的公式转换为具有上标或下标基线的公式。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回一个 32 位的整数，该整数表示在其中创建加载项的应用程序。只读 Long 类型。
E返回一个 OMath 对象，该对象代表指定公式对象的基对象。只读。
Parent返回一个 Object 类型的值，该值代表指定 OMathScrPre 对象的父对象。
Sub返回一个 OMath 对象，该对象代表前缀-下标-上标对象的下标。只读。
Sup返回一个 OMath 对象，该对象代表前缀-下标-上标对象的上标。只读。
