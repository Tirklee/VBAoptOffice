ReflectionFormat 对象成员

代表形状或形状区域的反射格式。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。只读。
Blur返回或设置一个 Single 值，该值指定应用于指定对象的虚化效果程度。可读写。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Offset返回或设置形状的反射图像的间隔量（以磅为单位）。可读写。
Parent返回一个 Object 类型的值，该值代表指定 ReflectionFormat 对象的父对象。
Size返回或设置反射的大小作为反射形状的百分比，该值介于 0 到 100 之间。可读写。
Transparency返回或设置反射效果的透明度，其值为从 0.0（不透明）到 1.0（全透明）。可读写 Single。
Type返回或设置一个 MsoLightRigType 常量，该常量代表形状反射的照明类型和方向。可读写。
