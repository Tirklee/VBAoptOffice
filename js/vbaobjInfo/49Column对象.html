Column 对象成员

代表单个表格列。Column 对象是 Columns 集合的成员。Columns 集合包括一个表格、选定内容或区域中的所有列。

方法

 名称说明
AutoFit改变表格列宽，使之在单元格文本换行方式不变的情况下，适应文本宽度。
Delete删除指定列。
Select选择指定表格列。
SetWidth设置表格列的宽度。
Sort对指定表格列进行排序。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Borders返回一个 Borders 集合，该集合代表指定对象的所有边框。
Cells返回一个 Cells 集合，该集合代表表格列中的表格单元格。只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Index返回 Long 值，该值表示项目在集合中的位置。只读。
IsFirst 如果指定的列或行是表格的第一列或第一行，则返回 True。只读 Boolean 类型。
IsLast 如果指定的列或行是表格的最后一列或最后一行，则返回 True。只读 Boolean 类型。
NestingLevel返回指定列的嵌套层。只读 Long 类型。
Next返回表格列集合中的下一列。只读。
Parent返回一个 Object 类型的值，该值代表指定 Column 对象的父对象。
PreferredWidth返回或设置指定的单元格、列或表格的首选宽度（以磅为单位或表示为窗口宽度的百分比）。Single 类型，可读写。
PreferredWidthType返回或设置用于指定表格列的宽度的首选度量单位。可读/写 WdPreferredWidthType 类型。
Previous返回表格列集合中的前一列。只读。
Shading返回一个引用指定列的底纹格式的 Shading 对象。
Width 返回或设置指定列的宽度，以磅为单位。可读/写 Long 类型。
