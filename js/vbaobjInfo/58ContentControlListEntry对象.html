ContentControlListEntry 对象成员

ContentControlListEntry 对象表示下拉列表或组合框内容控件中的列表项。ContentControlListEntry 对象是 ContentControl 对象的 ContentControlListEntries 集合的成员。

方法

 名称说明
Delete 删除组合框或下拉列表内容控件中的指定项。
MoveDown 将下拉列表或组合框内容控件中的项下移一项，以使它位于最初在它后面的某项之后。
MoveUp 将下拉列表或组合框内容控件中的项上移一项，以使它位于最初在它前面的某项之前。
Select 选择下拉列表或组合框内容控件中的列表条目，并将内容控件的文本设置为该项的值。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回一个 32 位的整数，该整数表示在其中创建加载项的应用程序。只读 Long 类型。
Index返回或设置 Long 值，该值表示内容控件列表项在列表项集合中的序号位置。可读/写。
Parent返回一个 Object 类型的值，该值代表指定 ContentControlListEntry 对象的父对象。
Text返回或设置 String 值，该值表示下拉列表或组合框内容控件中的某个列表项的显示文本。可读/写。
Value返回或设置 String 值，该值表示某项在下拉列表或组合框内容控件中的编程值。可读/写。
