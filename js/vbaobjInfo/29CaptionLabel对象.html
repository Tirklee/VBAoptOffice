CaptionLabel 对象成员

代表单个题注标签。CaptionLabel 对象是 CaptionLabels 集合的成员。CaptionLabels 集合中的项目列在“题注”对话框的“标签”框中。

方法

 名称说明
Delete删除指定的题注标签。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
BuiltIn 如果指定的题注标签是 Word 的内置题注标签之一，则该属性值为 True。Boolean 类型，只读。
ChapterStyleLevel当指定题注标签包含章节号时，返回或设置用来标记新章节的标题样式。Long 类型，可读写。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
ID如果 CaptionLabel 对象的 BuiltIn 属性为 True，则返回代表指定题注标签类型的 WdCaptionLabelID 常量。只读。
IncludeChapterNumber 如果页码或题注标签中包含章节号，则该属性值为 True。Boolean 类型，可读写。
Name返回指定对象的名称。只读 String 类型。
NumberStyle返回或设置 CaptionLabel 对象的编号样式。WdCaptionNumberStyle 类型，可读写。
Parent返回一个 Object 类型的值，该值代表指定 CaptionLabel 对象的父对象。
Position返回或设置题注标签文本的位置。可读/写 WdCaptionPosition 类型。
Separator返回或设置章节编号和序号间的字符。可读/写 WdSeparatorType 类型。
