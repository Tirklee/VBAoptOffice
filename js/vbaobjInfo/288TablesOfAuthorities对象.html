TablesOfAuthorities 对象成员

TableOfAuthorities 对象（TOA 字段）的集合，该集合代表指定文档中的引文目录。

方法

 名称说明
Add返回一个 TableOfAuthorities 对象，该对象代表添加到文档中的引文目录。
Item返回集合中的单个 TableOfAuthorities 对象。
MarkAllCitations在所有 ShortCitation 文本的实例后面插入 TA（引文目录）域。
MarkCitation插入一个 TA（引文目录项）域并作为 Field 对象返回该域。
NextCitation查找并选定由 ShortCitation 参数所指定的文本的下一个实例。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Count返回一个 Long 类型的值，该值代表集合中引文目录的数量。只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Format返回或设置指定文档中引文目录的格式。WdToaFormat 类型，可读写。
Parent返回一个 Object 类型的值，该值代表指定 TablesOfAuthorities 对象的父对象。
