KeyBinding 对象成员

代表当前上下文中的一个自定义按键分配方案。KeyBinding 对象是 KeyBindings 集合的一个成员。

方法

 名称说明
Clear从 KeyBindings 集合中删除指定的键绑定，并将内置命令重新设置为默认按键分配方案。
Disable如果指定组合键当前被赋予某一命令，则删除该组合键。使用该方法后，组合键将不再起作用。
Execute运行与指定的组合键相关联的命令。
Rebind更改分配给指定的组合键的命令。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Command返回分配给指定组合键的命令。String 类型，只读。
CommandParameter返回分配给指定快捷键的命令参数。只读 String 类型。
Context返回一个 Object 类型的值，该值代表指定键绑定的存储位置。只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
KeyCategory返回分配给指定组合键的项目的类型。只读 WdKeyCategory 类型。
KeyCode返回指定键绑定中第一个键的唯一代码。Long 类型，只读。
KeyCode2返回指定键绑定中第二个键的唯一代码。Long 类型，只读。
KeyString返回指定键的组合键字符串（例如，Ctrl+Shift+A）。只读 String 类型。
Parent返回一个 Object 类型的值，该值代表指定 KeyBinding 对象的父对象。
Protected 如果无法更改“自定义键盘”对话框中的指定键绑定，则该属性值为 True。Boolean 类型，只读。
