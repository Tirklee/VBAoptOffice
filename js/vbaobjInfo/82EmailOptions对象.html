EmailOptions 对象成员
全部显示
全部隐藏

在创建、编辑电子邮件和答复邮件时，该对象将包含 Microsoft Word 使用的全局应用程序级属性。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
AutoFormatAsYouTypeApplyBorders 如果按下 Enter 后，三个或更多的连字符 (-)、等号 (=) 或下划线 (_) 将自动替换成指定的边框线，则该属性值为 True。Boolean 类型，可读写。
AutoFormatAsYouTypeApplyBulletedLists 如果用项目符号替换项目符号字符（如星号、连字符和大于符号），则该属性值为 True。Boolean 类型，可读写。
AutoFormatAsYouTypeApplyClosings 如果为 True，则 Microsoft Word 会在键入时自动为信函的结束语应用“结束语”样式。Boolean 类型，可读写。
AutoFormatAsYouTypeApplyDates 如果为 True，则 Microsoft Word 自动为键入的日期应用“日期”样式。可读写。
AutoFormatAsYouTypeApplyFirstIndents 如果为 True，则 Microsoft Word 在键入时自动用首行缩进替代段落开始处输入的空格。可读写。
AutoFormatAsYouTypeApplyHeadings 如果在键入标题时，自动为标题设置样式，则该属性值为 True。Boolean 类型，可读写。
AutoFormatAsYouTypeApplyNumberedLists 如果将段落格式自动设置为编号列表，则该属性值为 True。Boolean 类型，可读写。
AutoFormatAsYouTypeApplyTables 如果在键入一个加号、一串连字符、另一个加号等，然后按下 Enter 后，Word 自动创建一张表格，则该属性值为 True。Boolean 类型，可读写。
AutoFormatAsYouTypeAutoLetterWizard 如果为 True，则在用户输入信函称呼或结束语时，Microsoft Word 会自动启动“英文信函向导”。可读写。
AutoFormatAsYouTypeDefineStyles 如果 Word 自动根据手动设置的格式新建一种样式，则该属性值为 True。Boolean 类型，可读写。
AutoFormatAsYouTypeDeleteAutoSpaces 如果为 True，则 Microsoft Word 会在用户键入时自动删除日文和拉丁文文字之间插入的空格。可读写。
AutoFormatAsYouTypeFormatListItemBeginning 如果 Word 将列表项开始处的字符格式重复应用于下一个列表项，则该属性值为 True。Boolean 类型，可读写。
AutoFormatAsYouTypeInsertClosings 如果为 True，则当用户输入备忘录标题时，Microsoft Word 会自动插入相应的备忘录结束语。可读写。
AutoFormatAsYouTypeInsertOvers 如果为 True，则当用户输入“ ”或“ ”时，Microsoft Word 将自动插入“ ”。Boolean 类型，可读写。
AutoFormatAsYouTypeMatchParentheses 如果为 True，Microsoft Word 会自动更正不正确成对的括号，可读写。
AutoFormatAsYouTypeReplaceFarEastDashes 如果为 True，Microsoft Word 会自动更正长元音和划线，可读写。
AutoFormatAsYouTypeReplaceFractions 如果在您键入时，Word 自动用当前字符集的分数替换键入的分数（例如，将“1/2”替换为“?”），则该属性值为 True。Boolean 类型，可读写。
AutoFormatAsYouTypeReplaceHyperlinks 如果本属性值为 True，则 Word 会在键入时自动将电子邮件地址、服务器和共享名称（即 UNC 路径）和 Internet 地址（即 URL）改为超链接。Boolean 类型，可读写。
AutoFormatAsYouTypeReplaceOrdinals 如果键入时，Word 会用相同字母的上标格式替换序数后缀“st”、“nd”、“rd”和“th”（例如，“1st”被替换为“1”后接上标“st”），则该属性值为 True。Boolean 类型，可读写。
AutoFormatAsYouTypeReplacePlainTextEmphasis 如果键入时，手动着重号自动替换为相应字符格式；例如，将“*bold*”改为“bold”，则该属性值为 True。Boolean 类型，可读写。
AutoFormatAsYouTypeReplaceQuotes 如果键入时自动将所键入的直引号替换为弯引号，则该属性值为 True。Boolean 类型，可读写。
AutoFormatAsYouTypeReplaceSymbols 如果键入的两个连续的连字符 (--) 被替换为短破折号 (–) 或长破折号 (?)，则该属性值为 True。Boolean 类型，可读写。
ComposeStyle返回一个 Style 对象，该对象代表撰写新电子邮件时使用的样式。只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
EmailSignature返回一个 EmailSignature 对象，该对象代表 Microsoft Word 附加在待发电子邮件正文后面的签名。只读。
HTMLFidelity删除那些用于在 Word 中打开 HTML 文件，但在显示时不需要的 HTML 标记。WdEmailHTMLFidelity 类型，可读写。
MarkComments 如果 Microsoft Word 在电子邮件中标记用户的批注，则该属性值为 True。Boolean 类型，可读写。
MarkCommentsWith返回或设置 Microsoft Word 在电子邮件中标记批注的字符串。String 类型，可读写。
NewColorOnReply 如果为 True，则指定用户要在回复电子邮件时为回复邮件的文本选择一种新颜色。Boolean 类型，可读写。
Parent返回一个 Object 类型的值，该值代表指定 EmailOptions 对象的父对象。
PlainTextStyle返回 Style 对象，该对象代表用纯文本发送或接收的电子邮件的文本属性。
RelyOnCSS 如果在 Web 浏览器中查看保存的文档时，对字体格式使用了级联样式表 (CSS)，则该属性为 True。可读/写 Boolean 类型。
ReplyStyle返回一个 Style 对象，该对象代表答复电子邮件时使用的样式。
TabIndentKey 如果可分别使用 Tab 和 Backspace 键增加和减少段落的左缩进，并且可使用 Backspace 键将右对齐的段落更改为居中的段落而将居中的段落更改为左对齐的段落，则该属性值为 True。可读写 Boolean 类型。
ThemeName该属性返回或设置用于新的电子邮件的主题 （主题：一组统一的设计元素，使用颜色、字体和图形设置文档的外观。）名称以及所有主题格式选项的名称。String 类型，可读写。
UseThemeStyle 如果新电子邮件使用默认的电子邮件主题 （主题：一组统一的设计元素，使用颜色、字体和图形设置文档的外观。）所定义的字符样式，则该属性值为 True。Boolean 类型，可读写。
UseThemeStyleOnReply 如果为 True，则 Microsoft Word 将在回复电子邮件时应用主题。Boolean 类型，可读写。
