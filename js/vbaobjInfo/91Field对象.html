Field 对象成员

代表一个域。Field 对象是 Fields 集合的一个成员。Fields 集合代表所选内容、范围或文档中的域。

方法

 名称说明
Copy将指定域复制到剪贴板。
Cut将指定域从文档中移到剪贴板上。
Delete删除指定的域。
DoClick单击指定域。
Select选择指定的域。
Unlink将指定域替换为其最新结果。
Update更新域的结果。如果该域更新成功，则返回 True。
UpdateSource将对 INCLUDETEXT 域所做的更改保存回源文档。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Code返回一个代表域代码的 Range 对象。可读写。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Data返回或设置 ADDIN 域中的数据。可读写 String 类型。
Index返回 Long 值，该值表示项目在集合中的位置。只读。
InlineShape返回一个 InlineShape 对象，该对象代表图片、OLE 对象或作为 INCLUDEPICTURE 或 EMBED 域结果的 ActiveX 控件。
Kind返回 Field 对象的链接类型。只读 WdFieldKind 类型。
LinkFormat返回一个 LinkFormat 对象，该对象代表指定域的链接选项。只读。
Locked 如果指定域已被锁定，则该属性值为 True。可读写 Boolean 类型。
Next返回集合中的下一个对象。只读。
OLEFormat返回一个 OLEFormat 对象，该对象代表指定域的 OLE 特征（链接除外）。只读。
Parent返回一个 Object 类型的值，该值代表指定 Field 对象的父对象。
Previous返回集合中的上一个对象。只读。
Result返回一个 Range 对象，该对象代表域的结果。可读写。
ShowCodes 如果显示指定域的域代码而不是域结果，则该属性值为 True。Boolean 类型，可读写。
Type返回域的类型。只读 WdFieldType 类型。
