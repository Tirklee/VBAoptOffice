Word 开发人员参考
LetterContent 对象成员

表示由“英文信函向导”创建的信函的元素。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
AttentionLine返回或设置由“英文信函向导”创建的信件的信封用语文本。String 类型，可读写。
CCList返回或设置由“英文信函向导”创建的信件的抄送人。String 类型，可读写。
Closing返回或设置由“英文信函向导”创建的信件的结束语。（例如：“Sincerely yours”）。String 类型，可读写。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
DateFormat返回或设置由“英文信函向导”创建的信件的日期。String 类型，可读写。
Duplicate返回一个只读 LetterContent 对象，该对象表示由“英文信函向导”创建的信件的内容。
EnclosureNumber返回或设置由“英文信函向导”创建的信件的附件编号。String 类型，可读写。
IncludeHeaderFooter 如果用“英文信函向导”创建的信函中包含页面设计模板中的页眉和页脚，则该属性值为 True。可读/写 Boolean 类型。使用 PageDesign 属性可设置附加到用“英文信函向导”创建的文档中的模板名称。
InfoBlock与 Microsoft Word 的“英文信函向导”相关。在 Word 的英文版本中没有使用该属性。
Letterhead 如果在用“英文信函向导”创建的信件中留出了预打印信头的空间，则该属性值为 True。可读/写 Boolean 类型。LetterheadSize 属性可控制预留信头空间的尺寸。
LetterheadLocation返回或设置由“英文信函向导”创建的信件中预打印信头的位置。WdLetterheadLocation 类型，可读写。
LetterheadSize返回或设置为信件的预打印信头预留的空格量（以磅为单位），该信件由“英文信函向导”创建。Single 类型，可读写。
LetterStyle返回或设置“英文信函向导”所建信件的版式。WdLetterStyle 类型，可读写。
MailingInstructions返回或设置由“英文信函向导”创建的信件的邮寄说明文本（如：“Certified Mail”）。String 类型，可读写。
PageDesign为“英文信函向导”所建文档返回或设置所附加的模板的名称。String 类型，可读写。
Parent返回一个 Object 类型的值，该值代表指定 LetterContent 对象的父对象。
RecipientAddress返回或设置由“英文信函向导”创建的信件的收件人的邮件地址。String 类型，可读写。
RecipientCode返回或设置收件人代码。String 类型，可读写。
RecipientGender返回或设置收件人的性别（如果已知）。在 Microsoft Word 美国英语版中不使用该属性。WdSalutationGender 类型，可读写。
RecipientName返回或设置由“英文信函向导”创建的信件的收件人的姓名。String 类型，可读写。
RecipientReference返回或设置由“英文信函向导”创建的信件的参考行（如“In reply to:”），String 类型，可读写。
ReturnAddress返回或设置用“英文信函向导”创建的信函的寄信人地址。可读写 String 类型。
ReturnAddressShortForm返回或设置缩写地址。String 类型，可读写。
Salutation返回或设置由“英文信函向导”创建的信件的问候语文本，String 类型，可读写。
SalutationType返回或设置由“英文信函向导”创建的信件的称呼类型。WdSalutationType 类型，可读写。
SenderCity返回或设置发件人所处的城市。在 Microsoft Word 英文版中没有使用该属性。String 类型，可读写。
SenderCode返回或设置发件人代码。在 Microsoft Word 英文版中没有使用该属性。String 类型，可读写。
SenderCompany返回或设置通过“英文信函向导”创建信函的用户的公司名称。String 类型，可读写。
SenderGender返回或设置称呼中使用的性别。在 Microsoft Word 美国英语版中不使用该属性。WdSalutationGender 类型，可读写。
SenderInitials返回或设置通过“英文信函向导”创建信函的用户的缩写。String 类型，可读写。
SenderJobTitle返回或设置通过“英文信函向导”创建信函的用户的职务。String 类型，可读写。
SenderName返回或设置通过“英文信函向导”创建信函的用户的姓名。String 类型，可读写。
SenderReference在 Microsoft Word 美国英语版中不使用该属性。String 类型，可读写。
Subject返回或设置用“英文信函向导”创建的信函的主题文本。可读写 String 类型。
