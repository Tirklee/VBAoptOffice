InlineShapes 对象成员

InlineShape 对象的集合，这些对象代表文档、范围或所选内容中的所有内嵌形状。

方法

 名称说明
AddChart将指定类型的图表作为内嵌形状插入到活动文档中，并且在 Microsoft Office Excel 中打开包含 Microsoft Word 用于创建该图表的默认数据的工作表。
AddHorizontalLine向当前文档添加一条基于图像文件的横线。
AddHorizontalLineStandard向当前文档添加一条横线。
AddOLEControl创建一个 ActiveX 控件（以前称为 OLE 控件）。返回代表新 ActiveX 控件的 InlineShape 对象。
AddOLEObject创建一个 OLE 对象。返回代表新 OLE 对象的 InlineShape 对象。
AddPicture在文档中添加一幅图片。返回一个代表该图片的 InlineShape 对象。
AddPictureBullet向当前文档添加一个基于图像文件的图片项目符号。返回一个 InlineShape 对象。
AddSmartArt将 SmartArt 图形作为内嵌形状插入活动文档。
Item返回集合中的单个 InlineShape 对象。
New插入一个空白的 Word 图片对象，该对象是边长为一英寸的正方形，四周带有边框。此方法以 InlineShape 对象的形式返回新图形。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Count返回一个 Long 类型的值，该值代表集合中内嵌图形的数目。只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Parent返回一个 Object 类型的值，该值代表指定 InlineShapes 对象的父对象。
