Word 开发人员参考
Find 对象成员

代表查找操作的执行条件。

方法

 名称说明
ClearAllFuzzyOptions清除所有与日语文本相关联的非特定搜索选项。
ClearFormatting取消在查找或替换操作中所指定文本的文本格式和段落格式。
ClearHitHighlight 取消突出显示在点击突出显示的查找后找到的所有文本，并且返回一个代表该操作是否成功的 Boolean 类型的值。
Execute运行指定的查找操作。如果查找成功，则返回 True。Boolean 类型。
Execute2007运行指定的查找操作。如果查找操作成功，则返回 True。
HitHighlight 突出显示找到的所有匹配项，并且返回一个代表是否找到匹配项的 Boolean 类型的值。
SetAllFuzzyOptions激活所有与日语文本相关的非特定搜索选项。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
CorrectHangulEndings 如果在替换朝鲜文字文本时，Microsoft Word 自动更正朝鲜文字结尾，则该属性值为 True。Boolean 类型，可读写。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Font返回或设置一个 Font 对象，该对象代表指定对象的字符格式。可读写 Font。
Format 如果在查找操作中包含格式，则该属性值为 True。Boolean 类型，可读写。
Forward 如果查找操作在文档中往前搜索，则该属性值为 True。Boolean 类型，可读写。
Found 如果生成搜索匹配，则该属性值为 True。Boolean 类型，只读。
Frame返回一个 Frame 对象，该对象代表指定样式或查找替换操作的图文框格式。只读。
HanjaPhoneticHangul返回或设置一个 Boolean 类型的值，该值代表在朝鲜语查找操作中是否查找朝鲜文拼音和朝鲜文汉字字符。可读写。
Highlight 如果在查找条件中包含突出显示格式，则该属性值为 True。Long 类型，可读写。
IgnorePunct返回或设置 Boolean 值，该值表示查找操作是否应忽略找到的文本中的标点符号。可读/写。
IgnoreSpace 返回或设置 Boolean 值，该值表示查找操作是否应忽略找到的文本中的额外空格。可读/写。
LanguageID返回或设置指定 Find 对象的语言。WdLanguageID 类型，可读写。
LanguageIDFarEast返回或设置指定对象的东亚语言。可读写 WdLanguageID 类型。
LanguageIDOther返回或设置指定对象的语言。可读写 WdLanguageID 类型。
MatchAlefHamza 如果查找操作将在阿拉伯语文档中查找与“alef hamzas”相匹配的文字，则该属性值为 True。Boolean 类型，可读写。
MatchAllWordForms 如果为 True，则在查找操作需查找文本的所有形式（例如，如果要查找的单词是“sit”，那么也查找“sat”和“sitting”）。Boolean 类型，可读写。
MatchByte 如果 Microsoft Word 在搜索过程中区分全角和半角的字符或字母，则该属性值为 True。Boolean 类型，可读写。
MatchCase 如果为 True，则查找操作区分大小写。默认值为 False。Boolean. 类型，可读写。
MatchControl 如果在从右向左语言的文档中，查找操作按照匹配双向控制字符来匹配文字，则该属性值为 True。Boolean 类型，可读写。
MatchDiacritics 如果在从右向左的语言的文档中，查找操作按照匹配音调符号来匹配文字，则该属性值为 True。Boolean 类型，可读写。
MatchFuzzy 如果 Microsoft Word 在搜索过程中使用日文文本的非特定搜索选项，则该属性值为 True。Boolean 类型，可读写。
MatchKashida 如果查找操作将在阿拉伯语文档中按照匹配 kashidas 来匹配文字，则该属性值为 True。Boolean 类型，可读写。
MatchPhrase如果该属性值为 True，则忽略单词之间的所有空格和控制字符。可读写。
MatchPrefix如果该属性值为 True，则匹配以搜索字符串开头的单词。可读写。
MatchSoundsLike 如果查找操作返回与待查找文字发音相近的单词，则该属性值为 True。Boolean 类型，可读写。
MatchSuffix如果该属性值为 True，则匹配以搜索字符串结尾的单词。可读写。
MatchWholeWord 如果查找操作只查找完整单词，而不是一个长单词的一部分，则该属性值为 True。Boolean 类型，可读写。
MatchWildcards 如果要查找的文字中包含通配符，则该属性值为 True。Boolean 类型，可读写。
NoProofing 如果 Microsoft Word 查找或替换那些拼写和语法检查程序忽略的文本，则该属性值为 True。可读写 Long 类型。
ParagraphFormat返回或设置一个 ParagraphFormat 对象，该对象代表指定查找操作的段落设置。可读写。
Parent返回一个 Object 类型的值，该值代表指定 Find 对象的父对象。
Replacement返回一个 Replacement 对象，该对象包含替换操作的条件。
Style返回或设置指定对象的样式。可读写 Variant 类型。
Text返回或设置要查找的文本。String 类型，可读写。
Wrap如果从文档开头之外的位置开始搜索且到达文档末尾（或如果 Forward 设置为 False，则反之），或者如果在指定的所选内容或区域中未找到搜索文本，则返回或设置其结果。WdFindWrap 类型，可读写。
