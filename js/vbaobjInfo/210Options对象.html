Options 对象成员

全部隐藏

代表 Word 中的应用程序和文档选项。Options 对象的许多属性与“选项”对话框中的项对应。

属性

 名称说明
AddBiDirectionalMarksWhenSavingTextFile 如果 Microsoft Word 在将文档保存为文本文件时添加双向控制字符，则该属性值为 True。Boolean 类型，可读写。
AddControlCharacters 如果 Microsoft Word 在剪切和复制文本时添加双向控制字符，则该属性值为 True。Boolean 类型，可读写。
AddHebDoubleQuote 如果 Microsoft Word 会将数字格式括在双引号 (") 中，则该属性值为 True。Boolean 类型，可读写。
AllowAccentedUppercase 如果当一个法语字符改成大写字母时重音仍保留，则该属性值为 True。Boolean 类型，可读写。
AllowClickAndTypeMouse 如果启用“即点即输”功能，则该属性值为 True。Boolean 类型，可读写。
AllowCombinedAuxiliaryForms 如果 Microsoft Word 在朝鲜语文档中执行拼写检查时将忽略辅助动词形式，则该属性值为 True。Boolean 类型，可读写。
AllowCompoundNounProcessing 如果 Microsoft Word 在朝鲜语文档中执行拼写检查时将忽略复合名词，则该属性值为 True。Boolean 类型，可读写。
AllowDragAndDrop 如果可以使用拖动来移动或复制所选内容，则该属性值为 True。Boolean 类型，可读写。
AllowOpenInDraftView返回或设置 Boolean 值，该值表示是否允许用户在草稿视图中打开文档。可读/写。
AllowPixelUnits 如果 Microsoft Word 用像素作为默认度量单位（这是 HTML 功能所支持的度量单位），则该属性值为 True。Boolean 类型，可读写。
AllowReadingMode 如果为 True，则表明 Microsoft Word 将文档在“阅读版式”视图中打开。Boolean 类型，可读写。
AlwaysUseClearType返回或设置 Boolean 值，该值表示是否使用 ClearType 来显示菜单、功能区和对话框文本中的字体。可读/写。
AnimateScreenMovements 如果 Word 激活动态鼠标功能、使用动态光标，并且激活后台保存与查找和替换操作之类的动态操作效果，则该属性值为 True。Boolean 类型，可读写。
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
ApplyFarEastFontsToAscii 如果 Microsoft Word 将中文字体应用于西文文字，则该属性值为 True。Boolean 类型，可读写。
ArabicMode返回或设置阿拉伯语的拼写检查模式。WdAraSpeller 类型，可读写。
ArabicNumeral返回或设置一篇阿拉伯语文档的数字样式。WdArabicNumeral 类型，可读写。
AutoCreateNewDrawings 如果为 True，则 Microsoft Word 在绘图画布 （绘图画布：一个区域，可在该区域上绘制多个形状。因为形状包含在绘图画布内，所以它们可作为一个单元移动和调整大小。）上画出新创建的图形。Boolean 类型，可读写。
AutoFormatApplyBulletedLists 如果当 Word 自动设置文档或区域格式时，会用“格式”菜单的“项目符号和编号”对话框中的项目符号替换列表段落的开始字符（例如星号、连字符和大于号符号），则该属性值为 True。Boolean 类型，可读写。
AutoFormatApplyFirstIndents 如果为 True，则 Microsoft Word 在自动设置文档或区域的格式时，用首行缩进替换在段落开始处输入的空格。Boolean 类型，可读写。
AutoFormatApplyHeadings 如果当 Word 自动设置文档或区域格式时，将自动为标题设置样式，则该属性值为 True。Boolean 类型，可读写。
AutoFormatApplyLists 如果当 Word 自动设置文档或区域格式时，自动为列表应用样式，则该属性值为 True。Boolean 类型，可读写。
AutoFormatApplyOtherParas 如果当 Word 自动设置文档或区域格式时，自动为非标题项或列表项的段落设置样式，则该属性值为 True。Boolean 类型，可读写。
AutoFormatAsYouTypeApplyBorders 如果按下 Enter 后，三个或更多的连字符 (-)、等号 (=) 或下划线 (_) 将自动替换成指定的边框线，则该属性值为 True。Boolean 类型，可读写。
AutoFormatAsYouTypeApplyBulletedLists 如果在键入时用“格式”菜单的“项目符号和编号”对话框中的项目符号替换当前的项目符号（例如星号、连字符和大于号），则该属性值为 True。Boolean 类型，可读写。
AutoFormatAsYouTypeApplyClosings 如果为 True，则 Microsoft Word 会在键入时自动为信函的结束语应用“结束语”样式。Boolean 类型，可读写。
AutoFormatAsYouTypeApplyDates 如果为 True，则 Microsoft Word 自动为键入的日期应用“日期”样式。可读写。
AutoFormatAsYouTypeApplyFirstIndents 如果为 True，则 Microsoft Word 在键入时自动用首行缩进替代段落开始处输入的空格。可读写。
AutoFormatAsYouTypeApplyHeadings 如果在键入标题时，自动为标题设置样式，则该属性值为 True。Boolean 类型，可读写。
AutoFormatAsYouTypeApplyNumberedLists 如果根据键入的内容，以“格式”菜单的“项目符号和编号”对话框中的编号方案将段落自动设置为编号列表，则该属性值为 True。例如，如果一个段落以“1.1”和一个制表符开始，则当按下 Enter 后 Word 将自动插入“1.2”和一个制表符。Boolean 类型，可读写。
AutoFormatAsYouTypeApplyTables 如果为 True，则当您键入一个加号、一串连字符、另一个加号等，然后按下 Enter 后，Word 将自动创建一张表格。加号变为列边框，连字符变为列宽度。Boolean 类型，可读写。
AutoFormatAsYouTypeAutoLetterWizard 如果为 True，则在用户输入信函称呼或结束语时，Microsoft Word 会自动启动“英文信函向导”。可读写。
AutoFormatAsYouTypeDefineStyles 如果 Word 自动根据手动设置的格式新建一种样式，则该属性值为 True。Boolean 类型，可读写。
AutoFormatAsYouTypeDeleteAutoSpaces 如果为 True，则 Microsoft Word 会在用户键入时自动删除日文和拉丁文文字之间插入的空格。可读写。
AutoFormatAsYouTypeFormatListItemBeginning 如果 Word 将列表项开始处的字符格式重复应用于下一个列表项，则该属性值为 True。Boolean 类型，可读写。
AutoFormatAsYouTypeInsertClosings 如果为 True，则当用户输入备忘录标题时，Microsoft Word 会自动插入相应的备忘录结束语。可读写。
AutoFormatAsYouTypeInsertOvers 如果为 True，则当用户输入“ ”或“ ”时，Microsoft Word 将自动插入“ ”。Boolean 类型，可读写。
AutoFormatAsYouTypeMatchParentheses 如果为 True，Microsoft Word 会自动更正不正确成对的括号，可读写。
AutoFormatAsYouTypeReplaceFarEastDashes 如果为 True，Microsoft Word 会自动更正长元音和划线，可读写。
AutoFormatAsYouTypeReplaceFractions 如果在您键入时，将使用当前字符集的分数替换键入的分数，则该属性值为 True。例如，将“1/2”替换为“?”。Boolean 类型，可读写。
AutoFormatAsYouTypeReplaceHyperlinks 如果本属性值为 True，则 Word 会在键入时自动将电子邮件地址、服务器和共享名称（即 UNC 路径）和 Internet 地址（即 URL）改为超链接。Boolean 类型，可读写。
AutoFormatAsYouTypeReplaceOrdinals 如果为 True，则键入时，Word 会用相同字母的上标格式替换序数后缀“st”、“nd”、“rd”和“th”。例如，“1st”被替换为“1”后接上标“st”。Boolean 类型，可读写。
AutoFormatAsYouTypeReplacePlainTextEmphasis 如果键入时，手动强调的字符将自动替换为相应字符格式，则该属性值为 True。例如，将“*bold*”改为“bold”、将“_underline_”改为“underline”。Boolean 类型，可读写。
AutoFormatAsYouTypeReplaceQuotes 如果键入时自动将所键入的直引号替换为弯引号，则该属性值为 True。Boolean 类型，可读写。
AutoFormatAsYouTypeReplaceSymbols 如果键入的两个连续的连字符 (--) 将替换为短划线 (–) 或长划线 (—)，则该属性值为 True。Boolean 类型，可读写。如果键入连字符时带有前导和尾部空格，则 Word 将连字符替换为短划线。如果没有尾部空格，则连字符替换为长划线。
AutoFormatDeleteAutoSpaces 如果 Microsoft Word 在自动设置文档或区域的格式时，删除日文和西文文字之间插入的空格，则该属性值为 True。Boolean 类型，可读写。
AutoFormatMatchParentheses 如果 Microsoft Word 在自动设置文档或区域的格式时，会更正不恰当匹配的括号，则该属性值为 True。Boolean 类型，可读写。
AutoFormatPlainTextWordMail 如果在 Word 中打开纯文本格式的电子邮件时，Word 将自动进行格式设置，则该属性值为 True。Boolean 类型，可读写。
AutoFormatPreserveStyles 如果当 Word 自动设置文档或区域格式时，保留先前所用的样式，则该属性值为 True。Boolean 类型，可读写。
AutoFormatReplaceFarEastDashes 如果 Microsoft Word 在自动设置文档或区域的格式时，会更正长元音和划线的使用，则该属性值为 True。Boolean 类型，可读写。
AutoFormatReplaceFractions 如果当 Word 自动设置文档或区域格式时，会用当前字符集中的分数替换所键入的分数，则该属性值为 True。例如，将“1/2”替换为“?”。Boolean 类型，可读写。
AutoFormatReplaceHyperlinks 如果为 True，则 Word 在对某一文档或范围自动套用格式时，将对其中的电子邮件地址、服务器和共享名称（即 UNC 路径）和 Internet 地址（即 URL）自动进行格式设置。Boolean 类型，可读写。
AutoFormatReplaceOrdinals 如果为 True，则当 Word 自动设置文档或区域格式时，用相同字母的上标格式替换序数后缀“st”、“nd”、“rd”和“th”。例如，将“1st”替换为“1”后接上标“st”。Boolean 类型，可读写。
AutoFormatReplacePlainTextEmphasis 如果当 Word 自动设置文档或区域的格式时，将手动强调的字符替换为相应的字符格式（例如，“*bold*”更改为“bold”，“_underline_”更改为“underline”），则该属性值为 True。Boolean 类型，可读写。
AutoFormatReplaceQuotes 如果当 Word 自动设置文档或区域格式时，直引号会自动更改为弯引号，则该属性值为 True。Boolean 类型，可读写。
AutoFormatReplaceSymbols 如果当 Word 自动设置文档或区域的格式时，将两个连续的连字号 (--) 替换为短破折号 (–) 或长破折号 (—)，则该属性值为 True。Boolean 类型，可读写。
AutoKeyboardSwitching 如果 Microsoft Word 随时自动切换键盘语言并与键入的内容匹配，则该属性值为 True。可读/写 Boolean 类型。
AutoWordSelection 如果在拖动时，一次选定一个单词，而不是选定一个字符，则该属性为 True。Boolean 类型，可读写。
BackgroundSave 如果为 True，则 Word 在后台保存文档。当 Word 在后台保存文档时，用户可以继续键入内容或执行命令。Boolean 类型，可读写。
BibliographySort返回或设置一个 String 类型的值，该值代表在“源管理器”对话框中显示源的次序。可读写。
BibliographyStyle返回或设置一个 String 类型的值，该值代表用于设置书目格式的样式的名称。可读写。
BrazilReform返回或设置巴西葡萄牙语的拼写检查模式。可读写。
ButtonFieldClicks返回或设置一个数字，该数字是运行 GOTOBUTTON 或 MACROBUTTON 域所需的点击次数（单击或双击）。Long 类型，可读写。
CheckGrammarAsYouType 如果 Word 在键入时自动对文档进行语法检查并自动标记错误，则该属性值为 True。Boolean 类型，可读写。
CheckGrammarWithSpelling 如果 Word 在检查拼写的同时也检查语法，则该属性值为 True。Boolean 类型，可读写。
CheckHangulEndings 如果 Microsoft Word 自动检测朝鲜文字结尾，并在将朝鲜文字转换为朝鲜文汉字过程中忽略朝鲜文字结尾，则该属性值为 True。Boolean 类型，可读写。
CheckSpellingAsYouType 如果 Microsoft Word 在您键入的同时自动进行拼写检查并标记错误，则该属性值为 True。Boolean 类型，可读写。
CommentsColor该属性返回或设置一个 WdColorIndex 常量，代表文档中批注的颜色。可读写。
ConfirmConversions 如果 Word 在打开或者插入一个非 Word 文档或模板的文件之前，会显示一个“转换文件”对话框，则该属性值为 True。在“转换文件”对话框中，用户可以选择需要转换的文件格式。Boolean 类型，可读写。
ContextualSpeller返回或设置 Boolean 值，该值表示是否使用上下文拼写检查器，基于单词上下文和它前后的单词来检查拼写。可读/写。
ConvertHighAnsiToFarEast 如果在打开文档时，Microsoft Word 将与东亚字体相关的文字转换为相应的字体，则该属性值为 True。Boolean 类型，可读写。
CreateBackup 如果 Word 在每次保存文档时都建立一个备份，则该属性值为 True。Boolean 类型，可读写。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
CtrlClickHyperlinkToOpen 如果 Microsoft Word 需要单击同时按住 Ctrl 键来打开超链接，则该属性值为 True。Boolean 类型，可读写。
CursorMovement返回或设置在双向文字中插入点移动的方式。WdCursorMovement 类型，可读写。
DefaultBorderColor返回或设置用于新的 Border 对象的默认 24 位颜色。可读写。
DefaultBorderColorIndex返回或设置边框的默认线条颜色。WdColorIndex 类型，可读写。
DefaultBorderLineStyle返回或设置默认边框的样式。WdLineStyle 类型，可读写。
DefaultBorderLineWidth返回或设置边框的默认线条宽度。WdLineWidth 类型，可读写。
DefaultEPostageApp设置或返回一个 String 类型的值，该值代表默认的电子邮政应用程序的路径和文件名。可读写。
DefaultFilePath返回或设置各项（例如文档、模板和图形）的默认文件夹。String 类型，可读写。
DefaultHighlightColorIndex返回或设置突出显示文本所用的颜色，用“格式”工具栏的“突出显示”按钮可将文本格式设为突出显示。WdColorIndex 类型，可读写。
DefaultOpenFormat返回或设置打开文档的默认转换器。可为由 OpenFormat 属性返回的一个数字，也可为下列 WdOpenFormat 常量之一。
DefaultTextEncoding返回或设置一个 MsoEncoding 常量，该常量代表 Microsoft Word 用于所有保存为编码文本文件的文档的代码页或字符集。可读写。
DefaultTray返回或设置默认纸盒，以便打印机使用该纸盒打印文档。String 类型，可读写。
DefaultTrayID返回或设置默认纸盒，以便打印机使用该纸盒打印文档。WdPaperTray 类型，可读写。
DeletedCellColor返回或设置一个 WdCellColor 常量，该常量代表删除的单元格的颜色。可读写。
DeletedTextColor当激活修订选项时，返回或设置要删除的文字的颜色。WdColorIndex 类型，可读写。
DeletedTextMark当激活修订选项时，返回或设置要删除的文字的格式。WdDeletedTextMark 类型，可读写。
DiacriticColorVal返回或设置用于从右向左语言文档中的音调符号的 24 位颜色。
DisableFeaturesbyDefault 如果为 True，则 Microsoft Word 将在所有文档中禁用 DisableFeaturesIntroducedAfterbyDefault 中指定的 Word 版本后引入的所有功能。默认值为 False。Boolean 类型，可读写。
DisableFeaturesIntroducedAfterbyDefault对所有文档禁用指定版本后引入的所有功能。WdDisableFeaturesIntroducedAfter 类型，可读写。
DisplayGridLines 如果 Microsoft Word 显示文档网格，则该属性值为 True。该属性相当于“视图”菜单中的“网格线”命令。Boolean 类型，可读写。
DisplayPasteOptions 如果为 True，则 Microsoft Word 直接在新粘贴的文本下显示“粘贴选项”按钮。Boolean 类型，可读写。
DocumentViewDirection返回或设置整篇文档的对齐方式和阅读顺序。WdDocumentViewDirection 类型，可读写。
DoNotPromptForConvert设置或返回一个 Boolean，它代表在为处于兼容模式下的文档调用“转换”命令时是否出现一个警告对话框。可读写。
EnableHangulHanjaRecentOrdering 如果 Microsoft Word 在进行朝鲜文字和朝鲜文汉字间的转换时，在建议列表的顶端显示最近用过的单词，则该属性值为 True。Boolean 类型，可读写。
EnableLegacyIMEMode返回或设置一个 Boolean 类型的值，该值代表是否启用旧式 IME 模式。可读写。
EnableLivePreview设置或返回 Boolean 值，该值表示显示还是隐藏在使用支持预览的库时所出现的库预览。如果为 True，将在应用命令之前在文档中显示预览。可读/写。
EnableMisusedWordsDictionary 如果 Microsoft Word 在检查文档的拼写和语法时，也检查误用的单词，则该属性值为 True。Boolean 类型，可读写。
EnableSound 如果 Word 在导致出错时使计算机发出声音进行响应，则该属性值为 True。Boolean 类型，可读写。
EnvelopeFeederInstalled 如果当前的打印机为信封指定了送纸器，则该属性值为 True。Boolean 类型，只读。
FormatScanning 如果为 True，则 Microsoft Word 对文档中的所有格式进行跟踪。Boolean 类型，可读写。
FrenchReform返回或设置一个 WdFrenchSpeller 常量，该常量代表要对语言格式设置为法语的文本区域使用哪个拼写词典。可读写。
GridDistanceHorizontal当在新文档中绘制、移动和调整自选图形或东亚语言字符时，返回或设置 Word 使用的不可见网格线的水平间距。Single 类型，可读写。
GridDistanceVertical在新文档中绘制、移动和调整自选图形或东亚语言字符时，返回或设置 Word 所用的不可见网格线间的垂直间距。Single 类型，可读写。
GridOriginHorizontal返回或设置不可见网格相对于页面左边的起点位置，当在新文档中绘制、移动和调整自选图形或东亚语言字符时将使用该网格。Single 类型，可读写。
GridOriginVertical返回或设置不可见网格相对于页面顶边的起点位置，当在新文档中绘制、移动和调整自选图形或东亚语言字符时将使用该网格。Single 类型，可读写。
HangulHanjaFastConversion 如果 Microsoft Word 在朝鲜文字和朝鲜文汉字间进行转换时，自动转换具有单独建议的单词，则该属性值为 True。Boolean 类型，可读写。
HebrewMode返回或设置希伯来语拼写检查工具的模式。WdHebSpellStart 类型，可读写。
IgnoreInternetAndFileAddresses 如果为 True，则进行拼写检查时忽略文件扩展名、MS-DOS 路径、电子邮件地址、服务器名称、共享名（也称为 UNC 路径）和 Internet 地址（也称为 URL）。Boolean 类型，可读写。
IgnoreMixedDigits 如果进行拼写检查时忽略包含数字的单词，则该属性值为 True。Boolean 类型，可读写。
IgnoreUppercase 如果进行拼写检查时忽略全部大写的单词，则该属性值为 True。Boolean 类型，可读写。
IMEAutomaticControl 如果设置 Microsoft Word 自动打开和关闭日文输入法编辑器 (IME)，则该属性值为 True。Boolean 类型，可读写。
InlineConversion 如果 Microsoft Word 将中文输入法 (IME) 中的未确认字符串显示为已有（已确认的）字符串之间的插入项，则该属性值为 True。Boolean 类型，可读写。
InsertedCellColor返回或设置一个 WdCellColor 常量，该常量代表插入的表格单元格的颜色。可读写。
InsertedTextColor当启用修订选项时，返回或设置插入文字的颜色。WdColorIndex 类型，可读写。
InsertedTextMark返回或设置启用修订时 Microsoft Word 设置插入文本格式的方式（TrackRevisions 属性为 True）。WdInsertedTextMark 类型，可读写。
INSKeyForOvertype如果可以用 Ins 键来打开和关闭“改写”，则该属性值为 True。Boolean 类型，可读写。
INSKeyForPaste 如果可用 INS 键来粘贴“剪贴板”的内容，则该属性值为 True。Boolean 类型，可读写。
