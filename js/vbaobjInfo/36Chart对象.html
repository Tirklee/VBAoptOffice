Chart 对象成员

代表文档中的图表。

方法

 名称说明
ApplyChartTemplate将标准图表类型或自定义图表类型应用于图表。
ApplyDataLabels将数据标签应用于图表中的所有系列。
ApplyLayout应用功能区中显示的版式。
Axes返回图表中的坐标轴集合。
ChartWizard修改给定图表的属性。可使用本方法快速设置图表的格式，而不必逐个设置所有属性。本方法是非交互式的，并且仅更改指定的属性。
ClearToMatchStyle清除图表元素格式以改为自动格式。
Copy此对象不支持的操作。
CopyPicture将所选对象作为图片复制到剪贴板。
Delete删除对象。
Export以图形格式导出图表。
GetChartElement返回指定 x 坐标和 y 坐标处的图表元素的相关信息。
Paste将剪贴板中的图表数据粘贴到图表中。
Refresh立即重新绘制指定的图表。
SaveChartTemplate向可用图表模板的列表中添加自定义图表模板。
Select选择对象。
SeriesCollection返回图表中的所有系列。
SetBackgroundPicture设置图表的背景图形。
SetDefaultChart指定 Microsoft Word 新建图表时使用的图表模板的名称。
SetElement设置图表上的图表元素。可读/写 MsoChartElementType 类型。
SetSourceData设置图表的源数据范围。

属性

 名称说明
Application如果不与对象识别符一起使用，则该属性返回一个代表 Microsoft Word 应用程序的 Application 对象。如果与对象识别符一起使用，则该属性返回代表指定对象的创建者的 Application 对象（可对自动操作对象使用该属性来返回该对象的应用程序）。只读。
AutoScaling 如果 Microsoft Word 自动对三维图表进行缩放，使之与等效的二维图表的大小相近，则为 True。RightAngleAxes 属性必须为 True。可读/写 Boolean 类型。
BackWall返回允许用户单独对三维图表的背景墙进行格式设置的对象。只读 Walls 对象。
BarShape返回或设置用于三维条形图或柱形图中每个系列的形状。可读/写 XlBarShape。
ChartArea返回图表的整个图表区。只读 ChartArea 对象。
ChartData返回与图表相关联的链接数据或嵌入数据的有关信息。只读 ChartData 对象。
ChartGroups返回一个对象，它代表图表中单个图表组或所有图表组的集合。
ChartStyle返回或设置图表的图表样式。可读/写 Variant 类型。
ChartTitle返回指定图表的标题。只读 ChartTitle 对象。
ChartType返回或设置图表类型。可读/写 XlChartType。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
DataTable返回图表模拟运算表。只读 DataTable 对象。
DepthPercent返回或设置三维图表的深度，以图表宽度的百分比表示（有效范围从 20% 到 2000%）。可读/写 Long 类型。
DisplayBlanksAs返回或设置在图表中绘制空白单元格的方式。可以是 XlDisplayBlanksAs 常量之一。可读/写 Long 类型。
Elevation以度为单位返回或设置三维图表视图的上下仰角。可读/写 Long 类型。
Floor返回三维图表的基底。只读 Floor 对象。
GapDepth返回或设置三维图表中各数据系列之间的距离，以数据标记宽度的百分比表示。可读/写 Long 类型。
HasAxis返回或设置图表上显示的坐标轴。可读/写 Variant 类型。
HasDataTable 如果图表有模拟运算表，则该属性值为 True。可读/写 Boolean 类型。
HasLegend 如果图表有图例，则该属性值为 True。可读/写 Boolean 类型。
HasTitle 如果坐标轴或图表有可见标题，则为 True。可读/写 Boolean 类型。
HeightPercent以图表宽度的百分比形式返回或设置三维图表的高度（值范围从 5% 到 500%）。可读/写 Long 类型。
Legend返回图表的图例。只读 Legend 对象。
Parent返回指定对象的父对象。只读 Object 对象。
Perspective返回或设置三维图表视图的透视系数。可读/写 Long 类型。
PivotLayout此对象不支持的操作。
PlotArea返回图表的绘图区。只读 PlotArea 对象。
PlotBy返回或设置列或行在图表中作为数据系列使用的方式。可读/写 Long 类型。
PlotVisibleOnly 如果仅绘制可见单元格，则该值为 True。如果可见单元格和隐藏单元格都绘制，则该值为 False。可读/写 Boolean 类型。
RightAngleAxes 如果图表的坐标轴为直角，与图表的转角或仰角无关，则为 True。可读/写 Boolean 类型。
Rotation以度为单位返回或设置三维图表视图的转角（绘图区绕 Z 轴的转角）。可读/写 Variant 类型。
Shapes返回代表图表工作表中所有形状的集合。只读 Shapes 对象。
ShowAllFieldButtons返回或设置是否在数据透视图上显示所有域按钮。可读写。已弃用。
ShowAxisFieldButtons返回或设置是否在数据透视图上显示坐标轴域按钮。可读写。已弃用。
ShowDataLabelsOverMaximum返回或设置表示在数值大于数值轴上的最大值时是否显示数据标签的值。可读/写 Boolean 类型。
ShowLegendFieldButtons返回或设置是否在数据透视图上显示图例域按钮。可读写。已弃用。
ShowReportFilterFieldButtons返回或设置是否在数据透视图上显示报表筛选域按钮。可读写。已弃用。
ShowValueFieldButtons返回或设置是否在数据透视图上显示值域按钮。可读写。已弃用。
SideWall返回一个 Walls 对象，该对象允许用户单独对三维图表的侧面墙进行格式设置。只读。
Walls返回三维图表的背景墙。只读 Walls 对象。
