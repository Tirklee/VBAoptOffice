FillFormat 对象成员

代表形状的填充格式。形状可以有单色、渐变、纹理、图案、图片或半透明填充。

方法

 名称说明
OneColorGradient将指定填充设置为单色渐变。
Patterned将指定填充设置为一种图案。
PresetGradient将指定填充设置为预设渐变。
PresetTextured将指定填充设置为预设纹理。
Solid将指定填充设置为统一的颜色。
TwoColorGradient设置指定填充为双色渐变。
UserPicture用一张大图像填充指定形状。
UserTextured用一张图像的平铺形式填充指定形状。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
BackColor返回或设置一个 ColorFormat 对象，该对象代表填充的背景色。可读写。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
ForeColor返回或设置一个 ColorFormat 对象，该对象代表填充的前景色。可读写。
GradientAngle返回或设置指定填充格式的渐变填充角度。可读写。
GradientColorType返回指定填充的渐变颜色的样式。只读 MsoGradientColorType 类型。
GradientDegree返回值表明单色渐变填充的深浅程度。Single 类型，只读。
GradientStops返回与指定的填充格式相关的 GradientStops 集合。只读。
GradientStyle返回指定填充的渐变样式。只读 MsoGradientStyle 类型。
GradientVariant返回指定填充的渐变变形；对大多数渐变填充来说，其返回值为 1 到 4 之间的整数。Long 类型，只读。
Parent返回一个 Object 类型的值，该值代表指定 FillFormat 对象的父对象。
Pattern返回或设置一个 MsoPatternType 常量，该常量代表应用于指定填充或线条的图案。只读。
PictureEffects返回一个 PictureEffects 对象，该对象可用于将图片效果应用于指定的填充格式。只读。
PresetGradientType返回指定填充的预设渐变类型。只读 MsoPresetGradientType 类型。
PresetTexture返回指定填充的预设纹理。只读 MsoPresetTexture 类型。
RotateWithObject返回或设置填充是否与指定形状一起旋转。可读/写。
TextureAlignment返回或设置用于平铺纹理填充的对齐方式（坐标网格的原点）。可读/写。
TextureHorizontalScale返回或设置一个 Single 值，该值指定纹理填充的水平缩放因子。可读写。
TextureName返回指定填充的自定义纹理文件的名称。只读 String 类型。
TextureOffsetX返回或设置一个 Long 值，该值指定纹理距原点的水平偏移量（磅）。可读写。
TextureOffsetY返回或设置一个 Long 值，该值指定纹理距原点的垂直偏移量（磅）。可读写。
TextureTile返回或设置一个值，该值指示纹理填充是平铺还是居中。可读/写。
TextureType返回指定填充的纹理类型。MsoTextureType 类型，只读。
TextureVerticalScale返回或设置一个 Single 值，该值指定纹理填充的垂直缩放因子。可读写。
Transparency返回或设置形状的指定填充的透明度，可以是 0.0（不透明）到 1.0（透明）之间的数值。可读写 Single 类型。
Type返回图形填充格式的类型。只读 MsoFillType 类型。
Visible如果指定对象或应用于该对象的格式是可见的，则该属性值为 True。MsoTriState 类型，可读写。
