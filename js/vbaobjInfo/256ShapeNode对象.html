ShapeNode 对象成员
全部显示
全部隐藏

代表用户定义的任意多边形的顶点的几何图形以及几何图形编辑属性。顶点包括任意多边形的线段之间的顶点以及曲线段的控制点。ShapeNode 对象是 ShapeNodes 集合的一个成员。ShapeNodes 集合包含任意多边形的所有顶点。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
EditingType如果指定的节点为顶点，则该属性返回一个值，该值表示对节点所做的更改将如何影响与该节点相连的两个线段。MsoEditingType 类型，只读。
Parent返回一个 Object 类型的值，该值代表指定 ShapeNode 对象的父对象。
Points以坐标对 （坐标对：一对值，表示两维数组中存储的点的 x 和 y 坐标，该数组中包含许多点的坐标。）的形式返回指定节点的位置。Variant 类型，只读。
SegmentType返回一个值，该值代表与指定节点相关联的线段是直线段还是曲线段。MsoSegmentType 类型，只读。
