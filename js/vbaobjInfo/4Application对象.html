Application 对象成员

全部隐藏

代表 Microsoft Word 应用程序。Application 对象包含可返回顶级对象的属性和方法。例如，ActiveDocument 属性返回 Document 对象。

方法

 名称说明
Activate激活指定的对象。
AddAddress本方法在通讯簿中新增一项。每项都有一个或多个标签 ID 值。
AutomaticChange当“Office 助手”建议进行更改时，将执行“自动套用格式”操作。如果未激活任何“自动套用格式”操作，该方法将导致出错。
BuildKeyCode为指定的组合键返回唯一的代码。
CentimetersToPoints将度量单位由厘米转换为磅数（1 厘米=28.35 磅）。所返回的转换结果为 Single 类型。
ChangeFileOpenDirectory设置查找文档时 Word 所指向的文件夹。
CheckGrammar检查字符串的语法错误。返回一个 Boolean 类型的值，以表示字符串中是否包含语法错误。如果字符串未包含语法错误，则该值为 True。
CheckSpelling检查字符串的拼写错误。返回一个 Boolean 类型的值，以显示字符串是否包含拼写错误。如果字符串未包含拼写错误，则该值为 True。
CleanString从指定字符串中删除非打印字符（字符代码为 1-29）及 Word 的特殊字符，或将它们替换为空格（字符代码为 32）。返回的结果为 String 类型。
CompareDocuments 比较两个文档，并返回一个 Document 对象，该对象表示包含两个文档之间的差别的文档，该文档用修订进行标记。
DDEExecute通过指定的 DDE（即动态数据交换）通道，向应用程序发送一条或一组命令。
DDEInitiate打开通向其他应用程序的 DDE（动态数据交换）通道，并返回通道序号。
DDEPoke通过打开的 DDE（动态数据交换）通道向应用程序发送数据。
DDERequest通过打开的 DDE（动态数据交换）通道向接收应用程序查询信息，并以 String 类型返回该信息。
DDETerminate关闭指定的通向另一应用程序的 DDE（动态数据交换）通道。
DDETerminateAll关闭所有由 Microsoft Word 打开的动态数据交换 (DDE) 通道。
DefaultWebOptions返回 DefaultWebOptions 对象，该对象包含应用程序级的全局属性，每当将文档另存为网页或打开网页时，Microsoft Word 就会使用这些属性。
GetAddress返回默认通讯簿中的地址。
GetDefaultTheme返回代表默认主题 （主题：格式的集合，用于确定数据访问页上控件、节、正文和文本的外观。）名称的 String，以及 Microsoft Word 用于新文档、电子邮件或网页的主题格式选项。
GetSpellingSuggestions返回一个 SpellingSuggestions 集合，该集合代表指定单词的建议拼写替换单词。
GoBack将插入点在活动文档中最近进行过编辑操作的三个位置间移动，其作用等同于按 Shift+F5。
GoForward将插入点在活动文档中进行编辑的最后三个位置之间向前移动。
Help显示安装的帮助信息。
HelpTool指针从箭头改为问号，表明这是关于将要单击的命令或屏幕元素的快捷帮助。如果单击文本，Word 将显示描述当前段落及字符格式的对话框。按 Esc 可将指针恢复为原始状态。
InchesToPoints将度量单位从英寸转换为磅（1 英寸 = 72 磅）。将转换后的度量单位以 Single 类型返回。
Keyboard返回或设置键盘的语言和布局设置。
KeyboardBidi将键盘语言设置为从右向左的语言，并将文字的输入方向设置为从右向左。
KeyboardLatin将键盘语言设置为从左向右的语言，并将文字的输入方向设置为从左向右。
KeyString返回指定键的组合键字符串（例如，Ctrl+Shift+A）。
LinesToPoints将度量单位从行转换为磅（1 行 = 12 磅）。将转换后的度量单位以 Single 类型返回。
ListCommands新建文档，然后插入带有相关快捷键和菜单设定的 Word 命令表。
LoadMasterList 加载书目源文件。
LookupNameProperties在全局通讯簿列表中查找姓名，并显示“属性”对话框，该对话框中包含有关指定姓名的信息。
MergeDocuments 比较两个文档，并返回一个 Document 对象，该对象表示包含两个文档之间的差别的文档，该文档用修订进行标记。
MillimetersToPoints将度量单位从毫米转换为磅（1 毫米 = 2.85 磅）。将转换后的度量单位以 Single 类型返回。
Move设置任务窗口或活动文档窗口的位置。
NewWindow打开一个新窗口，该窗口与指定窗口显示同一个文档。返回一个 Window 对象。
NextLetter您已经就仅在 Macintosh 上使用的 Visual Basic 关键字请求帮助。有关 Application 对象的 NextLetter 方法的信息，请参阅 Microsoft Office Macintosh Edition 所附带的语言参考帮助。
OnTime启动在指定的时间运行宏的后台计时器。
OrganizerCopy将指定的“自动图文集”词条、工具栏、样式或宏方案项从源文档或模板中复制到目标文档或模板上。
OrganizerDelete从文档或模板中删除指定的样式、“自动图文集”词条、工具栏或宏方案项。
OrganizerRename重新命名文档或模板中指定的样式、“自动图文集”词条、工具栏或宏方案项。
PicasToPoints将度量单位从十二点活字转换为磅（1 十二点活字 = 12 磅）。将转换后的度量单位以 Single 类型返回。
PixelsToPoints将度量单位从像素转换为磅。将转换后的度量单位以 Single 类型返回。
PointsToCentimeters将度量单位从磅转换为厘米（1 厘米 = 28.35 磅）。将转换后的度量单位以 Single 类型返回。
PointsToInches将磅转换为英寸（1 英寸 = 72 磅）。以 Single 类型返回转换结果。
PointsToLines将度量单位从磅转换为行（1 行 = 12 磅）。将转换后的度量单位以 Single 类型返回。
PointsToMillimeters将度量单位从磅转换为毫米（1 毫米 = 2.835 磅）。将转换后的度量单位以 Single 类型返回。
PointsToPicas将度量单位从磅转换为十二点活字（1 十二点活字 = 12 磅）。将转换后的度量单位以 Single 类型返回。
PointsToPixels将度量单位从磅转换为像素。将转换后的度量单位以 Single 类型返回。
PrintOut打印指定文档的全部或部分内容。
ProductCode返回一个 String 类型的 Microsoft Word 全球唯一标识符 (GUID)。
PutFocusInMailHeader如果活动窗口中的文档是电子邮件文档，则将插入点置于邮件标题的“收件人”行。
Quit退出 Microsoft Word，并可选择保存或传送打开的文档。
Repeat重复一次或多次最近的编辑操作。如果成功地重复了命令，则返回 True。
ResetIgnoreAll在拼写检查时，取消以前的忽略词汇表。
Resize调整 Word 应用程序或某一任务的窗口大小。
Run运行 Visual Basic 宏。
ScreenRefresh使用视频内存缓冲区的当前信息更新显示器的显示。
SetDefaultTheme为 Microsoft Word 设置用于新文档、电子邮件或网页的默认主题 （主题：格式的集合，用于确定数据访问页上控件、节、正文和文本的外观。）。
ShowClipboard显示“剪贴板”任务窗格。
ShowMe当存在更多可用信息时，显示“Office 助手”或“帮助”窗口。
SubstituteFont设置字体映射选项。
ToggleKeyboard在从左向右的语言和从右向左语言之间切换键盘语言设置。

属性

 名称说明
ActiveDocument返回一个 Document 对象，该对象代表活动文档（焦点所在的文档）。如果没有打开的文档，则会出现错误。只读。
ActiveEncryptionSession返回一个 Long 类型的值，该值代表与活动文档相关的加密会话。只读。
ActivePrinter返回或设置活动打印机名称。可读/写 String 类型。
ActiveProtectedViewWindow返回一个 ProtectedViewWindow 对象，该对象代表活动的“受保护的视图”窗口。只读。
ActiveWindow返回一个 Window 对象，该对象代表活动窗口（焦点所在的窗口）。如果没有打开的窗口，则会出现错误。只读。
AddIns返回一个 AddIns 集合，该集合代表所有可用的加载项，而不考虑当前是否已加载它们。只读。
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
ArbitraryXMLSupportAvailable返回一个 Boolean 类型的值，代表 Microsoft Word 是否接受自定义的 XML 架构。值为 True 表明 Word 接受自定义的 XML 架构。
Assistance返回一个代表 Microsoft Office 帮助查看器的 Assistance 对象。只读。
AutoCaptions返回一个 AutoCaptions 集合，该集合代表在将表格和图片等项目插入文档时自动添加的题注。只读。
AutoCorrect返回一个 AutoCorrect 对象，该对象包含当前“自动更正”的选项、词条和例外项。只读。
AutoCorrectEmail返回一个 AutoCorrect 对象，该对象代表对电子邮件进行的自动更正。
AutomationSecurity返回或设置一个 MsoAutomationSecurity 常量，该常量代表当用编程方法打开文件时 Microsoft Word 所使用的安全设置。
BackgroundPrintingStatus返回后台打印队列中的打印任务数目。Long 类型，只读。
BackgroundSavingStatus返回后台保存队列中的文件数。Long 类型，只读。
Bibliography返回一个 Bibliography 对象，该对象代表 Microsoft Word 中存储的书目参考源。只读。
BrowseExtraFileTypes如果将该属性设置为“text/html”，则用 Microsoft Word（而不是默认的 Internet 浏览器）可以打开超链接的 HTML 文件。String 类型，可读写。
Browser返回一个 Browser 对象，该对象代表垂直滚动条上的“选择浏览对象”工具。只读。
Build返回 Word 应用程序的版本号及编译序号。String 类型，只读。
CapsLock 如果已打开 Caps Lock 键，则该属性值为 True。Boolean 类型，只读。
Caption返回或设置出现在应用程序窗口的标题栏中的文本。String 类型，可读写。
CaptionLabels返回一个 CaptionLabels 集合，该集合代表所有可用的题注标签。只读。
CheckLanguage 如果 Microsoft Word 在键入时自动检测使用的语言，则该属性值为 True。Boolean 类型，可读写。
COMAddIns返回对 COMAddIns 集合的引用，该集合代表 Microsoft Word 中当前加载的所有组件对象模型 (COM) 加载项 （COM 加载项：通过添加自定义命令和指定的功能来扩展 Microsoft Office 程序的功能的补充程序。COM 加载项可在一个或多个 Office 程序中运行。COM 加载项使用文件扩展名 .dll 或 .exe。）。
CommandBars返回一个 CommandBars 集合，该集合代表 Microsoft Word 中的菜单栏以及所有工具栏。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
CustomDictionaries返回一个 Dictionaries 对象，该对象代表活动自定义词典的集合。只读。
CustomizationContext返回或设置一个 Template 或 Document 对象，该对象代表存储菜单栏、工具栏和键绑定的更改的模板或文档。可读写。
DefaultLegalBlackline 如果为 True，则 Microsoft Word 使用“比较并合并文档”对话框中的“精确比较”选项比较和合并文档。Boolean 类型，可读写。
DefaultSaveFormat返回或设置将在“另存为”对话框上的“保存类型”框中显示的默认格式。String 类型，可读写。
DefaultTableSeparator返回或设置一个字符；在将文本转换为表格时，该字符用来将文本分隔为单元格。String 类型，可读写。
Dialogs返回一个 Dialogs 集合，该集合代表 Word 中的所有内置对话框。只读。
DisplayAlerts返回或设置运行宏时产生的一些警告和消息的处理方式。WdAlertLevel 类型，可读写。
DisplayAutoCompleteTips 如果 Word 显示有关所键入整个单词、日期或词组的建议的提示，则该属性值为 True。Boolean 类型，可读写。
DisplayDocumentInformationPanel返回或设置 Boolean 值，该值表示是否显示文档属性面板。可读/写。
DisplayRecentFiles 如果在“文件”菜单中显示最近使用的文件名，则该属性值为 True。Boolean 类型，可读写。
DisplayScreenTips 如果为 True，则批注、脚注、尾注和超链接以提示形式显示。突出显示那些标记有批注的文本。Boolean 类型，可读写。
DisplayScrollBars 如果为 True，则 Word 在至少一个文档窗口中显示滚动条；如果为 False，则在任何窗口中均不显示滚动条。Boolean 类型，可读写。
Documents返回一个 Documents 集合，该集合代表所有打开的文档。只读。
DontResetInsertionPointProperties返回或设置一个 Boolean 类型的值，该值代表 Microsoft Word 在运行其他代码后是否保持位于插入点位置的文本的格式属性。可读写。
EmailOptions返回一个 EmailOptions 对象，该对象代表用于创作电子邮件的全局首选项。只读。
EmailTemplate返回或设置一个 String 类型的值，该值代表发送电子邮件时使用的文档模板。可读写。
EnableCancelKey返回或设置 Word 处理 Ctrl+Break 用户中断的方式。WdEnableCancelKey 类型，可读写。
FeatureInstall返回或设置 Microsoft Word 将如何处理对所需功能尚未安装的方法和属性的调用。MsoReatureInstall 类型，可读写。
FileConverters返回一个 FileConverters 集合，该集合代表 Microsoft Word 可以使用的所有文件转换器。只读。
FileDialog返回一个 FileDialog 对象，该对象代表文件对话框的单个实例。
FileValidation返回或设置在 Word 打开文件前如何验证文件。可读写 MsoFileValidationMode。
FindKey返回一个 KeyBinding 对象，该对象代表指定的组合键。只读。
FocusInMailHeader 如果插入点位于电子邮件标题域中（如“收件人”域），则该属性值为 True。Boolean 类型，只读。
FontNames返回一个 FontNames 对象，该对象包含所有可用字体的名称。只读。
HangulHanjaDictionaries返回一个 HangulHanjaConversionDictionaries 集合，该集合代表所有活动的自定义转换词典。
Height返回或设置活动文档窗口的高度。Long 类型，可读写。
International返回当前的国家/地区和国际设置信息。Variant 类型，只读。
IsObjectValid 如果引用某个对象的指定变量有效，则该属性值为 True。只读 Boolean 类型。
IsSandboxed 如果应用程序窗口是“受保护的视图”窗口，则该属性值为 True。只读。
KeyBindings返回一个 KeyBindings 集合，该集合代表自定义的按键分配方案，其中包含键代码、键类别和命令。
KeysBoundTo返回一个 KeysBoundTo 对象，该对象代表分配给指定项目的所有组合键。
LandscapeFontNames返回一个 FontNames 对象，该对象包含所有可用的横向字体的名称。
Language返回一个 MsoLanguageID 常量，该常量代表为 Microsoft Word 用户界面选定的语言。
Languages返回一个 Languages 集合，该集合代表在“语言”对话框（在“工具”菜单上，单击“语言”，然后单击“设置语言”）中列出的校对语言。
LanguageSettings返回一个 LanguageSettings 对象，包含 Microsoft Word 中语言设置的信息。
Left返回或设置一个 Long 类型的值，该值代表活动文档的水平位置，以磅为单位。可读写。
ListGalleries返回一个 ListGalleries 集合，该集合代表三个列表模板库。
MacroContainer返回一个 Template 或 Document 对象，该对象代表其中存储模块（该模块中包含正在运行的过程）的模板或文档。
MailingLabel返回一个 MailingLabel 对象，该对象代表邮件标签。
MailMessage返回一个 MailMessage 对象，该对象代表活动的电子邮件。
MailSystem返回主机上安装的邮件系统（或系统）。WdMailSystem 类型，只读。
MAPIAvailable 如果已经安装了 MAPI，则该属性值为 True。Boolean 类型，只读。
MathCoprocessorAvailable 如果安装了数学协处理器并且该处理器适用于 Microsoft Word，则该属性值为 True。Boolean 类型，只读。
MouseAvailable 如果系统有可用的鼠标，则该属性值为 True。Boolean 类型，只读。
Name返回指定对象的名称。只读 String 类型。
NewDocument返回一个 NewFile 对象，该对象代表在“新建文档”任务窗格上所列的一个文档。
NormalTemplate返回一个 Template 对象，该对象代表 Normal 模板。
NumLock此属性返回 Num Lock 键的状态。如果为 True，则数字键盘可用于输入数字；如果为 False，则该键盘用于移动插入点。Boolean 类型，只读。
OMathAutoCorrect返回一个 OMathAutoCorrect 对象，该对象代表公式的自动更正条目。只读。
OpenAttachmentsInFullScreen返回或设置 Boolean 值，该值表示 Microsoft Word 是否在读取模式下打开电子邮件附件。可读写。
Options返回一个 Options 对象，该对象代表 Microsoft Word 中的应用程序设置。
Parent返回一个 Object 类型的值，该值代表指定 Application 对象的父对象。
Path返回指定对象的磁盘或 Web 路径。只读 String 类型。
PathSeparator返回用于分隔文件夹名称的字符。在 Windows 中，该属性返回一个反斜线 (\)。只读 String 类型。
PickerDialog返回一个 PickerDialog 对象，该对象提供在对话框中选择人员或数据的功能。只读。
PortraitFontNames返回一个 FontNames 对象，该对象包含所有可用的纵向字体的名称。
PrintPreview 如果当前视图是打印预览，则该属性值为 True。可读写 Boolean 类型。
ProtectedViewWindows返回一个 ProtectedViewWindows 集合，该集合代表所有“受保护的视图”窗口。只读。
RecentFiles返回一个 RecentFiles 集合，该集合代表最近访问过的文件。
RestrictLinkedStyles返回或设置 Boolean 值，该值表示 Microsoft Word 是否允许链接样式。可读写。
ScreenUpdating 如果启用屏幕更新，则该属性值为 True。Boolean 类型，可读写。
Selection返回一个 Selection 对象，该对象代表一个选定范围或插入点。只读。
ShowStartupDialog 如果为 True，则在启动 Microsoft Word 时显示“任务窗格”。Boolean 类型，可读写。
ShowStylePreviews返回或设置 Boolean 值，该值表示 Microsoft Word 是否在“样式”对话框中显示样式的格式预览。可读/写。
ShowVisualBasicEditor 如果显示“Visual Basic 编辑器”窗口，则该属性值为 True。可读写 Boolean 类型。
ShowWindowsInTaskbar 如果为 True，则在任务栏中显示打开的文档，默认值为 Single Document Interface (SDI)。如果为 False，则只在“窗口”菜单中列出打开的文档，提供 Multiple Document Interface (MDI) 的外观。Boolean 类型，可读写。
SmartArtColors返回一个 SmartArtColors 对象，该对象代表当前在应用程序中加载的颜色样式集。只读。
SmartArtLayouts返回一个 SmartArtLayouts 对象，该对象代表当前在应用程序中加载的 SmartArt 布局集。只读。
SmartArtQuickStyles返回一个 SmartArtQuickStyles 对象，该对象代表当前在应用程序中加载的 SmartArt 样式集。只读。
SpecialMode 如果 Microsoft Word 处于某个特殊模式（如 CopyText 模式或 MoveText 模式）下，则该属性值为 True。Boolean 类型，只读。
StartupPath返回或设置启动文件夹的完整路径，不包括最后的分隔符。String 类型，可读写。
StatusBarMicrosoft Word Visual Basic for Applications 不再支持此属性。
SynonymInfo返回一个 SynonymInfo 对象，该对象包含同义词库中有关指定单词或短语的同义词、反义词或相关单词和表达方式的信息。
System返回一个 System 对象，该对象可用于返回与系统相关的信息，并执行与系统相关的任务。
TaskPanes返回一个 TaskPanes 集合，该集合代表 Microsoft Word 中最常执行的任务。
Tasks返回一个 Tasks 集合，该集合代表所有正在运行的应用程序。
Templates返回一个 Templates 集合，该集合代表所有可用的模板，包括共用模板和附加到打开文档的模板。
Top返回或设置活动文档的垂直位置。Long 类型，可读写。
UndoRecord返回一个 UndoRecord 对象，该对象提供撤消堆栈的自定义入口点。只读。
UsableHeight返回 Microsoft Word 文档窗口高度可设置的最大值（以磅为单位）。Long 类型，只读。
UsableWidth返回 Microsoft Word 文档窗口可设置的最大宽度（以磅为单位）。Long 类型，只读。
UserAddress该属性返回或设置用户的通讯地址。String 类型，可读写。
UserControl 如果文档或应用程序是由用户创建或打开的，则该属性值为 True。Boolean 类型，只读。
UserInitials该属性返回或设置用户名缩写，Microsoft Word 使用该缩写构成批注标记。String 类型，可读写。
UserName该属性返回或设置用户姓名，Word 将其用于信封和文档的“作者”属性。String 类型，可读写。
VBE返回一个 VBE 对象，该对象代表“Visual Basic 编辑器”。
Version返回 Microsoft Word 版本号。String 类型，只读。
Visible 如果指定的对象可见，则该属性值为 True。Boolean 类型，可读写。
Width返回或设置应用程序窗口的宽度（以磅为单位）。Long 类型，可读写。
Windows返回一个 Windows 集合，该集合代表所有文档窗口。只读。
WindowState返回或设置指定文档窗口或任务窗口的状态。WdWindowState 类型，可读写。
WordBasic返回一个自动化对象 (Word.Basic)，其中包含适用于 Word 6.0 和 Word for Windows 95 中所有 WordBasic 语句和函数的方法。只读。
XMLNamespaces返回一个 XMLNamespaces 集合，该集合代表架构库中的 XML 架构。
