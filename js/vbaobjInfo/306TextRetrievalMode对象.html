TextRetrievalMode 对象成员

代表控制如何从 Range 对象中检索文本的选项。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Duplicate返回一个只读 TextRetrievalMode 对象，该对象代表与从 Range 对象检索文字相关的选项。
IncludeFieldCodes 如果从指定区域获取的文本包含域代码，则该属性值为 True。Boolean 类型，可读写。
IncludeHiddenText 如果从指定区域检索的文本包含隐藏文字，则该属性值为 True。Boolean 类型，可读写。
Parent返回一个 Object 类型的值，该值代表指定 TextRetrievalMode 对象的父对象。
ViewType返回或设置 TextRetrievalMode 对象的视图。WdViewType 类型，可读写。
