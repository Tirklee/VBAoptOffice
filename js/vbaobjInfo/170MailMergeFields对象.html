MailMergeFields 对象成员

MailMergeField 对象的集合，这些对象代表文档中与邮件合并相关的域。

方法

 名称说明
Add返回一个 MailMergeField 对象，该对象代表添加到数据源文档中的邮件合并域。
AddAsk将一个 ASK 域添加到邮件合并主文档。返回一个 MailMergeField 对象。
AddFillIn将一个 FILLIN 域添到邮件合并主文档。返回一个 MailMergeField 对象。
AddIf将一个 IF 域添到邮件合并主文档。返回一个 MailMergeField 对象。
AddMergeRec将一个 MERGEREC 域添加到邮件合并主文档。返回一个 MailMergeField 对象。
AddMergeSeq将一个 MERGESEQ 域添加到邮件合并主文档。返回一个 MailMergeField 对象。
AddNext将一个 NEXT 域添加到邮件合并主文档。返回一个 MailMergeField 对象。
AddNextIf将一个 NEXTIF 域添加到邮件合并主文档。返回一个 MailMergeField 对象。
AddSet将一个 SET 域添加到邮件合并主文档。返回一个 MailMergeField 对象。
AddSkipIf将一个 SKIPIF 域添加到邮件合并主文档。返回一个 MailMergeField 对象。
Item返回集合中的单个 MailMergeField 对象。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Count返回一个 Long 类型的值，该值代表集合中邮件合并域的数目。只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Parent返回一个 Object 类型的值，该值代表指定 MailMergeFields 对象的父对象。
