Pane 对象成员

代表一个窗格。Pane 对象是 Panes 集合的一个成员。Panes 集合包含单个窗口的所有窗格。

方法

 名称说明
Activate激活指定的窗格。
AutoScroll在指定的窗格中自动滚动。
Close关闭指定的邮件合并数据源、窗格或任务。
LargeScroll将窗口或窗格滚动指定的屏数。
NewFrameset基于指定的窗格新建一个框架页。
PageScroll将指定的窗口按页滚动。
SmallScroll将窗口滚动指定的行数。
TOCInFrameset基于指定文档创建一个目录，并将其放入框架页左边的新框架中。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
BrowseWidth返回指定窗格中正文换行区域的宽度（以磅为单位）。Long 类型，只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
DisplayRulers 如果在指定窗格中显示标尺，则该属性值为 True。Boolean 类型，可读写。
DisplayVerticalRuler 如果在指定窗格中显示垂直标尺，则该属性值为 True。Boolean 类型，可读写。
Document返回一个 Document 对象，该对象与指定窗格相关。只读。
Frameset返回一个 Frameset 对象，该对象代表整个框架页或框架页中的单个图文框。只读。
HorizontalPercentScrolled返回或设置水平滚动条位置（以文档宽度的百分比表示）。可读写 Long 类型。
Index返回 Long 值，该值表示项目在集合中的位置。只读。
MinimumFontSize返回或设置在指定窗格中显示的最小字号（以磅为单位）。Long 类型，可读写。
Next返回一个 Pane 对象，该对象代表集合中的下一个文档窗格。只读。
Pages返回一个 Pages 集合，该集合代表文档中的页面。
Parent返回一个 Object 类型的值，该值代表指定 Pane 对象的父对象。
Previous返回一个 Pane 对象，该对象代表集合中的上一个文档窗格。只读。
Selection返回 Selection 对象，该对象代表文档窗格中的所选内容或插入点。只读。
VerticalPercentScrolled返回或设置垂直滚动条的位置（以文档长度的百分比表示）。可读写 Long 类型。
View返回一个 View 对象，该对象代表指定窗格的视图。
Zooms返回一个 Zooms 集合，该集合代表每个视图（如普通视图、大纲视图和页面视图）的缩放选项。
