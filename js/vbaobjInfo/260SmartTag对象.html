SmartTag 对象成员

代表文档或范围中包含识别类型信息的字符串。SmartTag 对象是 SmartTags 集合的一个成员。SmartTags 集合包含文档或文档的文本范围中的所有智能标记。Microsoft Word 使用识别器文件标记智能标记，并使用操作文件执行与智能标记相关的操作，例如链接到网站。

方法

  名称 说明
Delete 删除指定的智能标记。
Select 选择指定的智能标记。

属性

  名称 说明
Application 返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator 返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
DownloadURL 返回一个 String 类型的值，该值代表智能标签的 URL 地址。只读。
Name 返回指定对象的名称。只读 String 类型。
Parent 返回一个 Object 类型值，该值代表指定 SmartTag 对象的父对象。
Properties 返回一个 CustomProperties 集合，该集合代表智能标记的属性。
Range 返回一个 Range 对象，该对象代表智能标记中包含的文档部分。
SmartTagActions 返回一个 SmartTagActions 集合，该集合代表智能标记上可用操作的集合。
XML 返回一个 String 类型的值，该值代表与智能标记相关的 XML。只读。
XMLNode 返回一个 XMLNode 对象，该对象代表智能标记的 XML 元素。
