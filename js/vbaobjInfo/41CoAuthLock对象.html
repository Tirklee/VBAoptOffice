CoAuthLock 对象成员

代表文档内的一个锁。CoAuthLock 对象是 CoAuthLocks 集合的一个成员。

方法

 名称说明
Unlock解除对指定锁的锁定。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
HeaderFooter如果锁是文档的页眉-页脚锁，则返回 True。只读。
Owner返回拥有指定锁的 CoAuthor。只读。
Parent返回一个 Object 类型的值，该值代表指定 CoAuthLock 对象的父对象。
Range返回一个 Range 对象，该对象代表指定对象中包含的文档部分。只读。
Type返回一个指定锁定类型的 WdLockType 常量。只读。
