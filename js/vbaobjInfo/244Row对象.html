Row 对象成员

代表表格中的一行。Row 对象是 Rows 集合的一个成员。Rows 集合包含指定的所选内容、范围或表格中的所有行。

方法

 名称说明
ConvertToText将表格转换为文本并返回一个 Range 对象，该对象代表带分隔符的文本。
Delete删除指定的表格行。
Select选择指定的表格行。
SetHeight设置表格行的高度。
SetLeftIndent设置表格中行的缩进。

属性

 名称说明
Alignment返回或设置一个 WdRowAlignment 常量，该常量代表指定行的对齐方式。可读写。
AllowBreakAcrossPages 如果允许分页符拆分表格中一行或多行中的文本，则该属性值为 True。可读写 Long 类型。
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Borders返回一个 Borders 集合，该集合代表指定对象的所有边框。
Cells返回一个 Cells 集合，该集合代表在某一列、行、选定内容或区域中的表格单元格。只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
HeadingFormat 如果为 True，则将指定一行或数行的格式设置为表格标题。当表格不止一页时，可将多行的格式重复设置为表格标题。可以是 True、False 或 wdUndefined。Long 类型，可读写。
Height返回或设置表格中指定行的高度。Single 类型，可读写。
HeightRule返回或设置确定指定单元格或行高度的规则。WdRowHeightRule 类型，可读写。
ID在文档另存为网页时返回或设置指定表格行的标识标签。可读写 String 类型。
Index返回 Long 值，该值表示项目在集合中的位置。只读。
IsFirst 如果指定的行是表格中的首行，则该属性值为 True。只读 Boolean 类型。
IsLast 如果指定的行是表格中的末行，则该属性值为 True。只读 Boolean 类型。
LeftIndent返回或设置一个 Single 类型的值，该值代表指定表格行的左缩进值（以磅为单位）。可读写。
NestingLevel返回指定表格行的嵌套层。只读 Long 类型。
Next返回一个 Row 对象，该对象代表表格中行的集合中的下一个表格行。只读。
Parent返回一个 Object 类型的值，该值代表指定 Row 对象的父对象。
Previous返回一个 Row 对象，该对象代表指定行的前一个表格行。只读。
Range返回一个 Range 对象，该对象代表指定表格行内包含的文档部分。
Shading返回一个 Shading 对象，该对象代表指定对象的底纹格式。
SpaceBetweenColumns返回或设置指定的一行或多行中相邻列的文本之间的距离（以磅为单位）。可读写 Single 类型。
