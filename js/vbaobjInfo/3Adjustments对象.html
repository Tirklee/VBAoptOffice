Adjustments 对象成员

包含指定“自选图形”或“艺术字”对象的调整值的集合。每个调整值代表调整控点进行调整的方式。因为某些调整控点可以通过两种方式进行调整（例如有些控点可以水平和垂直调整），所以一个图形的调整值可多于它的调整控点。一个图形最多可有 8 个调整值。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Count返回 Adjustments 集合中的项目数。只读 Long 类型。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Item返回或设置由 Index 参数指定的调整值。Single 类型，可读写。
Parent返回一个 Object 类型的值，该值代表指定对象的父对象。通常为 Shape 或 ShapeRange 对象。
