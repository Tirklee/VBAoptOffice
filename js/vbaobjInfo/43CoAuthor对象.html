CoAuthor 对象成员

代表文档中的单个共同作者。CoAuthor 对象是 CoAuthors 集合的成员。CoAuthors 集合包含文档中的所有共同作者（当前正在编辑文档的作者）。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
EmailAddress返回一个字符串，该字符串指示指定共同作者的电子邮件地址。只读。
ID返回一个 String 类型的值，该值指定某个指定作者的唯一标识符。只读。
IsMe如果此作者代表当前用户，则返回 true。只读。
Locks返回一个 CoAuthLocks 集合，该集合代表文档中与指定共同作者相关联的锁。只读。
Name返回一个 String 类型的值，该值包含指定的共同作者的显示名称。只读。
Parent返回一个 Object 类型的值，该值代表指定 CoAuthor 对象的父对象。
