ProtectedViewWindows 对象成员

Word 2010 中当前打开的所有 ProtectedViewWindow 对象的集合。

方法

 名称说明
Item返回集合中的单个 ProtectedViewWindows 对象。
Open在新的受保护的视图窗口中打开指定的文档。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。只读。
Count返回一个 Long 值，该值代表集合中受保护的视图窗口的数量。只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Parent返回一个 Object 类型的值，该值代表指定 ProtectedViewWindows 对象的父对象。
