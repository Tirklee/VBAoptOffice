Cell 对象成员

代表单个表格单元格。Cell 对象是 Cells 集合的成员。Cells 集合代表指定对象中所有的单元格。

方法

 名称说明
AutoSum插入一个 = (Formula) 域，以计算和显示表达式指定的位于表格某一单元格上方或左侧的各单元格的数值之和。
Delete删除一个或多个表格单元格并可选择控制如何移动剩余的单元格。
Formula将包含指定公式的 = (Formula) 域插入到单元格中。
Merge将指定单元格与另一表格单元格合并，成为一个单独的表格单元格。
Select选择指定的对象。
SetHeight设定表格单元格的高度。
SetWidth设置表格的列或单元格的宽度。
Split将一个表格单元格拆分为多个。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Borders返回一个 Borders 集合，该集合代表指定对象的所有边框。
BottomPadding返回或设置在表格中单独的单元格或所有单元格内容的下方添加的间距（以磅为单位）。Single 类型，可读写。
Column返回一个 Column 对象，该对象代表包含指定单元格的表格列。只读。
ColumnIndex返回包含指定单元格的表格列数。Long 类型，只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
FitText 如果 Microsoft Word 会减小键入单元格中的文字的可视大小，使其适应列宽，则该属性值为 True。Boolean 类型，可读写。
Height返回或设置指定表格单元格的高度。
HeightRule返回或设置一个 WdRowHeightRule 常量，该常量代表确定指定单元格或行高度的规则。可读写。
ID在当前文档另存为网页时，返回或设置指定对象的标识标签。可读写 String 类型。
LeftPadding返回或设置要添加在表格的单个单元格或所有单元格内容左侧的间距（以磅为单位）。可读/写 Single 类型。
NestingLevel返回指定单元格的嵌套层。Long 类型，只读。
Next返回一个 Cell 对象，该对象代表 Cells 集合中的下一个表格单元格。只读。
Parent返回一个 Object 类型的值，该值代表指定 Cell 对象的父对象。
PreferredWidth返回或设置指定单元格的首选宽度（以磅为单位或表示为窗口宽度的百分比）。Single 类型，可读写。
PreferredWidthType返回或设置用于指定单元格的宽度的首选度量单位。WdPreferredWidthType 类型，只读。
Previous返回一个 Cell 对象，该对象代表 Cells 集合中的前一个表格单元格。只读。
Range返回一个 Range 对象，该对象代表指定对象中包含的文档部分。
RightPadding返回或设置要添加在表格的单个单元格或所有单元格的内容右侧的间距（以磅为单位）。可读/写 Single 类型。
Row返回一个 Row 对象，该对象代表指定单元格所在的行。
RowIndex返回指定单元格所在行的行数。Long 类型，只读。
Shading返回一个 Shading 对象，该对象代表指定对象的底纹格式。
Tables返回一个 Tables 集合，该集合代表指定表格单元格中的所有嵌套表格。只读。
TopPadding返回或设置要添加在表格的单个单元格或所有单元格的内容上方的间距（以磅为单位）。可读/写 Single 类型。
VerticalAlignment返回或设置表格中一个或多个单元格中文字的垂直对齐方式。WdCellVerticalAlignment 类型，可读写。
Width返回或设置指定表格单元格的宽度（以磅为单位）。Long 类型，可读写。
WordWrap 如果为 True，则 Microsoft Word 使文本自动换行并加长单元格，使单元格的宽度保持相同。Boolean 类型，可读写。
