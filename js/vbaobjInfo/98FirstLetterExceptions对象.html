FirstLetterExceptions 对象成员

FirstLetterException 对象的集合，这些对象代表不应用自动更正的缩写。

方法

 名称说明
Add返回一个 FirstLetterException 对象，该对象代表添加到“自动更正”例外项列表中的一个新例外项。
Item返回集合中的单个 FirstLetterException 对象。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Count返回一个 Long 类型的值，该值代表集合中首字母大写例外项的数目。只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Parent返回一个 Object 类型的值，该值代表指定 FirstLetterExceptions 对象的父对象。
