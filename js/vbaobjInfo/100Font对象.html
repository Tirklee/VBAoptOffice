Word 开发人员参考
Font 对象成员

包含对象的字体属性（如字体名称、字号、颜色等）。

方法

 名称说明
Grow将字号增大为下一个可用的字号。
Reset删除手动字符格式（不使用样式应用的格式）。例如，如果手动将单词设置为加粗格式，但基本样式为纯文本（不是粗体），则可用 Reset 方法删除加粗格式。
SetAsTemplateDefault将指定的字体格式设置为活动文档和所有基于活动模板创建的新文档的默认格式。
Shrink将字号减小为下一个可用的字号。

属性

 名称说明
AllCaps 如果字体格式为全部字母大写，则该属性值为 True。Long 类型，可读写。
Animation返回或设置应用于字体的动态效果类型。WdAnimation 类型，可读写。
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Bold 如果字体格式为加粗，则该属性值为 True。Long 类型，可读写。
BoldBi 如果字体格式为加粗，则该属性值为 True。Long 类型，可读写。
Borders返回一个 Borders 集合，该集合代表指定字体的所有边框。
ColorIndex返回或设置一个 WdColorIndex 常量，该常量代表指定字体的颜色。可读写。
ColorIndexBi返回或设置从右向左语言文档中指定 Font 对象的颜色。WdColorIndex 类型，可读写。
ContextualAlternates指定是否为指定的字体启用上下文替换。可读写 Long。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
DiacriticColor返回或设置用于指定 Font 对象的音调符号的 24 位颜色。
DisableCharacterSpaceGrid 如果 Microsoft Word 忽略相应 Font 对象的每行字符数，则该属性值为 True。Boolean 类型，可读写。
DoubleStrikeThrough 如果将指定字体的格式设置为双删除线文本，则该属性值为 True。
Duplicate返回一个只读 Font 对象，该对象代表指定字体的字符格式。
Emboss 如果指定字体格式为阳文，则该属性值为 True。Long 类型，可读写。
EmphasisMark返回或设置 WdEmphasisMark 常量，该常量代表字符或指定字符串的着重号。可读写。
Engrave 如果字体格式为阴文，则该属性值为 True。Long 类型，可读写。
Fill返回一个 FillFormat 对象，该对象包含指定文本区域所用字体的填充格式属性。只读。
Glow返回一个 GlowFormat 对象，该对象代表指定文本区域所用字体的发光格式。只读。
Hidden 如果字体格式为隐藏文字，则该属性值为 True。Long 类型，可读写。
Italic 如果将字体或范围设置为倾斜格式，则该属性值为 True。Long 类型，可读写。
ItalicBi 如果将字体或范围设置为倾斜格式，则该属性值为 True。Long 类型，可读写。
Kerning返回或设置 Microsoft Word 根据字号自动调整字距时的最小字号。Single 类型，可读写。
Ligatures返回或设置指定的 Font 对象的连字设置。可读写 WdLigatures。
Line返回一个 LineFormat 对象，该对象指定行的格式。可读写。
Name返回或设置指定对象的名称。String 类型，可读写。
NameAscii返回或设置拉丁语（字符代码从 0（零）到 127 的字符）所用的字体。String 类型，可读写。
NameBi返回或设置使用从右向左语言的文档中的字体的名称。String 类型，可读写。
NameFarEast返回或设置一种东亚字体名称。String 类型，可读写。
NameOther返回或设置字符代码从 128 到 255 的字符所用的字体。String 类型，可读写。
NumberForm返回或设置 OpenType 字体的数字形式设置。可读写 WdNumberForm。
NumberSpacing返回或设置字体的数字间距设置。可读写 WdNumberSpacing。
Outline 如果字体格式为镂空，则该属性值为 True。Long 类型，可读写。
Parent返回一个 Object 类型的值，该值代表指定 Font 对象的父对象。
Position返回或设置相对于基准线的文本位置（以磅为单位）。可读写 Long 类型。
Reflection返回一个 ReflectionFormat 对象，该对象代表形状的反射格式。只读。
Scaling返回或设置应用于字体的缩放百分比。Long 类型，可读写。
Shading返回一个 Shading 对象，该对象代表指定字体的底纹格式。
Shadow 如果将指定字体设置为阴影格式，则该属性值为 True。可读写 Long 类型。
Size返回或设置字号（以磅为单位）。Single 类型，可读写。
SizeBi返回或设置字体大小（以磅为单位）。Single 类型，可读写。
SmallCaps 如果字体格式为小型大写字母，则该属性值为 True。Long 类型，可读写。
Spacing返回或设置字符的间距（以磅为单位）。可读写 Single 类型。
StrikeThrough 如果字体格式为带有删除线，则该属性值为 True。Long 类型，可读写。
StylisticSet指定所指定字体的样式集。可读写 WdStylisticSet。
Subscript 如果字体格式为下标，则该属性值为 True。Long 类型，可读写。
Superscript 如果字体格式为上标，则该属性值为 True。Long 类型，可读写。
TextColor返回一个 ColorFormat 对象，该对象代表指定字体的颜色。只读。
TextShadow返回一个 ShadowFormat 对象，该对象指定所指定字体的阴影格式。
ThreeD返回一个 ThreeDFormat 对象，该对象包含指定字体的三维效果格式属性。只读。
Underline返回或设置应用于字体的下划线类型。可读写 WdUnderline 类型。
UnderlineColor返回或设置指定 Font 对象的下划线的 24 位颜色。
