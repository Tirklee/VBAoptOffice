Frameset 对象成员

代表整个框架页或框架页中的单个图文框。

方法

 名称说明
AddNewFrame该方法在框架页中添加一个新框架。
Delete删除指定的 Frameset 对象。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
ChildFramesetCount返回指定 Frameset 对象的关联 Frameset 子对象的数目。Long 类型，只读。
ChildFramesetItem返回一个 Frameset 对象，该对象代表 Index 参数指定的 Frameset 子对象。只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
FrameDefaultURL返回或设置当打开框架页时在指定框架中显示的网页或其他文档。String 类型，可读写。
FrameDisplayBorders 如果显示指定框架页上的框架边框，则该属性值为 True。Boolean 类型，可读写。
FrameLinkToFile 如果 FrameDefaultURL 属性指定的网页或其他文档是一个外部文件，Microsoft Word 仅保留从指定框架到该文件的一个链接，则该属性值为 True。Boolean 类型，可读写。
FrameName返回或设置框架页上指定框架的名称。String 类型，可读写。
FrameResizable 如果用户在 Web 浏览器中查看框架页时可调整指定框架的大小，则该属性值为 True。Boolean 类型，可读写。
FrameScrollbarType返回或设置当用 Web 浏览器浏览指定框架的框架页时，何时可以在该指定框架中使用滚动条。WdScrollbarType 类型，可读写。
FramesetBorderColor返回或设置指定框架页上框架边框的颜色。
FramesetBorderWidth返回或设置指定框架页上框架周围边框的宽度（以磅为单位）。Single 类型，可读写。
Height返回或设置一个 Float 类型的值，该值代表指定 Frameset 对象的高度。可读写。
HeightType返回或设置框架页上指定框架的宽度类型。WdFramesetSizeType 类型，可读写。
Parent返回一个 Object 类型的值，该值代表指定 Frameset 对象的父对象。
ParentFrameset返回一个 Frameset 对象，该对象代表框架页上指定 Frameset 对象的父级。
Type返回 Frameset 对象的类型。只读 WdFramesetType 类型。
Width返回或设置指定的 Frameset 对象的宽度。可读写 Long 类型。
WidthType返回或设置指定的 Frameset 对象的宽度类型。WdFramesetSizeType 类型，可读写。
