Series 对象成员

代表图表上的系列。

方法

 名称说明
ApplyDataLabels向系列应用数据标签。
ClearFormats清除对象的格式设置。
Copy如果系列具有图片填充，则将图片复制到剪贴板。
DataLabels返回一个代表系列中单个数据标签（DataLabel 对象）或所有数据标签集合（DataLabels 集合）的对象。
Delete删除对象。
ErrorBar将误差线应用于系列。
Paste从剪贴板中粘贴图片，将其作为所选系列的标记。
Points返回系列中所有数据点的集合。
Select选择对象。
Trendlines返回系列的所有趋势线的集合。

属性

 名称说明
Application如果不与对象识别符一起使用，则该属性返回一个代表 Microsoft Word 应用程序的 Application 对象。如果与对象识别符一起使用，则该属性返回代表指定对象的创建者的 Application 对象（可对自动操作对象使用该属性来返回该对象的应用程序）。只读。
ApplyPictToEnd 如果图片应用于系列中的某个数据点或所有数据点之后，则为 True。可读/写 Boolean 类型。
ApplyPictToFront 如果图片应用于系列中的某个数据点或所有数据点之前，则为 True。可读/写 Boolean 类型。
ApplyPictToSides 如果将图片应用于系列中的某个数据点或所有数据点的两侧，则为 True。可读/写 Boolean 类型。
AxisGroup返回坐标轴组的类型。可读/写 XlAxisGroup。
BarShape返回或设置用于三维条形图或三维柱形图中单个系列的形状。可读/写 XlBarShape。
Border返回对象的边框。只读 ChartBorder 对象。
BubbleSizes返回或设置一个字符串，该字符串引用包含气泡图 x 值、y 值和大小数据的工作表单元格。可读/写 Variant 类型。
ChartType返回或设置图表类型。可读/写 XlChartType。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
ErrorBars返回系列的误差线。只读 ErrorBars 对象。
Explosion返回或设置饼图或圆环图的扇区的分离程度值。可读/写 Long 类型。
Format返回对象的线条、填充和效果格式。只读 ChartFormat。
Formula用宏语言以 A1 样式表示法返回或设置对象的公式。可读/写 String 类型。
FormulaLocal返回或设置对象的公式，使用用户语言 A1 格式引用。可读/写 String 类型。
FormulaR1C1返回或设置对象的公式，使用宏语言 R1C1 格式符号表示。可读/写 String 类型。
FormulaR1C1Local返回或设置对象的公式，使用用户语言 R1C1 格式符号表示。可读/写 String 类型。
Has3DEffect 如果系列有三维外观，则该属性值为 True。可读/写 Boolean 类型。
HasDataLabels 如果系列有数据标签，则该属性值为 True。可读/写 Boolean 类型。
HasErrorBars 如果系列有误差线，则为 True。可读/写 Boolean 类型。
HasLeaderLines 如果系列有引导线，则该属性值为 True。可读/写 Boolean 类型。
InvertColor返回或设置一个系列中负数数据点的填充颜色。可读写。
InvertColorIndex返回或设置一个系列中负数数据点的填充颜色。可读写。
InvertIfNegative 如果对象与一个负数相对应时 Microsoft Word 就将其反色，则为 True。可读/写 Variant 类型。
LeaderLines返回系列的引导线。只读 LeaderLines 对象。
MarkerBackgroundColor将数据标记的背景色设置为 RGB 值或返回对应的颜色索引值。可读/写 Long 类型。
MarkerBackgroundColorIndex返回或设置数据标记的背景色，表示为当前调色板中的索引或以下 XlColorIndex 常量之一：xlColorIndexAutomatic 或 xlColorIndexNone。可读/写 Long 类型。
MarkerForegroundColor将数据标记的前景色设置为 RGB 值或返回对应的颜色索引值。可读/写 Long 类型。
MarkerForegroundColorIndex返回或设置数据标记的前景色，表示为当前调色板中的索引或以下 XlColorIndex 常量之一：xlColorIndexAutomatic 或 xlColorIndexNone。可读/写 Long 类型。
MarkerSize以磅为单位返回或设置数据标记的大小。可读/写 Long 类型。
MarkerStyle返回或设置折线图、散点图或雷达图中数据点或系列的数据标记样式。可读/写 XlMarkerStyle。
Name返回或设置对象的名称。可读/写 String 类型。
Parent返回指定对象的父对象。只读 Object 对象。
PictureType返回或设置一个指定图片在柱形图或条形图上的显示方式的值。可读/写 XlChartPictureType。
PictureUnit2如果 PictureType 属性设置为 xlStackScale，则返回或设置图表上每张图片的单位；否则，忽略该属性。可读写 Double。
PlotColorIndex返回在内部用于将系统格式与图表元素相关联的索引值。只读。
PlotOrder返回或设置图表组中选定系列的绘制顺序。可读/写 Long 类型。
Shadow返回或设置一个表示对象是否有阴影的值。可读/写 Boolean 类型。
Smooth 如果对折线图或散点图应用了平滑线效果，则为 True。可读/写 Boolean 类型。
Type返回或设置系列类型。可读/写 Long 类型。
Values返回或设置系列中所有值的集合。可读/写 Variant 类型。
XValues返回或设置图表系列中 x 值的数组。可读/写 Variant 类型。
