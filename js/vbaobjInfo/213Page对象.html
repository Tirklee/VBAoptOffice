Page 对象成员

代表文档中的页面。使用 Page 对象及相关方法和属性可通过编程方式定义文档的页面版式。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Breaks返回一个 Breaks 集合，该集合代表页面上的分隔符。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
EnhMetaFileBits返回一个 Variant 类型的值，该值代表文本页显示方式的图片表示形式。只读。
Height返回一个 Long 类型的值，该值代表页面的高度（以像素为单位）。
Left返回一个 Long 类型的值，该值代表页面的左边缘。只读。
Parent返回一个 Object 类型的值，该值代表指定 Page 对象的父对象。
Rectangles返回一个 Rectangles 集合，该集合代表文档页面上的文字或图形部分。
Top返回一个 Long 类型的值，该值代表页面的上边缘。只读。
Width返回一个 Long 类型的值，该值代表在“页面设置”对话框中定义的纸张宽度，以磅为单位。只读 Long 类型。
