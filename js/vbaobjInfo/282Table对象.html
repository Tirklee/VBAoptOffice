Table 对象成员

代表单个表格。Table 对象是 Tables 集合的一个成员。Tables 集合包含指定的所选内容、范围或文档中的所有表格。

方法

 名称说明
ApplyStyleDirectFormatting 应用指定的样式，但保留用户直接应用的所有格式。
AutoFitBehavior确定 Microsoft Word 如何使用自动调整功能来调整表格的大小。
AutoFormat将预定义外观应用于表格。
Cell返回一个 Cell 对象，该对象代表表格中的一个单元格。
ConvertToText将表格转换为文本并返回一个 Range 对象，该对象代表带分隔符的文本。
Delete删除指定的表格。
Select选择指定的表格。
Sort对指定的表格进行排序。
SortAscending按字母数字升序对段落或表格行进行排序。
SortDescending以字母数字降序排列表格行。
Split在表格中紧靠指定行的上面插入一空段落，并且返回一个 Table 对象，此对象包含指定行及其下一行。
UpdateAutoFormat使用预定义表格式对表格式进行更新。

属性

 名称说明
AllowAutoFit使 Microsoft Word 可以自动调整表格中的单元格的大小以适应内容。Boolean 类型，可读写。
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
ApplyStyleColumnBands返回或设置 Boolean 值，该值表示如果所应用的预置表样式为列提供了样式带，则是否对表中的列应用样式带。可读/写。
ApplyStyleFirstColumn 如果为 True，则 Microsoft Word 对指定表格的第一列应用第一列的格式。Boolean 类型，可读写。
ApplyStyleHeadingRows 如果为 True，则 Microsoft Word 为选定表格的第一行应用标题行的格式。Boolean 类型，可读写。
ApplyStyleLastColumn 如果为 True，则 Microsoft Word 对指定表格的最后一列应用最后一列的格式。Boolean 类型，可读写。
ApplyStyleLastRow 如果为 True，则 Microsoft Word 对指定表格的最后一行应用最后一行的格式。Boolean 类型，可读写。
ApplyStyleRowBands返回或设置 Boolean 值，该值表示如果所应用的预置表样式为行提供了样式带，则是否对表中的行应用样式带。可读/写。
AutoFormatType返回指定表格所应用的自动套用格式类型。Long 类型，只读。
Borders返回一个 Borders 集合，该集合代表指定对象的所有边框。
BottomPadding返回或设置在表格中单独的单元格或所有单元格内容的下方添加的间距（以磅为单位）。Single 类型，可读写。
Columns返回一个 Columns 集合，该集合代表表格中的所有表格列。只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Descr返回或设置一个 String 值，该值包含指定表格的说明。可读写。
ID在将文档另存为网页时，返回或设置指定的表的标识标签。可读/写 String 类型。
LeftPadding返回或设置在表格中所有单元格的内容左侧要增加的间距（以磅为单位）。可读写 Single 类型。
NestingLevel返回指定表格的嵌套层。只读 Long 类型。
Parent返回一个 Object 类型的值，该值代表指定 Table 对象的父对象。
PreferredWidth返回或设置指定表格的指定宽度（以磅为单位或以窗口宽度的百分比表示）。可读写 Single 类型。
PreferredWidthType返回或设置用于指定表格宽度的指定度量单位。可读写 WdPreferredWidthType 类型。
Range返回一个 Range 对象，该对象代表指定表格中所含的文档部分。
RightPadding返回或设置在表格的所有单元格的内容右侧要增加的间距（以磅为单位）。可读写 Single 类型。
Rows返回一个 Rows 集合，该集合代表表格中所有的表格行。只读。
Shading返回一个 Shading 对象，该对象代表指定对象的底纹格式。
Spacing返回或设置表格中单元格的间距（以磅为单位）。可读写 Single 类型。
Style返回或设置指定表格的样式。可读写 Variant 类型。
TableDirection返回或设置 Microsoft Word 对指定表格中的单元格进行排序的方向。可读写 WdTableDirection 类型。
Tables返回一个 Tables 集合，该集合代表指定表格中嵌套的所有表格。只读。
Title返回或设置一个 String 值，该值包含指定表格的标题。可读写。
TopPadding返回或设置表格中所有单元格的内容上方要增加的间距（以磅为单位）。可读写 Single 类型。
Uniform 如果表格中所有的行具有相同的列数，则该属性值为 True。Boolean 类型，只读。
