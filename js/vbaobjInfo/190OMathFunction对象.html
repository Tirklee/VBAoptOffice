OMathFunction 对象成员

代表 Microsoft Word 所支持的数学函数或结构，如分数、积分、和以及根。OMathFunction 对象是 OMathFunctions 集合的成员。

方法

 名称说明
Remove从公式中包含的函数集合内删除一个函数，并且返回一个 OMathFunction 对象。

属性

 名称说明
Acc返回一个 OMathAcc 对象，该对象代表带组合重音符号的基本字符。只读。
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Args返回一个 OMathArgs 对象，该对象代表公式中的参数。只读。
Bar返回一个 OMathBar 对象，该对象代表某个对象的数学顶线。只读。
BorderBox返回一个 OMathBorderBox 对象，该对象代表围绕整个公式或部分公式绘制的边框。BorderBox 对象还可用于通过 BorderBox 参数绘制水平、垂直和对角删除线。只读。
Box返回一个 OMathBox 对象，该对象代表整个公式或部分公式周围不可见的框，您可以对该公式应用属性，以改变其数学属性或格式属性（如换行）。只读。
Creator返回一个 32 位的整数，该整数表示在其中创建加载项的应用程序。只读 Long 类型。
Delim返回一个代表定界符函数的 OMathDelim 对象。只读。
EqArray返回一个代表公式数组函数的 OMathEqArray 对象。只读。
Frac返回一个代表分数的 OMathFrac 对象。只读。
Func返回一个 OMathFunc 对象，该对象代表由函数名（如 sin 或 cos）和参数组成的数学函数类型。只读。
GroupChar返回一个 OMathGroupChar 对象，该对象代表放在公式中文本上方或下方的水平字符，这通常是为了对文本进行直观分组。只读。
LimLow返回一个代表函数下限的 OMathLimLow 对象。只读。
LimUpp返回一个代表函数上限的 OMathLimUpp 对象。只读。
Mat返回一个代表数学矩阵的 OMathMat 对象。只读。
Nary返回一个代表 n 元运算的 OMathNary 对象。只读。
OMath返回一个代表公式的 OMath 对象。只读。
Parent返回一个 Object 类型的值，该值代表指定 OMathFunction 对象的父对象。
Phantom返回一个 OMathPhantom 对象，该对象代表用于设置公式高级版式的对象。只读。
Rad返回一个代表数学开方函数的 OMathRad 对象。只读。
Range返回一个 Range 对象，该对象代表指定对象中包含的文档部分。只读。
ScrPre返回一个 OMathScrPre 对象，该对象代表底数左侧的上标和下标。只读。
ScrSub代表一个代表数学下标函数的 OMathScrSub 对象。只读。
ScrSubSup返回一个 OMathScrSubSup 对象，该对象代表由底数、下标和上标组成的数学下标-上标对象。只读。
ScrSup返回一个代表数学上标函数的 OMathScrSup 对象。只读。
Type返回一个代表函数类型的 WdOMathFunctionType 常量。只读。
