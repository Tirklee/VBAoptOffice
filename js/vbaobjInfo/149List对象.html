List 对象成员

代表已应用于文档中指定段落的单一列表格式。List 对象是 Lists 集合的一个成员。

方法

 名称说明
ApplyListTemplate为指定的 ListFormat 对象设置列表格式。
ApplyListTemplateWithLevel以后的版本中将提供关于此成员的说明。
CanContinuePreviousList返回一个 WdContinue 常量（wdContinueDisabled、wdResetList 或 wdContinueList），该常量指示是否可以继续使用上一列表中的格式。
ConvertNumbersToText更改指定的 List 对象中的列表编号和 LISTNUM 域。
CountNumberedItems返回指定 List 对象中的项目符号或编号项目和 LISTNUM 域的数量。
RemoveNumbers删除指定列表中的编号或项目符号。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
ListParagraphs返回一个 ListParagraphs 集合，该集合代表列表、文档或区域内的所有编号段落。只读。
Parent返回一个 Object 类型的值，该值代表指定 List 对象的父对象。
Range返回 Range 对象，该对象表示文档包含在指定对象中的一部分。
SingleListTemplate 如果整个列表都使用同一列表模板，则该属性值为 True。只读 Boolean 类型。
StyleName返回应用于指定的“自动图文集”词条的样式名称。只读 String 类型。
