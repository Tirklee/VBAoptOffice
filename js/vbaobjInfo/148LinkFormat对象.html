LinkFormat 对象成员

代表 OLE 对象或图片的链接特征。

方法

 名称说明
BreakLink断开源文件与指定 OLE 对象、图片或链接域之间的链接。
Update更新指定的链接格式。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
AutoUpdate 如果在打开容器文件或更改源文件时，自动更新指定链接，则该属性值为 True。Boolean 类型，可读写。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Locked 如果锁定 Field、InlineShape 或 Shape 对象以防止自动更新，则该属性值为 True。可读写 Boolean 类型。
Parent返回一个 Object 类型的值，该值代表指定 LinkFormat 对象的父对象。
SavePictureWithDocument 如果指定的图片与文档一起保存，则该属性值为 True。Boolean 类型，可读写。
SourceFullName返回或设置指定的链接 OLE 对象、图片或域的源文件的路径和名称。String 类型，可读写。
SourceName返回指定的链接 OLE 对象、图片或域的源文件的名称。String 类型，可读写。
SourcePath返回指定的链接 OLE 对象、图片或域的源文件的路径。String 类型，可读写。
Type返回链接的类型。只读 WdLinkType 类型。
