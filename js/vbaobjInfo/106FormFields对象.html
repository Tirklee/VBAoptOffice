FormFields 对象成员

FormField 对象的集合，这些对象代表所选内容、范围或文档中的所有窗体域。

方法

 名称说明
Add返回一个 FormField 对象，该对象代表添加到区域中的新的窗体域。
Item返回集合中的单个 FormField 对象。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Count返回一个 Long 类型的值，该值代表集合中域的数量。只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Parent返回一个 Object 类型的值，该值代表指定 FormFields 对象的父对象。
Shaded如果将底纹应用于窗体域，则该属性值为 True。Boolean 类型，可读写。
