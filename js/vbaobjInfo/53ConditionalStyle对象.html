ConditionalStyle 对象成员

用指定的表格样式设置选定表格的格式时，该对象代表应用于表格中指定区域的特殊格式。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Borders返回一个 Borders 集合，该集合代表指定 ConditionalStyle 对象的所有边框。
BottomPadding返回或设置代表在表格中单独的单元格或所有单元格内容的下方添加的间距（以磅为单位）的 Single 类型。可读写。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Font返回或设置一个 Font 对象，该对象代表指定对象的字符格式。可读写。
LeftPadding返回或设置要添加在表格的单个单元格或所有单元格内容左侧的间距（以磅为单位）。可读/写 Single 类型。
ParagraphFormat返回或设置一个 ParagraphFormat 对象，该对象代表指定条件样式的段落设置。可读写。
Parent返回一个 Object 类型的值，该值代表指定 ConditionalStyle 对象的父对象。
RightPadding返回或设置要添加在表格的单个单元格或所有单元格的内容右侧的间距（以磅为单位）。可读/写 Single 类型。
Shading返回一个表示表样式格式的底纹格式的 Shading 对象。
TopPadding返回或设置要添加在表格的单个单元格或所有单元格的内容上方的间距（以磅为单位）。可读/写 Single 类型。
