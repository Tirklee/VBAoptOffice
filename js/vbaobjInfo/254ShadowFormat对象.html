ShadowFormat 对象成员

代表形状的阴影格式。

方法

 名称说明
IncrementOffsetX按指定磅值改变阴影的水平偏移量。
IncrementOffsetY按指定磅值改变阴影的垂直偏移量。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Blur返回或设置一个 Single 类型的值，该值代表阴影格式的模糊级别。可读写。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
ForeColor返回或设置一个 ColorFormat 对象，该对象代表填充、线条或阴影的前景色。可读写。
Obscured 如果不论是否填充了指定图形，都将填充该图形的阴影部分且为图形所遮掩，则该属性值为 MsoTrue。如果不填充阴影，并且在没有填充图形的情况下，可透过图形看到阴影轮廓，则该属性值为 MsoFalse。MsoTriState 类型，可读写。
OffsetX返回或设置阴影根据指定图形的水平偏移量（以磅为单位）。正值表示阴影向图形右侧偏移，负值表示向左偏移。Single 类型，可读写。
OffsetY返回或设置阴影相对指定图形的垂直偏移量（以磅为单位）。Single 类型，可读写。
Parent返回一个 Object 类型的值，该值代表指定 ShadowFormat 对象的父对象。
RotateWithShape返回或设置一个 MsoTriState，代表在旋转形象时是否旋转阴影。可读写。
Size返回或设置一个 Single 类型的值，该值代表阴影的宽度。可读写。
Style返回或设置一个 MsoShadowType，代表应用于形状的阴影格式的类型。可读写。
Transparency返回或设置指定阴影的透明度，其值为从 0.0（不透明）到 1.0（全透明）。可读写 Single 类型。
Type返回或设置形状阴影的类型。可读/写 MsoShadowType 类型。
Visible 如果指定对象或应用于该对象的格式是可见的，则该属性值为 True。MsoTriState 类型，可读写。
