ListFormat 对象成员

代表可应用于范围中各段落的列表格式属性。

方法

 名称说明
ApplyBulletDefault向指定的 ListFormat 对象的区域中的段落添加项目符号和格式。
ApplyListTemplate为指定的 ListFormat 对象设置列表格式。
ApplyListTemplateWithLevel以后的版本中将提供关于此成员的说明。
ApplyNumberDefault向指定的 ListFormat 对象的区域中的段落添加默认的编号格式。
ApplyOutlineNumberDefault向指定的 ListFormat 对象的区域中的段落添加默认的多级符号格式。
CanContinuePreviousList返回一个 WdContinue 常量（wdContinueDisabled、wdResetList 或 wdContinueList），该常量表明是否接续上一列表格式。
ConvertNumbersToText将指定 ListFormat 对象中的列表编号和 LISTNUM 域更改为文本。
CountNumberedItems返回指定 ListFormat 对象中的项目符号或编号项目和 LISTNUM 域的数量。
ListIndent增加指定 ListFormat 对象的区域中的段落的列表级别，增加量为一个级别。
ListOutdent降低指定的 ListFormat 对象的区域中段落的列表级别，减量为一个级别。
RemoveNumbers删除指定列表中的编号或项目符号。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
List返回一个 List 对象，该对象代表指定 ListFormat 对象中包含的第一个带格式列表。
ListLevelNumber返回或设置指定的 ListFormat 对象中第一段的列表级别。可读写 Long 类型。
ListPictureBullet返回 InlineShape 对象，该对象代表用作图片项目符号列表中的项目符号的图片。
ListString返回一个 String 类型的值，该值代表指定的 ListFormat 对象区域中第一段的列表值的外观。例如，字母顺序列表中第二段的返回值为“B”。只读。
ListTemplate返回一个 ListTemplate 对象，该对象代表指定 ListFormat 对象的列表格式。
ListType返回指定的 ListFormat 对象区域中所包含的列表类型。WdListType 类型，只读。
ListValue返回指定的 ListFormat 对象区域中第一段的数值。Long 类型，只读。
Parent返回一个 Object 类型的值，该值代表指定 ListFormat 对象的父对象。
SingleList 如果指定的 ListFormat 对象只包含一个列表，则该属性值为 True。Boolean 类型，只读。
SingleListTemplate 如果整个 ListFormat 对象都使用同一列表模板，则该属性值为 True。只读 Boolean 类型。
