Indexes 对象成员

Index 对象的集合，这些对象代表指定文档中的所有索引。

方法

 名称说明
Add返回一个 Index 对象，该对象代表添加到文档的新索引。
AutoMarkEntries用对应文件中的索引项将 XE（索引项）域自动添至指定文档。
Item返回集合中的单个 Index 对象。
MarkAllEntries在 Range 中文字的所有实例后插入 XE（索引项）域。
MarkEntry在指定范围后插入一个 XE（索引项）域。该方法返回一个代表 XE 域的 Field 对象。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Count返回一个 Long 类型的值，该值代表集合中索引的数目。只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Format返回或设置 WdIndexFormat，代表指定文档中索引的格式。可读写。
Parent返回一个 Object 类型的值，该值代表指定 Indexes 对象的父对象。
