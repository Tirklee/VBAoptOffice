MappedDataField 对象成员
全部显示
全部隐藏

作为包含在 Microsoft Word 中的字段，映射数据字段代表通常使用的名称或地址信息，如“First Name”。如果数据源包含一个“First Name”字段或一个变量（例如“First_Name”、“FirstName”、“First”或“FName”），则数据源中的字段将自动映射到 Word 中与之相应的数据字段。如果文档或模板与一个以上的数据源合并，映射数据字段使之不必在文档中重新输入与数据库中的字段名称一致的数据字段。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
DataFieldIndex返回或设置一个 Long 类型的值，该值代表映射数据字段 （映射数据字段：代表常用信息的字段，例如，“First Name”。如果数据源中包含“First Name”字段或其变体（例如，“FName”），该数据源字段自动映射到相应的映射数据字段。）映射到的邮件合并数据源中的相应域编号。可读写。
DataFieldName设置或返回一个 String 类型的值，该值代表映射数据字段 （映射数据字段：代表常用信息的字段，例如，“First Name”。如果数据源中包含“First Name”字段或其变体（例如，“FName”），该数据源字段自动映射到相应的映射数据字段。）映射到的邮件合并数据源中域的名称。可读写。
Index返回 Long 值，该值表示项目在集合中的位置。只读。
Name返回指定对象的名称。String 类型，只读。
Parent返回一个 Object 类型的值，该值代表指定 MappedDataField 对象的父对象。
Value返回当前记录的邮件合并数据域或映射数据字段的内容。String 类型，只读。
