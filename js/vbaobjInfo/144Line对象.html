Line 对象成员

代表 wdTextRectangle 类型的 Rectangle 对象中的单个行。使用 Line 对象及相关的方法和属性可通过编程方式定义文档的页面布局。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Height返回或设置线条的高度。
Left返回一个 Long 类型的值，该值代表从页面的左边缘到线条的左边缘的像素数。
LineType返回一个 wdLineType 常量，表明该行为文字行还是表格行。
Parent返回一个 Object 类型的值，该值代表指定 Line 对象的父对象。
Range返回一个 Range 对象，该对象代表线条中包含的文档部分。
Rectangles返回一个 Rectangles 集合，该集合代表文档页面上的文字或图形部分。
Top返回一个 Long 类型的值，该值代表从页面顶端到线条顶端的像素数。
Width返回或设置线条的宽度（以磅为单位）。可读写 Long 类型。
