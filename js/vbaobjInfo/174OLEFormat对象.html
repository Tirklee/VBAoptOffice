OLEFormat 对象成员
全部显示
全部隐藏

代表 OLE 对象、ActiveX 控件或域的 OLE（而不是链接）特性。

方法

 名称说明
Activate激活指定的 OLEFormat 对象。
ActivateAs设置确定用于激活指定 OLE 对象的默认应用程序的 Windows 注册表值。
ConvertTo将指定的 OLE 对象转换为另一种类型，以便能在另一种服务器应用程序中编辑该对象，或者改变对象在文档中的显示方式。
DoVerb请求 OLE 对象执行一个有效动作，以激活其内容。
Edit在创建指定 OLE 对象的应用程序中打开该对象，以便进行编辑。
Open打开指定的 OLEFormat 对象。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
ClassType返回或设置指定的 OLE 对象、图片或域的类型。String 类型，可读写。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
DisplayAsIcon 如果指定对象显示为图标，则该属性值为 True。Boolean 类型，可读写。
IconIndex返回或设置 DisplayAsIcon 属性为 True 时所用的图标。Long 类型，可读写。
IconLabel返回或设置 OLE 对象的图标下面显示的文字。String 类型，可读写。
IconName返回或设置存储 OLE 对象的图标的程序文件。String 类型，可读写。
IconPath返回存储 OLE 对象的图标的文件的路径。String 类型，只读。
Label返回用于标识源文件中被链接的部分的字符串。String 类型，只读。
Object返回一个 Object 类型的值，该值代表指定的 OLE 对象的最上层接口。
Parent返回一个 Object 类型的值，该值代表指定 OLEFormat 对象的父对象。
PreserveFormattingOnUpdate 如果为 True，则将 Microsoft Word 中的格式设置保存到链接的 OLE 对象中，例如链接到 Microsoft Excel 电子表格的表格。Boolean 类型，可读写。
ProgID返回指定 OLE 对象的程序标识符 (ProgID) （程序标识符：窗体 OLEServerName.ObjectName 中的标识符（例如，Excel.Sheet 或 PowerPoint.Slide），由 Windows 注册表用来唯一标识一个对象。）。只读 String 类型。
