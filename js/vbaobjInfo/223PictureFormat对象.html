PictureFormat 对象成员

包含应用于图片和 OLE 对象的属性和方法。LinkFormat 对象包含仅应用于链接 OLE 对象的属性和方法。OLEFormat 对象包含应用于链接和非链接 OLE 对象的属性和方法。

方法

 名称说明
IncrementBrightness按指定数值更改图片的亮度。
IncrementContrast按指定数值更改图片的对比度。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Brightness返回或设置指定图片或 OLE 对象的亮度。该属性的值必须是 0.0（最暗）到 1.0（最亮）之间的数。可读写。Single 类型。
ColorType返回或设置应用于指定图片或 OLE 对象的颜色转换类型。MsoPictureColorType 类型，可读写。
Contrast返回或设置指定图片或 OLE 对象的对比度。该属性的值必须是 0.0（最低对比度）到 1.0（最高对比度）的数。可读写。Single 类型。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Crop返回或设置一个代表图像裁剪的 Crop 对象。可读写。
CropBottom返回或设置从指定图片或 OLE 对象的底部裁剪下的磅数。Single 类型，可读写。
CropLeft返回或设置从指定图片或 OLE 对象的左边裁剪下的磅数。Single 类型，可读写。
CropRight返回或设置从指定图片或 OLE 对象的右边裁剪下的磅数。Single 类型，可读写。
CropTop返回或设置从指定图片或 OLE 对象的顶部裁剪下的磅数。Single 类型，可读写。
Parent返回一个 Object 类型的值，该值代表指定 PictureFormat 对象的父对象。
TransparencyColor返回或设置指定图片的透明色，以红、绿、蓝 (RGB) 值表示。Long 类型，可读写。
TransparentBackground 如果图片中使用透明色定义的部分实际上看似透明，则该属性值为 MsoTrue。可以使用 TransparencyColor 属性设置透明色。该属性仅应用于位图。MsoTriState 类型，可读写。
