PageNumbers 对象成员

PageNumber 对象的集合，这些对象代表单个页眉或页脚中的页码。

方法

 名称说明
Add返回一个 PageNumber 对象，该对象代表添加到节中的页眉或页脚中的页码。
Item返回集合中的单个 PageNumber 对象。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
ChapterPageSeparator返回或设置章节号和页码之间的分隔字符。可读/写 WdSeparatorType。
Count返回一个 Long 类型的值，该值代表集合中的页码数。只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
DoubleQuote 如果 Microsoft Word 将指定的 PageNumbers 对象置于双引号 (") 中，则该属性值为 True。Boolean 类型，可读写。
HeadingLevelForChapter返回或设置文档中章节标题应用的标题级别样式。Long 类型，可读写。
IncludeChapterNumber 如果页码或题注标签中包含章节号，则该属性值为 True。Boolean 类型，可读写。
NumberStyle返回或设置一个代表编号样式的 WdPageNumberStyle 常量。可读写。
Parent返回一个 Object 类型的值，该值代表指定 PageNumbers 对象的父对象。
RestartNumberingAtSection 如果指定节的页码从 1 重新开始，则该属性值为 True。Boolean 类型，可读写。
ShowFirstPageNumber 如果在节中第一页显示页码，则该属性值为 True。Boolean 类型，可读写。
StartingNumber返回或设置注释编号、行号或页码的起始值。可读/写 Long 类型。
