TabStop 对象成员

代表单个制表位。TabStop 对象是 TabStops 集合的一个成员。TabStops 集合代表一个段落或一组段落中的所有自定义和默认制表位。

方法

 名称说明
Clear删除指定的自定义制表位。

属性

 名称说明
Alignment返回或设置一个 WdTabAlignment 常量，该常量代表指定制表位的对齐方式，可读写。
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
CustomTab 如果指定的制表位为自定义制表位，则该属性值为 True。Boolean 类型，只读。
Leader返回或设置指定 TabStop 对象的前导字符。WdTabLeader 类型，可读写。
Next返回集合中的下一个制表位。只读。
Parent返回一个 Object 类型的值，该值代表指定 TabStop 对象的父对象。
Position返回或设置制表位相对于左边距的位置。可读写 Single 类型。
Previous返回集合中的上一个制表位。只读。
