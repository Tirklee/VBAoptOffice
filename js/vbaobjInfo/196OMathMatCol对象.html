OMathMatCol 对象成员

代表矩阵列。OMathMatCol 对象是 OMathMatCols 集合的成员。

方法

 名称说明
Delete 删除所指定的矩阵列。

属性

 名称说明
Align返回或设置一个 WdOMathHorizAlignType 常量，该常量代表矩阵列中参数的水平对齐方式。可读写。
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Args返回一个 OMathArgs 对象，该对象代表矩阵列中的参数。只读。
ColIndex返回一个 Long 类型的值，该值代表列在矩阵列集合中的序号位置。只读。
Creator返回一个 32 位的整数，该整数表示在其中创建加载项的应用程序。只读 Long 类型。
Parent返回一个 Object 类型的值，该值代表指定 OMathMatCol 对象的父对象。
