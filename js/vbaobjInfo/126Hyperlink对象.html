Hyperlink 对象成员

代表一个超链接。Hyperlink 对象是 Hyperlinks 集合的一个成员。

方法

 名称说明
AddToFavorites创建文档或超链接的快捷方式，并将其添加到“收藏夹”。
CreateNewDocument新建一篇链接到指定超链接的文档。
Delete删除指定的超链接。
Follow显示一个与指定的 Hyperlink 对象相关的缓存文档（如果该文档已下载）。否则，此方法将解析该超链接，下载目标文档，并在相应的应用程序中显示此文档。

属性

 名称说明
Address返回或设置指定超链接的地址（例如，文件名或 URL）。String 类型，可读写。
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
EmailSubject返回或设置指定超链接的主题行的文本字符串。String 类型，可读写。
ExtraInfoRequired 如果在解析指定的超链接时需要额外信息，则该属性值为 True。Boolean 类型，只读。
Name返回指定对象的名称。只读 String 类型。
Parent返回一个 Object 类型的值，该值代表指定 Hyperlink 对象的父对象。
Range返回一个 Range 对象，该对象代表超链接中包含的文档部分。
ScreenTip返回或设置当鼠标指针置于指定超链接之上时显示的“屏幕提示”文字。String 类型，可读写。
Shape返回一个指定超链接或图表顶点的 Shape 对象。
SubAddress返回或设置指定超链接目标中命名的位置。String 类型，可读写。
Target返回或设置在其中加载超链接的图文框或窗口的名称。可读写 String 类型。
TextToDisplay返回或设置文档中指定超链接的可视文本。String 类型，可读写。
Type返回超链接的类型。只读 MsoHyperlinkType 类型。
