OMathMat 对象成员

代表一个公式矩阵。

属性

 名称说明
Align返回或设置一个 WdOMathVertAlignType 常量，该常量代表矩阵的垂直对齐方式。可读写。
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Cell返回一个 OMath 对象，该对象代表矩阵中的单元格。只读。
ColGap返回或设置一个 Long 类型的值，该值代表矩阵中各列之间的间距。可读写。
ColGapRule返回或设置一个 WdOMathSpacingRule 常量，该常量代表矩阵中列间距的间距规则。可读写。
Cols返回一个 OMathMatCols 集合，该集合代表矩阵中的列。只读。
ColSpacing返回或设置一个 Long 类型的值，该值代表矩阵中列的间距。可读写。
Creator返回一个 32 位的整数，该整数表示在其中创建加载项的应用程序。只读 Long 类型。
Parent返回一个 Object 类型的值，该值代表指定 OMathMat 对象的父对象。
PlcHoldHidden返回或设置一个 Boolean 类型的值，该值代表是否隐藏矩阵中的占位符。如果为 True，则隐藏占位符文本。可读写。
Rows返回一个 OMathMatRows 对象，该对象代表矩阵中的行。只读。
RowSpacing返回或设置一个 Long 类型的值，该值代表矩阵中行的间距。可读写。
RowSpacingRule返回或设置一个 WdOMathSpacingRule 常量，该常量代表矩阵中行的间距规则。可读写。
