Word 开发人员参考
Axis 对象成员

以后的版本中将提供关于此成员的说明。

方法

 名称说明
Delete删除对象。
Select选择对象。

属性

 名称说明
Application如果不与对象识别符一起使用，则该属性返回一个代表 Microsoft Word 应用程序的 Application 对象。如果与对象识别符一起使用，则该属性返回代表指定对象的创建者的 Application 对象（可对自动操作对象使用该属性来返回该对象的应用程序）。只读。
AxisBetweenCategories 如果数值轴与分类坐标轴相交于分类之间，则该属性值为 True。可读/写 Boolean 类型。
AxisGroup返回坐标轴组的类型。只读 XlAxisGroup。
AxisTitle返回指定坐标轴的标题。只读 AxisTitle 对象。
BaseUnit返回或设置指定分类轴的基本单位。可读/写 XlTimeUnit。
BaseUnitIsAuto 如果由 Microsoft Word 为指定的分类轴选择适当的基本单位，则为 True。默认值为 True。可读/写 Boolean 类型。
Border返回对象的边框。只读 ChartBorder 对象。
CategoryNames以文本数组形式返回或设置指定坐标轴中所有分类的名称。可读/写 Variant 类型。
CategoryType返回或设置分类轴类型。可读/写 XlCategoryType。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Crosses返回或设置指定坐标轴与其他坐标轴相交的点。可读/写 Long 类型。
CrossesAt返回或设置数值轴中与分类坐标轴的交点。仅应用于数值轴。可读/写 Double 类型。
DisplayUnit返回或设置数值轴的单位标签。可读/写 XlDisplayUnit、xlCustom 或 xlNone 常量。
DisplayUnitCustom如果 DisplayUnit 属性的值为 xlCustom，则返回或设置显示的单位的值。可读写 Double。
DisplayUnitLabel返回指定坐标轴的 DisplayUnitLabel 对象。如果 HasDisplayUnitLabel 属性设置为 False，则返回 null。只读。
Format返回对象的线条、填充和效果格式。只读 ChartFormat。
HasDisplayUnitLabel 如果由 DisplayUnit 或 DisplayUnitCustom 属性指定的标签显示在指定坐标轴上，则为 True。默认值为 True。可读/写 Boolean 类型。
HasMajorGridlines 如果坐标轴有主要网格线，则为 True。可读/写 Boolean 类型。
HasMinorGridlines 如果坐标轴有次要网格线，则为 True。可读/写 Boolean 类型。
HasTitle 如果坐标轴或图表有可见标题，则为 True。可读/写 Boolean 类型。
Height以磅为单位返回对象的高度。只读 Double 类型。
Left以磅为单位返回从对象左边缘至图表区左边缘之间的距离。只读 Double 类型。
LogBase在使用对数刻度时返回或设置对数的底。可读/写 Double 类型。
MajorGridlines返回指定坐标轴的主要网格线。只读 Gridlines 对象。
MajorTickMark返回或设置指定坐标轴的主刻度线类型。可读/写 XlTickMark。
MajorUnit返回或设置数值轴的主要单位。可读/写 Double 类型。
MajorUnitIsAuto 如果 Microsoft Word 计算数值轴的主要单位，则为 True。可读/写 Boolean 类型。
MajorUnitScale当 CategoryType 属性设置为 xlTimeScale 时，返回或设置分类轴的主要单位刻度值。可读/写 XlTimeUnit。
MaximumScale返回或设置数值轴上的最大值。可读/写 Double 类型。
MaximumScaleIsAuto 如果 Microsoft Word 计算数值轴的最大值，则为 True。可读/写 Boolean 类型。
MinimumScale返回或设置数值轴上的最小值。可读/写 Double 类型。
MinimumScaleIsAuto 如果 Microsoft Word 计算数值轴的最小值，则为 True。可读/写 Boolean 类型。
MinorGridlines返回指定坐标轴的次要网格线。只读 Gridlines 对象。
MinorTickMark返回或设置指定坐标轴的次要刻度线类型。可读/写 XlTickMark。
MinorUnit返回或设置数值轴上的次要刻度单位。可读/写 Double 类型。
MinorUnitIsAuto 如果 Microsoft Word 计算数值轴的次要刻度单位，则为 True。可读/写 Boolean 类型。
MinorUnitScale当 CategoryType 属性设置为 xlTimeScale 时，返回或设置分类轴的次要单位刻度值。可读/写 XlTimeUnit。
Parent返回指定对象的父对象。只读 Object 对象。
ReversePlotOrder 如果 Microsoft Word 绘制数据点的顺序为从后往前，则为 True。可读/写 Boolean 类型。
ScaleType返回或设置数值轴的刻度类型。可读/写 XlScaleType。
TickLabelPosition说明指定坐标轴上刻度线标志的位置。可读/写 XlTickLabelPosition。
TickLabels返回指定坐标轴的刻度线标签。只读 TickLabels 对象。
TickLabelSpacing返回或设置刻度线标签之间的分类数或系列数。可读/写 Long 类型。
TickLabelSpacingIsAuto返回或设置表示是否自动设置刻度标签间距的值。可读/写 Boolean 类型。
TickMarkSpacing返回或设置刻度线之间的分类数或系列数。可读/写 Long 类型。
Top以磅为单位返回从对象上边缘至第一行顶端（在工作表中）或图表区顶端（在图表中）的距离。只读 Double 类型。
Type返回坐标轴类型。只读 XlAxisType。
Width以磅为单位返回对象的宽度。只读 Double 类型。
