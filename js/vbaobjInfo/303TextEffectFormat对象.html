TextEffectFormat 对象成员

包含应用于艺术字对象的属性和方法。

方法

 名称说明
ToggleVerticalText将指定的艺术字中的文字排列从水平切换为垂直，反之亦然。

属性

 名称说明
Alignment返回或设置一个 MsoTextEffectAlignment 常量，该常量代表指定文本效果的对齐方式，可读写。
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
FontBold设置指定艺术字图形的字体为加粗。MsoTriState 类型，可读写。
FontItalic将艺术字文字设置为倾斜。MsoTriState 类型，可读写。
FontName返回或设置首字下沉字母的字体名称。String 类型，可读写。
FontSize以磅为单位返回或设置指定的艺术字字体大小。Single 类型，可读写。
KernedPairs代表经过字距调整的艺术字对象中的字符对。MsoTriState 类型，可读写。
NormalizedHeight 如果指定艺术字中所有字符（不论大小写）的高度相同，则该属性值为 MsoTrue。MsoTriState 类型，可读写。
Parent返回一个 Object 类型的值，该值代表指定 TextEffectFormat 对象的父对象。
PresetShape返回或设置指定艺术字的形状。MsoPresetTextEffectShape 类型，可读写。
PresetTextEffect返回或设置指定艺术字的样式。该属性的值对应于“艺术字库”对话框（“插入”菜单）中的格式（从左到右，从上到下进行编号）。MsoPresetTextEffect 类型，可读/写。
RotatedChars 如果指定的艺术字中的字符相对于艺术字边框形状旋转 90 度，则该属性值为 MsoTrue。如果指定的艺术字中的字符相对于边框形状保持原来的方向，则该属性值为 MsoFalse。MsoTriState 类型，可读写。
Text返回或设置指定对象中的文本。String 类型，可读写。
Tracking该属性返回或设置指定的艺术字中分配给每个字符的水平间距与字符宽度之比。Single 类型，可读写。
