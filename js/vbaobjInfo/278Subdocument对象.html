Subdocument 对象成员

代表文档或范围中的一个子文档。Subdocument 对象是 Subdocuments 集合的一个成员。Subdocuments 集合包含范围或文档中的所有子文档。

方法

 名称说明
Delete删除指定的子文档。
Open打开指定的子文档。返回一个代表已打开的子文档的 Document 对象。
Split将一篇现有子文档拆分为两篇子文档，在主控文档视图或大纲视图中这两篇文档处于同一级别。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
HasFile 如果将指定的子文档保存为一个文件，则该属性值为 True。Boolean 类型，只读。
Level返回用来创建子文档的标题级别。只读 Long 类型。
Locked 如果主控文档中的子文档已被锁定，则该属性值为 True。可读写 Boolean 类型。
Name返回指定对象的名称。只读 String 类型。
Parent返回一个 Object 类型的值，该值代表指定 Subdocument 对象的父对象。
Path返回指定子文档的磁盘或 Web 路径。只读 String 类型。
Range返回一个 Range 对象，该对象代表子文档中所含的文档部分。
