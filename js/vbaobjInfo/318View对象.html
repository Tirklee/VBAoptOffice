View 对象成员

包含窗口或窗格的视图属性（如全部显示、域底纹和网格线）。

方法

 名称说明
CollapseOutline将选定内容或指定范围内的正文折叠一个标题级别。
ExpandOutline按照一个标题级别展开选定内容下的文本。
NextHeaderFooter根据视图中显示的是页眉还是页脚，移动到下一页眉或页脚。
PreviousHeaderFooter根据视图中显示的是页眉还是页脚，移动到前一页眉或页脚。
ShowAllHeadings在显示所有文本（包括标题和正文）和仅显示标题之间切换。
ShowHeading显示所有高于指定标题级别的标题，隐藏子标题和正文。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
ConflictMode如果文档处于冲突模式视图，则该属性值为 True。可读写。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
DisplayBackgrounds返回或设置一个 Boolean 类型的值，该值代表在页面视图中显示文档时，是否显示背景色和图像。
DisplayPageBoundaries 如果为 True，则显示文档顶部和底部页边距（空白区域）并在页面间显示灰色区域（灰色间隔）。Boolean 类型，可读写。
Draft 如果窗口中的所有文本以具有最少格式信息的相同的 sans-serif 字体显示以加速显示，则该属性值为 True。Boolean 类型，可读写。
FieldShading返回或设置域的屏幕底纹。WdFieldShading 类型，可读写。
FullScreen 如果窗口为全屏显示，则该属性值为 True。Boolean 类型，可读写。
Magnifier 如果为 True，则在打印预览中将鼠标指针显示为放大镜形状，表示用户可以通过单击来放大页面的某个区域或通过缩小显示比例查看整页或展开多页。Boolean 类型，可读写。
MailMergeDataView 如果在指定窗口中显示邮件合并数据，而不显示邮件合并域，则该属性值为 True。Boolean 类型，可读写。
MarkupMode返回或设置一个 WdRevisionsMode 常量，该常量代表修订的显示模式。
Panning返回或设置 Boolean 类型的值，该值表示 Microsoft Word 是否处于全景模式。可读写。
Parent返回一个 Object 类型的值，该值代表指定 View 对象的父对象。
ReadingLayout设置或返回一个 Boolean 类型的值，该值表示是否在阅读版式视图中查看文档。
ReadingLayoutActualView设置或返回一个 Boolean 类型的值，该值代表是否使用与打印页面相同的版式在阅读版式视图中显示页面。
ReadingLayoutAllowEditing返回 Boolean 值，该值表示是否允许在阅读布局模式下编辑文本。可读/写。
ReadingLayoutAllowMultiplePages设置或返回一个 Boolean 类型的值，该值代表阅读版式视图是否并排显示两页。
ReadingLayoutTruncateMargins返回或设置一个 WdReadingLayoutMargin 常量，该常量代表当在阅读版式中查看文档时是显示还是隐藏边距。可读写。
Reviewers返回一个代表所有审阅者的 Reviewers 对象。
RevisionsBalloonShowConnectingLines 如果为 True，则 Microsoft Word 显示从文本到修订和批注气球之间的连接线。Boolean 类型，可读写。
RevisionsBalloonSide设置或返回一个 WdRevisionsBalloonMargin 常量，指定 Word 是否在文档的左边距或右边距中显示修订气球。
RevisionsBalloonWidth设置或返回一个 Single 类型的值，该值代表 Microsoft Word 中用于指定修订气球宽度的全局设置。可读写。
RevisionsBalloonWidthType设置或返回一个 WdRevisionsBalloonWidthType 常量，该常量代表指定 Microsoft Word 如何测量修订气球的宽度的全局设置。可读写。
RevisionsView设置或返回一个代表全局选项的 WdRevisionsView 常量，该选项用于指定 Word 是显示文档的原始版本还是应用了修订和格式更改的版本。可读写。
SeekView返回或设置在页面视图中显示的文档元素。WdSeekView 类型，可读写。
ShadeEditableRanges返回或设置一个 Long 类型的值，该值代表是否将底纹应用于文档中用户有权修改的区域。
ShowAll 如果显示所有非打印字符（如隐藏文字、制表符、空格和段落标记），则该属性值为 True。可读写 Boolean 类型。
ShowAnimation 如果显示文本动画，则该属性值为 True。Boolean 类型，可读写。
ShowBookmarks 如果在每个书签的首尾显示方括号，则该属性值为 True。Boolean 类型，可读写。
ShowComments 如果为 True，则 Microsoft Word 在文档中显示批注。Boolean 类型，可读写。
ShowCropMarks返回或设置 Boolean 值，该值表示是否在页面的角显示裁剪标记，以指示页边距的位置。可读/写。
ShowDrawings 如果在页面视图中显示由绘图工具创建的对象，则该属性值为 True。Boolean 类型，可读写。
ShowFieldCodes 如果显示域代码，则该属性值为 True。Boolean 类型，可读写。
ShowFirstLineOnly 如果在大纲视图中仅显示正文的第一行，则该属性值为 True。Boolean 类型，可读写。
ShowFormat 如果在大纲视图中显示字符格式，则该属性值为 True。Boolean 类型，可读写。
ShowFormatChanges 如果为 True，则在启用“修订”功能后，Microsoft Word 显示对文档所作的格式更改。Boolean 类型，可读写。
ShowHiddenText 如果显示具有隐藏格式的文本，则该属性值为 True。Boolean 类型，可读写。
ShowHighlight 如果显示和打印文档中突出显示格式的文本，则该属性值为 True。Boolean 类型，可读写。
ShowHyphens 如果为 True，则显示可选连字符，该连字符指明在何处断开处于行尾的字。Boolean 类型，可读写。
ShowInkAnnotations返回或设置一个 Boolean 类型的值，该值表示显示还是隐藏手写墨迹注释。如果值为 True，则显示墨迹注释。如果值为 False，则隐藏墨迹注释。
ShowInsertionsAndDeletions 如果为 True，则 Microsoft Word 显示用“修订”功能在文档中插入和删除的文本。Boolean 类型，可读写。
ShowMainTextLayer 如果在显示页眉和页脚区域时同时显示指定文档的文字，则该属性值为 True。该属性等效于“页眉和页脚”工具栏上的“显示/隐藏文档文字”按钮。Boolean 类型，可读写。
ShowMarkupAreaHighlight返回或设置 Boolean 值，该值表示显示修订和注释气球的标记区域是否带底纹。可读/写。
ShowObjectAnchors 如果在可在页面视图中定位的项目附近显示对象锁定标记，则该属性值为 True。Boolean 类型，可读写。
ShowOptionalBreaks 如果 Microsoft Word 显示可选换行符，则该属性值为 True。Boolean 类型，可读写。
ShowOtherAuthors如果其他作者的联机状态应该在文档中可见，则该属性值为 True。可读写 Boolean。
ShowParagraphs 如果显示段落标记，则该属性值为 True。Boolean 类型，可读写。
ShowPicturePlaceHolders 如果将空白方框作为图片的占位符进行显示，则该属性值为 True。Boolean 类型，可读写。
ShowRevisionsAndComments 如果为 True，则 Microsoft Word 显示使用“修订”功能对文档所作的修订和批注。Boolean 类型，可读写。
ShowSpaces 如果显示空格字符，则该属性值为 True。Boolean 类型，可读写。
ShowTabs 如果显示制表符，则该属性值为 True。Boolean 类型，可读写。
ShowTextBoundaries 如果在页面视图中显示页边距、文本列、对象和框架周围的虚线，则该属性值为 True。Boolean 类型，可读写。
ShowXMLMarkup返回一个 Long 类型的值，该值表示是否在文档中显示 XML 标记。
SplitSpecial返回或设置活动窗口的窗格。WdSpecialPane 类型，可读写。
TableGridlines 如果显示表格虚框，则该属性值为 True。Boolean 类型，可读写。
Type返回或设置视图的类型。可读/写 WdViewType 类型。
WrapToWindow 如果文字在文档窗口的右边缘换行，而不是在右边距或文本栏的右边界，则该属性值为 True。Boolean 类型，可读写。
Zoom返回一个 Zoom 对象，该对象代表指定视图的缩放情况。
