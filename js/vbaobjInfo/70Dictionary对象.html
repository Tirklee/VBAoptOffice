Dictionary 对象成员

代表一个词典。代表自定义词典的 Dictionary 对象是 Dictionaries 集合的成员。其他词典对象由 Languages 集合的属性返回；这些属性包括 ActiveSpellingDictionary、ActiveGrammarDictionary、ActiveThesaurusDictionary 和 ActiveHyphenationDictionary 属性。

方法

 名称说明
Delete删除指定的词典。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
LanguageID返回或设置一个 WdLanguageID 常量，该常量代表指定对象的语言。可读写。
LanguageSpecific 如果自定义词典只用于格式为特定语言的文字，则该属性值为 True。Boolean 类型，可读写。
Name返回指定对象的名称。只读 String 类型。
Parent返回一个 Object 类型的值，该值代表指定 Dictionary 对象的父对象。
Path返回指定词典的路径。只读 String 类型。
ReadOnly 如果指定的词典无法更改，则该属性为 True。只读 Boolean 类型。
Type返回词典的类型。只读 WdDictionaryType 类型。
