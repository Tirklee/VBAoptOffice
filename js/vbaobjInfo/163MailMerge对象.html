MailMerge 对象成员

代表 Word 中的邮件合并功能。

方法

 名称说明
Check模拟邮件合并操作，在出错时停止合并操作并报告每个错误。
CreateDataSource创建一个 Microsoft Word 文档，该文档使用表格保存邮件合并数据。
CreateHeaderSource创建一个保存域名记录的 Microsoft Word 文档，该记录用于替换邮件合并中的数据源域名记录。
EditDataSource打开或者切换到邮件合并数据源。
EditHeaderSource打开附加到邮件合并主文档的域名源，如果已经打开，则激活该域名源。
EditMainDocument激活与指定域名源或者数据源文档相关的邮件合并主文档。
Execute执行指定的邮件合并操作。
OpenDataSource将一个数据源附加到指定文档，如果该文档不是主文档，则将变成主文档。
OpenHeaderSource将一个邮件合并域名源附加到指定文档。
ShowWizard在文档中显示“邮件合并向导”。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
DataSource返回一个 MailMergeDataSource 对象，该对象引用附加到邮件合并主文档的数据源。只读。
Destination返回或设置邮件合并结果的目标。WdMailMergeDestination 类型，可读写。
Fields返回一个只读 MailMergeFields 集合，该集合代表指定文档中的所有邮件合并域。
HighlightMergeFields 如果为 True，则在文档中突出显示合并域。Boolean 类型，可读写。
MailAddressFieldName返回或设置包含电子邮件地址的域名，如果邮件合并的目标文档是电子邮件，就使用该地址。String 类型，可读写。
MailAsAttachment 在邮件合并目标文档是电子邮件或传真时，如果将合并文档作为附件发送，则该属性值为 True。Boolean 类型，可读写。
MailFormat返回一个 WdMailMergeMailFormat 常量，当邮件合并的目标文档是电子邮件时，该常量代表所用的格式。可读写。
MailSubject当邮件合并的目标文档是电子邮件时，返回或设置主题行。String 类型，可读写。
MainDocumentType返回或设置邮件合并主文档的类型。WdMailMergeMainDocType 类型，可读写。
Parent返回一个 Object 类型的值，该值代表指定 MailMerge 对象的父对象。
ShowSendToCustom返回或设置一个 String 类型的值，该值与“邮件合并向导”的“完成合并”步骤（第六步）中自定义按钮上的题注相对应。可读写。
State返回邮件合并操作的当前状态。WdMailMergeState 类型，只读。
SuppressBlankLines 在邮件合并主文档中的邮件合并域为空时，如果删除该空行，则该属性值为 True。Boolean 类型，可读写。
ViewMailMergeFieldCodes 如果在邮件合并主文档中显示合并域的名称，则为 True。如果显示当前记录的信息，则为 False。Long 类型，可读写。
WizardState返回或设置一个 Long 类型的值，该值指示文档的当前“邮件合并向导”步骤。WizardState 方法返回一个等于当前“邮件合并向导”步骤的数值；如果返回零 (0)，则代表“邮件合并向导”已经关闭。可读写。
