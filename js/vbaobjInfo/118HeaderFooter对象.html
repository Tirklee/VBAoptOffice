HeaderFooter 对象成员

代表单一页眉或页脚。HeaderFooter 对象是 HeadersFooters 集合的一个成员。HeadersFooters 集合包含指定文档节中的所有页眉和页脚。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Exists 如果存在指定的 HeaderFooter 对象，则该属性值为 True。Boolean 类型，可读写。
Index返回 WdHeaderFooterIndex，代表文档或节中指定的页眉或页脚。只读。
IsHeader 如果指定的 HeaderFooter 对象是页眉，则该属性值为 True。Boolean 类型，只读。
LinkToPrevious 如果指定页眉或页脚链接至前一节中相应的页眉或页脚，则该属性值为 True。Boolean 类型，可读写。
PageNumbers返回一个 PageNumbers 集合，该集合代表指定页眉或页脚中包含的所有页码域。
Parent返回一个 Object 类型的值，该值代表指定 HeaderFooter 对象的父对象。
Range返回一个 Range 对象，该对象代表指包含在定页眉或页脚中的文档部分。
Shapes返回一个 Shapes 集合，该集合代表页眉或页脚中的所有的 Shape 对象。只读。
