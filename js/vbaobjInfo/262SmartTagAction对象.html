SmartTagAction 对象成员

代表智能标记的单个操作。

方法

  名称 说明
Execute 执行指定的智能标记操作。

属性

  名称 说明
ActiveXControl 返回一个 Object 类型的值，该值代表“文档操作”任务窗格中显示的 ActiveX 控件。
Application 返回一个代表 Microsoft Word 应用程序的 Application 对象。
CheckboxState 返回或设置一个 Boolean 类型的值，该值代表是否选中智能文档中的某个复选框。
Creator 返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
ExpandDocumentFragment 设置或返回一个 Boolean 类型的值，该值代表将指定文档碎片控件在“文档操作”任务窗格中展开或折叠。
ExpandHelp 设置或返回一个 Boolean 类型的值，代表将指定智能文档的“帮助”文字控件在“文档操作”任务窗格中展开或折叠。
ListSelection 设置或返回一个 Long 类型的值，该值代表智能文档列表框控件中选定项目的索引号。
Name 返回指定对象的名称。只读 String 类型。
Parent 返回一个 Object 类型值，该值代表指定 SmartTagAction 对象的父对象。
PresentInPane 返回一个 Boolean 类型的值，该值代表智能文档控件当前是否显示在“文档操作”任务窗格中。
RadioGroupSelection 设置或返回一个 Long 类型的值，代表在智能文档中的一组单选按钮控件中选定项目的索引号。
TextboxText 设置或返回一个 String 类型的值，该值代表智能文档文本框控件中的文字。
Type 返回一个 WdSmartTagControlType 类型，代表在“文档操作”任务窗格中显示的“智能文档”控件的类型。
