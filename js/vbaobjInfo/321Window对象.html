Window 对象成员

代表一个窗口。许多文档特征（如滚动条和标尺）实际上是窗口的属性。

方法

 名称说明
Activate激活指定窗口。
Close关闭指定的窗口。
GetPoint返回指定区域或图形的屏幕坐标。
LargeScroll将窗口或窗格滚动指定的屏数。
NewWindow打开一个新窗口，该窗口与指定窗口显示同一个文档。返回一个 Window 对象。
PageScroll使指定的窗格或窗口逐页滚动。
PrintOut打印指定窗口中显示的全部或部分文档。
RangeFromPoint返回 Range 或 Shape 对象，该对象位于由屏幕位置坐标对指定的位置。
ScrollIntoView滚动文档窗口，以便在文档窗口显示指定的区域或图形。
SetFocus在指定的文档窗口为电子邮件设置焦点。
SmallScroll将窗口或窗格滚动指定的行数。
ToggleRibbon 显示或隐藏功能区。
ToggleShowAllReviewers显示或隐藏包含备注和修订的文档中的所有备注。

属性

 名称说明
Active 如果指定窗口处于活动状态，则该属性值为 True。Boolean 类型，只读。
ActivePane返回一个 Pane 对象，该对象代表指定窗口的活动窗格。只读。
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Caption返回或设置显示在文档或应用程序窗口标题栏中的窗口题注文字。String 类型，可读写。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
DisplayHorizontalScrollBar 如果在指定的窗口中显示水平滚动条，则该属性值为 True。Boolean 类型，可读写。
DisplayLeftScrollBar 如果垂直滚动条显示在文档窗口左侧，则该属性值为 True。Boolean 类型，可读写。
DisplayRightRuler 如果垂直标尺显示在页面视图中文档窗口右侧，则该属性值为 True。Boolean 类型，可读写。
DisplayRulers 如果在指定的窗口或窗格中显示标尺，则该属性值为 True。Boolean 类型，可读写。
DisplayScreenTips 如果批注、脚注、尾注和超链接以提示形式显示，则该属性值为 True。Boolean 类型，可读写。
DisplayVerticalRuler 如果在指定的窗口或窗格中显示垂直标尺，则该属性值为 True。Boolean 类型，可读写。
DisplayVerticalScrollBar 如果在指定的窗口中显示垂直滚动条，则该属性值为 True。Boolean 类型，可读写。
Document返回一个 Document 对象，该对象与指定窗格、窗口或选定内容相关。只读。
DocumentMap 如果显示文档结构图，则该属性值为 True。Boolean 类型，可读写。
EnvelopeVisible 如果电子邮件的页眉在文档窗口是可见的，则该属性值为 True。默认值为 False。Boolean 类型，可读写。
Height返回或设置窗口的高度。Long 类型，可读写。
HorizontalPercentScrolled返回或设置水平滚动条位置（以文档宽度的百分比表示）。可读写 Long 类型。
IMEMode返回或设置日文输入法编辑器 (IME) 的默认启动模式。WdIMEMode 类型，可读写。
Index返回 Long 值，该值表示项目在集合中的位置。只读。
Left返回或设置一个 Long 类型的值，该值代表指定窗口的水平位置，以磅为单位。可读写。
Next返回打开的文档窗口集合中的下一个文档窗口。只读。
Panes返回一个 Panes 集合，该集合指定窗口的所有窗格。
Parent返回一个 Object 类型的值，该值代表指定 Window 对象的父对象。
Previous返回打开的文档窗口集合中的上一个文档窗口。只读。
Selection返回一个 Selection 对象，该对象代表一个选定范围或插入点。只读。
ShowSourceDocuments返回或设置一个 WdShowSourceDocuments 常量，该常量代表 Microsoft Word 在比较和合并过程之后显示源文档的方式。可读写。
Split 如果将窗口拆分为多个窗格，则该属性值为 True。Boolean 类型，可读写。
SplitVertical返回或设置指定窗口的垂直拆分百分比。Long 类型，可读写。
StyleAreaWidth该属性返回或设置样式区的宽度，以磅为单位。Single 类型，可读写。
Thumbnails设置或返回一个 Boolean 类型的值，代表是否沿 Microsoft Word 文档窗口的左侧显示文档页面的缩略图。
Top返回或设置指定文档窗口的垂直位置（以磅为单位）。可读写 Long 类型。
Type返回窗口的类型。只读 WdWindowType 类型。
UsableHeight返回指定的文档窗口中活动工作区的高度（以磅为单位）。Long 类型，只读。
UsableWidth返回指定的文档窗口中活动工作区的宽度（以磅为单位）。Long 类型，只读。
VerticalPercentScrolled返回或设置垂直滚动条的位置（以文档长度的百分比表示）。可读写 Long 类型。
View返回一个 View 对象，该对象代表指定窗口或窗格的视图。
Visible 如果指定的对象可见，则该属性值为 True。Boolean 类型，可读写。
Width返回或设置指定文档窗口的宽度（以磅为单位）。可读写 Long 类型。
WindowNumber该属性返回在指定窗口中显示的文档窗口序号。例如，如果窗口标题为“Sales.doc:2”，则该属性返回数值 2。Long 类型，只读。
WindowState返回或设置指定文档窗口或任务窗口的状态。WdWindowState 类型，可读写。
