LineFormat 对象成员

代表线条和箭头的格式设置。对于线条，LineFormat 对象包含该线条自身的格式信息；对于具有边框的形状，该对象包含形状的边框的格式信息。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
BackColor返回或设置一个 ColorFormat 对象，该对象代表带图案线条的背景色。可读写。
BeginArrowheadLength返回或设置指定线条开头的箭头长度。MsoArrowheadLength 类型，可读写。
BeginArrowheadStyle返回或设置指定线条开头的箭头样式。MsoArrowheadStyle 类型，可读写。
BeginArrowheadWidth返回或设置指定线条开头的箭头宽度。MsoArrowheadWidth 类型，可读写。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
DashStyle返回或设置指定线条的虚线样式。MsoLineDashStyle 类型，可读写。
EndArrowheadLength返回或设置指定线条末尾箭头的长度。MsoArrowheadLength 类型，可读写。
EndArrowheadStyle返回或设置指定线条末尾箭头的样式。MsoArrowheadStyle 类型，可读写。
EndArrowheadWidth返回或设置指定线条末尾箭头的宽度。MsoArrowheadWidth 类型，可读写。
ForeColor返回或设置一个 ColorFormat 对象，该对象代表线条的前景色。可读写。
InsetPen 如果为 MsoTrue，则在指定形状内绘制线条。MsoTriState 类型，可读写。
Parent返回一个 Object 类型的值，该值代表指定 LineFormat 对象的父对象。
Pattern设置或返回代表应用于指定线条的图案的值。可读写 MsoPatternType 类型。
Style返回或设置线条格式样式。可读写 MsoLineStyle 类型。
Transparency返回或设置线条的透明度。可读写 Single 类型。
Visible 如果指定对象或应用于该对象的格式是可见的，则该属性值为 True。MsoTriState 类型，可读写。
Weight返回或设置指定线条的粗细（以磅为单位）。Single 类型，可读写。
