FormField 对象成员

代表单个窗体域。FormField 对象是 FormFields 集合的一个成员。

方法

 名称说明
Copy将指定窗体域复制到剪贴板。
Cut将指定窗体域从文档中移到剪贴板上。
Delete删除指定窗体域。
Select选择指定的对象。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
CalculateOnExit 如果退出指定的窗体域时自动更新对该域的引用，则该属性值为 True。Boolean 类型，可读写。
CheckBox返回一个 CheckBox 对象，该对象代表复选框窗体域。只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
DropDown返回一个 DropDown 对象，该对象代表下拉型窗体域。只读。
Enabled如果启用窗体域，则该属性值为 True。Boolean 类型，可读写。
EntryMacro返回或设置指定窗体域（CheckBox、DropDown 或 TextInput）的输入宏的名称。String 类型，可读写。
ExitMacro返回或设置指定窗体域（CheckBox、DropDown 或 TextInput）的退出宏的名称。String 类型，可读写。
HelpText返回或设置当窗体域获得焦点并且用户按 F1 时在消息框中显示的文字。String 类型，可读写。
Name返回或设置指定对象的名称。String 类型，可读写。
Next返回集合中的下一个窗体域。只读。
OwnHelp指定当窗体域具有焦点并且用户按 F1 时在消息框中显示的文本的源。Boolean 类型，可读写。
OwnStatus指定当窗体域具有焦点时状态栏中显示的文本的源。Boolean 类型，可读写。
Parent返回一个 Object 类型的值，该值代表指定 FormField 对象的父对象。
Previous返回集合中的前一窗体域。只读。
Range返回一个 Range 对象，该对象代表窗体域中包含的文档部分。
Result返回一个代表指定的窗体域的结果的 String。可读写。
StatusText返回或设置当窗体域具有焦点时状态栏中显示的文本。String 类型，可读写。
TextInput返回一个 TextInput 对象，该对象代表文本窗体域。
Type返回域的类型。只读 WdFieldType 类型。
