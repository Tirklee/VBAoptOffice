XMLNode 对象成员

表示应用于文档的单个 XML 元素。

方法

 名称说明
Copy 将指定的 XML 元素（不包括 XML 标记）复制到剪贴板。
Cut将指定的 XML 元素从文档中移到剪贴板上。
Delete删除 XML 文档中指定的 XML 元素。
RemoveChild 从指定的元素中删除子元素。
SelectNodes 返回一个 XMLNodes 集合，该集合代表与 XPath 参数匹配的所有子元素（它们的顺序与它们在指定的 XML 元素中出现的顺序相同）。
SelectSingleNode返回一个 XMLNode 对象，该对象代表指定的 XML 元素中第一个与 XPath 参数匹配的子元素。
SetValidationError更改向用户显示的指定节点的验证错误文本，并强制 Word 将节点报告为无效。
Validate 针对附加到文档的 XML 架构验证单个 XML 元素。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Attributes返回一个 XMLNodes 集合，该集合代表指定元素的属性。
BaseName返回一个 String 类型的值，该值代表不带任何前缀的元素名称。
ChildNodes返回一个 XMLNodes 集合，该集合代表指定元素的子元素。
ChildNodeSuggestions返回一个 XMLChildNodeSuggestions 集合，代表指定元素的子元素列表。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
FirstChild返回一个 DiagramNode 对象，该对象代表父节点的第一个子节点。只读。
HasChildNodes返回 Boolean 值，该值表示 XML 节点是否有子节点。只读。
LastChild返回一个 XMLNode 对象，该对象代表 XML 元素的最后一个子节点。
Level返回一个 WdXMLNodeLevel 常量，该常量代表 XML 元素是段落的一部分、是整个段落、包含在表格单元格中还是包含表格行。只读。
NamespaceURI返回 String 值，该值表示指定对象的架构命名空间的统一资源标识符 (URI)。只读。
NextSibling返回一个 XMLNode 对象，该对象代表文档中与指定元素同级的下一个元素。
NodeType返回一个代表节点类型的 WdXMLNodeType 常量。
NodeValue 返回或设置一个 String 类型的值，该值代表 XML 元素的值。可读写。
OwnerDocument返回一个代表指定 XML 元素的父文档的 Document 对象。
Parent返回一个 Object 类型的值，该值代表指定 XMLNode 对象的父对象。
ParentNode返回一个代表指定元素的父元素的 XMLNode 对象。
PlaceholderText设置或返回一个 String 类型的值，该值代表为不包含文本的元素显示的文本。
PreviousSibling返回一个 XMLNode 对象，该对象代表文档中与指定元素同级的前一个元素。
Range返回一个 Range 对象，该对象代表指定对象中包含的文档部分。只读。
Text返回或设置 XML 元素中包含的文本。String 类型，可读写。
ValidationErrorText返回一个 String 类型的值，代表关于 XMLNode 对象验证错误的说明。
ValidationStatus 返回一个 WdXMLValidationStatus 常量，该常量表示根据附加的架构，元素或属性是否有效。
WordOpenXML返回一个 String 类型的值，该值以 Microsoft Word Open XML 格式表示节点的 XML。只读。
XML返回 String 值，该值表示包含在 XML 节点中的文本（有或没有 XML 标记）。只读。
