PageSetup 对象成员
全部显示
全部隐藏

代表页面设置说明。PageSetup 对象包含文档的所有页面设置属性（如左边距、下边距和纸张大小）。

方法

 名称说明
SetAsTemplateDefault将指定的页面设置格式设为活动文档和所有基于活动模板的新文档的默认格式。
TogglePortrait在文档或节的纵向和横向页面方向之间切换。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
BookFoldPrinting 如果为 True，则 Microsoft Word 将文档打印为一系列小册子，这样可以折叠打印的页面并以书籍的形式阅读。Boolean 类型，可读写。
BookFoldPrintingSheets返回或设置一个 Long 类型的值，该值代表每一本小册子的页数。Boolean 类型，可读写。
BookFoldRevPrinting 如果为 True，则 Microsoft Word 反转用于双向或亚洲语言文档的书籍折页打印 （书籍折页打印：将文档打印成一个或多个小册子，以便折叠打印页，并将其作为书籍阅读。）的打印顺序。Boolean 类型，可读写。
BottomMargin返回或设置页面底边与正文文本边界之间的距离（以磅为单位）。Single 类型，可读写。
CharsLine返回或设置文档网格中每行的字符数。Single 类型，可读写。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
DifferentFirstPageHeaderFooter 如果第一页使用了不同的页眉或页脚，则为 True。该属性值可以是 True、False 或 wdUndefined。Long 类型，可读写。
FirstPageTray返回或设置用于文档或章节首页的纸盒。WdPaperTray 类型，可读写。
FooterDistance返回或设置页脚与页面底边之间的距离（以磅为单位）。Single 类型，可读写。
Gutter返回或设置文档或节的每一页的额外页边距的大小（以磅为单位），以备装订需要。Single 类型，可读写。
GutterPos返回或设置文档中的装订线位于哪一侧。WdGutterStyle 类型，可读写。
GutterStyle返回或设置 Microsoft Word 将装订线应用于当前文档时，基于从右向左的语言还是从左向右的语言。WdGutterStyleOld 类型，可读写。
HeaderDistance返回或设置页眉与页面顶端之间的距离（以磅为单位）。Single 类型，可读写。
LayoutMode返回或设置当前文档的版式模式。WdLayoutMode 类型，可读写。
LeftMargin返回或设置页面左边缘与正文左边界之间的距离（以磅为单位）。Single 类型，可读写。
LineNumbering返回或设置一个 LineNumbering 对象，该对象代表指定的 PageSetup 对象的行号。
LinesPage返回或设置文档网格中每页的行数。Single 类型，可读写。
MirrorMargins 如果对开页的内外边距宽度相等，则该属性值为 True。Long 类型，可读写。
OddAndEvenPagesHeaderFooter 如果指定的 PageSetup 对象的奇数页和偶数页具有不同的页眉和页脚，则该属性值为 True。Long 类型，可读写。
Orientation返回或设置页面的方向。可读写 WdOrientation 类型。
OtherPagesTray返回或设置用于文档或节中除第一页以外其他所有页的纸盒。WdPaperTray 类型，可读写。
PageHeight返回或设置页面高度（以磅为单位）。Single 类型，可读写。
PageWidth返回或设置页面宽度（以磅为单位）。Single 类型，可读写。
PaperSize返回或设置纸张大小。WdPaperSize 类型，可读写。
Parent返回一个 Object 类型的值，该值代表指定 PageSetup 对象的父对象。
RightMargin返回或设置页面右边距与正文右边界之间的距离（以磅为单位）。Single 类型，可读写。
SectionDirection返回或设置指定节的阅读顺序和对齐方式。WdSectionDirection 类型，可读写。
SectionStart返回或设置指定对象的分节符类型。WdSectionStart 类型，可读写。
ShowGrid您请求了仅用于 Macintosh 上的 Visual Basic 关键字的帮助。有关 PageSetup 对象的 ShowGrid 属性的信息，请查阅 Microsoft Office Macintosh Edition 附带的语言参考帮助。
SuppressEndnotes 如果在下一个没有隐藏尾注的节的末尾打印尾注，则该属性值为 True。Long 类型，可读写。
TextColumns返回一个 TextColumns 集合，该集合代表指定的 PageSetup 对象的文本列集。
TopMargin返回或设置页面的上边缘与正文文本的上边界之间的距离（以磅为单位）。可读写 Single 类型。
TwoPagesOnOne 如果 Microsoft Word 将指定文档的每两页打印在同一张纸上，则该属性值为 True。Boolean 类型，可读写。
VerticalAlignment返回或设置文档或节中每页文本的垂直对齐方式。可读写 WdVerticalAlignment 类型。
