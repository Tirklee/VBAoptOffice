ColorFormat 对象成员
全部显示
全部隐藏

代表单色对象的颜色，或以渐变/图案填充的对象的前景色或背景色。通过使用 RGB 属性，可以将颜色设置为明确的 RGB（红-绿-蓝）值。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Brightness返回一个 Single 类型的值，该值代表指定形状颜色的亮度。可读写。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
ObjectThemeColor 返回或设置一个 WdThemeColorIndex 常量，该常量代表颜色格式的主题颜色。可读写。
Parent返回一个 Object 类型的值，该值代表指定 ColorFormat 对象的父对象。
RGB返回或设置指定颜色的 RGB 值 （RGB 值：RGB 函数的返回值；以红、绿、蓝的数值组合来指定颜色，其数值范围是由 0（零）到 255 的整数。）（红-绿-蓝值），Long 类型，可读写。
TintAndShade返回一个 Single 类型的值，该值代表指定图形颜色的明暗。可读写。
Type返回或设置形状的颜色类型。MsoColorType 类型，只读。
