TableOfFigures 对象成员

代表文档中的单个图表目录。TableOfFigures 对象是 TablesOfFigures 集合的一个成员。TablesOfFigures 集合包含文档中的所有图表目录。

方法

 名称说明
Delete删除指定的图表目录。
Update更新图表目录中显示的项。
UpdatePageNumbers更新图表目录中项目的页码。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Caption返回或设置标识图表目录中的项目的标签。String 类型，可读写。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
HeadingStyles返回一个 HeadingStyles 对象，该对象代表用于编译目录或图表目录的其他样式（标题 1 - 标题 9 样式之外的样式）。只读。
HidePageNumbersInWeb返回或设置向网站发布时是否隐藏目录或图表目录中的页码。Boolean 类型，可读写。
IncludeLabel 如果在图表目录中包含题注标签和题注编号，则该属性值为 True。Boolean 类型，可读写。
IncludePageNumbers 如果图表目录中包含页码，则该属性值为 True。可读写 Boolean 类型。
LowerHeadingLevel返回或设置图表目录的结束标题级别。可读写 Long 类型。
Parent返回一个 Object 类型的值，该值代表指定 TableOfFigures 对象的父对象。
Range返回一个 Range 对象，该对象代表指定图表目录中所含的文档部分。
RightAlignPageNumbers 如果页码与图表目录的右边距对齐，则该属性值为 True。可读写 Boolean 类型。
TabLeader返回或设置图表目录中的目录项及其页码之间的字符。可读写 WdTabLeader 类型。
TableID返回或设置用于从 TOC 域生成图表目录的单字母标识符。可读写 String 类型。
UpperHeadingLevel返回或设置图表目录的起始标题级别。可读写 Long 类型。
UseFields 如果使用目录项 (TC) 域创建图表目录，则该属性值为 True。可读写 Boolean 类型。
UseHeadingStyles 如果使用内置标题样式创建图表目录，则该属性值为 True。可读写 Boolean 类型。
UseHyperlinks返回或设置在网站上发布图表目录项时是否将其设为超链接格式。可读写 Boolean 类型。
