Browser 对象成员

代表用于将插入点移至文档中对象的浏览器工具。该工具由垂直滚动条底端的三个按钮组成。

方法

 名称说明
Next将选定内容移到浏览器目标所指的下一项。使用 Target 属性可以更改浏览器目标。
Previous将选定内容移到浏览器目标所指的前一项。使用 Target 属性可以更改浏览器目标。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Parent返回一个 Object 类型的值，该值代表指定 Browser 对象的父对象。
Target返回或设置 Previous 方法和 Next 方法定位的文档项。WdBrowseTarget 类型，可读写。
