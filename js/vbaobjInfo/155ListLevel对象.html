ListLevel 对象成员

代表单个列表级别，该列表级别可以是项目符号或编号列表中的唯一级别，也可以是多级符号列表中九个级别之一。ListLevel 对象是 ListLevels 集合的一个成员。

方法

 名称说明
ApplyPictureBullet用图片项目符号设置段落或段落区域的格式。

属性

 名称说明
Alignment返回或设置一个 WdListLevelAlignment 常量，该常量代表列表模板的列表级别的对齐方式。可读写。
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Font返回或设置一个 Font 对象，该对象代表指定对象的字符格式。可读写 Font。
Index返回 Long 值，该值表示项目在集合中的位置。只读。
LinkedStyle返回或设置与指定 ListLevel 对象链接的样式的名称。String 类型，可读写。
NumberFormat返回或设置指定列表级别的数字格式。可读/写 String 类型。
NumberPosition返回或设置指定 ListLevel 对象的编号或项目符号的位置（以磅为单位）。Single 类型，可读写。
NumberStyle返回或设置 ListLevel 对象的编号样式。WdListNumberStyle 类型，可读写。
Parent返回一个 Object 类型的值，该值代表指定 ListLevel 对象的父对象。
PictureBullet返回一个 InlineShape 对象，该对象代表一个图片项目符号。
ResetOnHigher设置或返回指定列表级别从 1 开始重新编号之前必须显示的列表级别。Long 类型，可读写。
StartAt返回或设置指定 ListLevel 对象的起始编号。Long 类型，可读写。
TabPosition返回或设置指定 ListLevel 对象的制表符位置。Single 类型，可读写。
TextPosition返回或设置指定 ListLevel 对象的第二行换行文字的位置（以磅为单位）。Single 类型，可读写。
TrailingCharacter返回或设置在指定列表级别编号之后插入的字符。WdTrailingCharacter 类型，可读写。
