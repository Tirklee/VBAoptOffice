CheckBox 对象成员

代表一个复选框窗体域。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
AutoSize 如果为 True，则根据环绕文字的字体大小调整复选框或文本框的大小。如果为 False，则根据 Size 属性调整复选框或文本框的大小。Boolean 类型，可读写。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Default返回或设置默认的复选框值。如果选中默认值，则该属性值为 True。Boolean 类型，可读写。
Parent返回一个 Object 类型的值，该值代表指定 CheckBox 对象的父对象。
Size返回或设置复选框的大小。Single 类型，可读写。
Valid 如果指定的窗体域对象为有效的复选框窗体域，则该属性值为 True。Boolean 类型，只读。
Value 如果选中此复选框，则该属性值为 True。Boolean 类型，可读写。
