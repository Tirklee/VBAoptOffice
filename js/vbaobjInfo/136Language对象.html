Language 对象成员

代表 Microsoft Word 中用于校对或进行格式设置的语言。Language 对象是 Languages 集合的成员。

属性

 名称说明
ActiveGrammarDictionary返回一个 Dictionary 对象，该对象代表指定语言的活动语法词典。只读。
ActiveHyphenationDictionary返回一个 Dictionary 对象，该对象代表指定语言的活动断字词典。只读。
ActiveSpellingDictionary返回一个 Dictionary 对象，该对象代表指定语言的活动拼写词典。
ActiveThesaurusDictionary返回一个 Dictionary 对象，该对象代表指定语言的活动同义词库词典。
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
DefaultWritingStyle返回或设置指定语言的语法检查器使用的默认写作风格。String 类型，可读写。
ID返回一个标识指定语言的数值。只读 WdLanguageID 类型。
Name返回指定对象的名称。只读 String 类型。
NameLocal返回使用用户语言表示的校对工具语言的名称。只读 String 类型。
Parent返回一个 Object 类型的值，该值代表指定 Language 对象的父对象。
SpellingDictionaryType返回或设置校对工具类型。WdDictionaryType 类型，可读写。
WritingStyleList该属性返回一个字符串数组，该数组包含可用于指定语言所有写作风格的名称。Variant 类型，只读。
