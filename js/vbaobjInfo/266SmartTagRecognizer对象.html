SmartTagRecognizer 对象成员

代表所安装的用信息类型来标记文本的组件。例如，Address (English) 智能标记组件包含在与地址相关的项目中识别街道、市/县、省/市/自治区和邮政编码的识别器。当用户通过代码或使用“自动更正”对话框中的“智能标记”选项卡激活已安装的组件时，Microsoft Word 会标记文档中与这些项匹配的文本。

属性

  名称 说明
Application 返回一个代表 Microsoft Word 应用程序的 Application 对象。
Caption 返回或设置一个 String 类型的值，该值代表安装的智能标记识别器显示的文字。可读写。
Creator 返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Enabled 如果为应用程序启用了智能标记识别器，则该属性值为 True。
FullName 指定智能标记的名称，其中包含驱动器或 Web 路径。String 类型，只读。
Parent 返回一个 Object 类型值，该值代表指定 SmartTagRecognizer 对象的父对象。
ProgID 返回指定 OLE 对象的程序标识符 (ProgID)。只读 String 类型。
