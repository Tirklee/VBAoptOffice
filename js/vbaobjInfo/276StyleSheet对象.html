StyleSheet 对象成员

代表附加到 Web 文档的单个级联样式表。StyleSheet 对象是 StyleSheets 集合的成员。StyleSheets 集合包含附加到指定文档的所有级联样式表。

方法

 名称说明
Delete删除指定的级联样式表。
Move移动样式表的优先顺序。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
FullName指定级联样式表的名称，其中包含驱动器或 Web 路径。String 类型，只读。
Index返回 Long 值，该值表示项目在集合中的位置。只读。
Name返回指定对象的名称。只读 String 类型。
Parent返回一个 Object 类型的值，该值代表指定 StyleSheet 对象的父对象。
Path返回指定样式表的磁盘或 Web 路径。String 类型，只读。
Title返回或设置一个 String 类型的值，该值代表 Web 样式表的标题。可读写。
Type返回或设置样式表的类型。可读/写 WdStyleSheetLinkType 类型。
