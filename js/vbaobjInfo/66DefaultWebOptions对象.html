DefaultWebOptions 对象成员

包含 Microsoft Word 在您打开网页或将文档另存为网页时使用的应用程序级全局属性。

属性

 名称说明
AllowPNG 如果不允许将 PNG（可移植网络图形）作为输出格式使用，则该属性值为 False。Boolean 类型，可读写。
AlwaysSaveInDefaultEncoding 如果在保存网页或纯文本文档时使用默认编码，而不管打开该文件时的原始编码如何，则该属性值为 True。Boolean 类型，可读写。
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
BrowserLevel返回或设置一个 WdBrowserLevel 常量，该常量代表在 Microsoft Word 中创建的新网页要指向的 Web 浏览器级别。可读写。
CheckIfOfficeIsHTMLEditor 如果在启动 Word 时，Microsoft Word 检查某个 Office 应用程序是否为默认 HTML 编辑器，则该属性值为 True。Boolean 类型，可读写。
CheckIfWordIsDefaultHTMLEditor 如果在启动 Word 时，Microsoft Word 检查自己是否为默认 HTML 编辑器，则该属性值为 True。Boolean 类型，可读写。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Encoding返回或设置在查看保存的文档时 Web 浏览器使用的文档编码（代码页或字符集）。MsoEncoding 类型，可读写。
FolderSuffix返回一个 String 类型的值，该值代表 Microsoft Word 在将文档另存为网页、使用长文件名或将支持文件保存在单独的文件夹中时使用的文件夹后缀。只读。
Fonts返回一个 WebPageFonts 集合，该集合代表在 Word 中打开网页时 Microsoft Word 所使用的字体集。
OptimizeForBrowser 如果 Microsoft Word 为 BrowserLevel 属性指定的 Web 浏览器优化在 Word 中创建的新网页，则该属性为 True。可读/写 Boolean 类型。
OrganizeInFolder 如果在将指定文档另存为网页时，所有的支持文件（如背景纹理和图形）组织在一个单独的文件夹中，则该属性值为 True；如果支持文件和网页保存在同一文件夹中，则该属性值为 False。默认值为 True。可读写 Boolean 类型。
Parent返回一个 Object 类型的值，该值代表指定 DefaultWebOptions 对象的父对象。
PixelsPerInch返回或设置网页中图形图像及表格单元格的密度（每英寸的像素数）。可读写 Long 类型。
RelyOnCSS 如果在 Web 浏览器中查看保存的文档时，对字体格式使用了级联样式表 (CSS)，则该属性为 True。可读/写 Boolean 类型。
RelyOnVML 如果在将文档另存为网页时没有从图形对象中生成图像文件，则该属性值为 True。如果生成了图像，则该属性值为 False。默认值为 False。可读写 Boolean 类型。
SaveNewWebPagesAsWebArchives 如果为 True，则 Microsoft Word 使用“单个文件网页”（以前称为“Web 存档”）格式保存新的网页。可读/写 Boolean 类型。
ScreenSize返回或设置在 Web 浏览器中查看保存的文档时应使用的理想最小屏幕尺寸（宽 x 高，以像素为单位）。可读写 MsoScreenSize 类型。
TargetBrowser设置或返回一个 MsoTargetBrowser 常量，该常量代表在 Web 浏览器中查看文档的目标浏览器。可读写。
UpdateLinksOnSave 如果在将文档另存为网页时，所有支持文件的超链接和路径都会自动更新，则该属性值为 True。Boolean 类型，可读写。
UseLongFileNames 如果将文档另存为网页时使用长文件名，则该属性值为 True。如果不使用长文件名，而使用 DOS 文件名格式 (8.3)，则该属性值为 False。默认值为 True。可读写 Boolean 类型。
