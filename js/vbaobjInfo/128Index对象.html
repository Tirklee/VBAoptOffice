Index 对象成员

代表单个索引。Index 对象是 Indexes 集合的一个成员。Indexes 集合包括指定文档中的所有索引。

方法

 名称说明
Delete删除指定的索引。
Update更新显示在指定索引中的项目。

属性

 名称说明
AccentedLetters 如果指定索引包含用于重音字母的单独的标题，则该属性值为 True（例如，以“à”开头的词位于一个标题下，而以“A”开头的词位于另一个标题下）。Boolean 类型，可读写。
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Filter返回或设置一个值，该值指定 Microsoft Word 对指定索引项的第一个字符进行分类的方式。Long 类型，可读写。可以是下列 wdIndexFilter 常量之一。
HeadingSeparator返回或设置索引中字母组（以相同字母开头的条目）之间的文字。与 INDEX 域中的 \h 开关相对应。WdHeadingSeparator 类型，可读写。
IndexLanguage返回或设置一个 WdLanguageID 常量，该常量代表用于指定索引的排序语言。可读写。
NumberOfColumns返回或设置索引中每页的列数。Long 类型，可读写。
Parent返回一个 Object 类型的值，该值代表指定 Index 对象的父对象。
Range返回一个 Range 对象，该对象代表索引中包含的文档部分。
RightAlignPageNumbers 如果页码与索引中的右边距对齐，则该属性值为 True。可读写 Boolean 类型。
SortBy返回或设置指定索引的排序条件。WdIndexSortBy 类型，可读写。
TabLeader返回或设置索引中各项与它们的相关联页码之间的前导符。可读写 WdTabLeader 类型。
Type返回或设置索引的类型。可读/写 WdIndexType 类型。
