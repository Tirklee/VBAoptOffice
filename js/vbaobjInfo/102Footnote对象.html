Footnote 对象成员

代表位于页面底端或文字下方的脚注。Footnote 对象是 Footnotes 集合的一个成员。Footnotes 集合代表所选内容、范围或文档中的脚注。

方法

 名称说明
Delete删除指定的脚注。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Index返回 Long 值，该值表示项目在集合中的位置。只读。
Parent返回一个 Object 类型的值，该值代表指定 Footnote 对象的父对象。
Range返回一个 Range 对象，该对象代表脚注中包含的文档部分。
Reference返回一个 Range 对象，该对象代表脚注引用标记。
