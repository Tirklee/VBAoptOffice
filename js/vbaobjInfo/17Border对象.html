Border 对象成员

代表一个对象的一个边框。Border 对象是 Borders 集合的成员。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
ArtStyle返回或设置文档的艺术型页面边框。WdPageBorderArt 类型，可读写。
ArtWidth返回或设置指定艺术型边框的宽度（以磅为单位）。Long 类型，可读写。
Color返回或设置指定 Border 对象的 24 位颜色。
ColorIndex返回或设置指定边框或字体对象的颜色。WdColorIndex 类型，可读写。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Inside 如果指定对象可以使用内部边框，则该属性值为 True。Boolean 类型，只读。
LineStyle返回或设置指定对象的边框样式。WdLineStyle 类型，可读写。
LineWidth返回或设置对象边框的线条宽度。可读写。
Parent返回一个 Object 类型的值，该值代表指定 Border 对象的父对象。
Visible 如果指定的对象可见，则该属性值为 True。Boolean 类型，可读写。
