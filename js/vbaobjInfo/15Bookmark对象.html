Bookmark 对象成员
全部显示
全部隐藏

代表文档、选定内容或区域中的单个书签。Bookmark 对象是 Bookmarks 集合的成员。Bookmarks 集合包括“书签”对话框中（“插入”菜单）列出的所有书签。

方法

 名称说明
Copy将书签复制到 Name 参数中指定的新书签处，并返回一个 Bookmark 对象。
Delete删除指定的书签。
Select选择指定书签。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Column 如果指定的书签是表格的一列，则该属性值为 True。Boolean 类型，只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Empty 如果指定书签为空，则该属性值为 True。只读 Boolean 类型。
End返回或设置选定内容、区域或书签中结束字符的位置。Long 类型，可读写。
Name返回指定对象的名称。只读 String 类型。
Parent返回一个 Object 类型的值，该值代表指定 Bookmark 对象的父对象。
Range返回一个 Range 对象，该对象代表指定对象中包含的文档部分。
Start返回或设置书签起始字符的位置。Long 类型，可读写。
StoryType返回指定范围、选定内容或书签的文章 （文章：单个文本框或一串相链接文本框中包含的文字。）类型。WdStoryType 类型，只读。
