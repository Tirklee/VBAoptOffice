CustomLabel 对象成员

代表一个自定义的邮件标签。CustomLabel 对象是 CustomLabels 集合的成员。CustomLabels 集合中包括“标签选项”对话框中列出的所有自定义邮件标签。

方法

 名称说明
Delete删除指定自定义标签。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
DotMatrix 如果为 True，则指定自定义标签的打印机类型为点阵打印机。如果为 False，则打印机类型为激光或喷墨打印机。Boolean 类型，只读。
Height返回或设置指定的自定义邮件标签的高度，以磅为单位。Single 类型，可读写。
HorizontalPitch返回或设置两个自定义邮件标签左边距之间的水平距离（以磅为单位）。Single 类型，可读写。
Index返回 Long 值，该值表示项目在集合中的位置。只读。
Name返回或设置指定对象的名称。CustomLabel 类型，可读写。
NumberAcross返回或设置一页中自定义邮件标签的数目。Long 类型，可读写。
NumberDown返回或设置在页面长度范围内自定义标签的行数。Long 类型，可读写。
PageSize返回或设置指定的自定义邮件标签的页面大小。WdCustomLabelPageSize 类型，可读写。
Parent返回一个 Object 类型的值，该值代表指定 CustomLabel 对象的父对象。
SideMargin返回或设置指定自定义邮件标签的边距宽度（以磅为单位）。Single 类型，可读写。
TopMargin返回或设置页面的上边缘与正文文本的上边界之间的距离（以磅为单位）。可读写 Single 类型。
Valid 如果指定的自定义标签的各种属性（例如 Height、Width 和 NumberDown）共同构成有效的邮件标签，则该属性值为 True。Boolean 类型，只读。
VerticalPitch该属性返回或设置上下相邻的两个邮件标签的上边缘之间的垂直距离。Single 类型，可读写。
Width返回或设置自定义邮件标签的宽度，以磅为单位。可读/写 Long 类型。
