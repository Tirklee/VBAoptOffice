Word 开发人员参考
Global 对象成员

包含不需要以 Application 属性为前导的顶级属性和方法。

方法

 名称说明
BuildKeyCode为指定的组合键返回唯一的代码。
CentimetersToPoints将度量单位由厘米转换为磅数（1 厘米=28.35 磅）。所返回的转换结果为 Single 类型。
ChangeFileOpenDirectory设置查找文档时 Word 所指向的文件夹。
CheckSpelling检查字符串的拼写错误。返回一个 Boolean 类型的值，以显示字符串是否包含拼写错误。如果字符串未包含拼写错误，则该值为 True。
CleanString从指定字符串中删除非打印字符（字符代码为 1 – 29）及 Word 的特殊字符，或将它们替换为空格（字符代码为 32）。这些在“说明”部分有详细描述。以 String 类型返回结果。
DDEExecute通过指定的 DDE（即动态数据交换）通道，向应用程序发送一条或一组命令。
DDEInitiate打开通向其他应用程序的 DDE（动态数据交换）通道，并返回通道序号。
DDEPoke通过打开的 DDE（动态数据交换）通道向应用程序发送数据。
DDERequest通过打开的 DDE（动态数据交换）通道向接收应用程序查询信息，并以 String 类型返回该信息。
DDETerminate关闭指定的通向另一应用程序的 DDE（动态数据交换）通道。
DDETerminateAll关闭所有由 Microsoft Word 打开的动态数据交换 (DDE) 通道。
GetSpellingSuggestions返回一个 SpellingSuggestions 集合，该集合代表指定单词的建议拼写替换单词。
Help此方法显示联机帮助信息。
InchesToPoints将度量单位从英寸转换为磅（1 英寸 = 72 磅）。将转换后的度量单位以 Single 类型返回。
KeyString返回指定键的组合键字符串（例如，Ctrl+Shift+A）。
LinesToPoints将度量单位从行转换为磅（1 行 = 12 磅）。将转换后的度量单位以 Single 类型返回。
MillimetersToPoints将度量单位从毫米转换为磅（1 毫米 = 2.85 磅）。将转换后的度量单位以 Single 类型返回。
NewWindow打开一个新窗口，该窗口与指定窗口显示同一个文档。返回一个 Window 对象。
PicasToPoints将度量单位从十二点活字转换为磅（1 十二点活字 = 12 磅）。将转换后的度量单位以 Single 类型返回。
PixelsToPoints将度量单位从像素转换为磅。将转换后的度量单位以 Single 类型返回。
PointsToCentimeters将度量单位从磅转换为厘米（1 厘米 = 28.35 磅）。将转换后的度量单位以 Single 类型返回。
PointsToInches将磅转换为英寸（1 英寸 = 72 磅）。以 Single 类型返回转换结果。
PointsToLines将度量单位从磅转换为行（1 行 = 12 磅）。将转换后的度量单位以 Single 类型返回。
PointsToMillimeters将度量单位从磅转换为毫米（1 毫米 = 2.835 磅）。将转换后的度量单位以 Single 类型返回。
PointsToPicas将度量单位从磅转换为十二点活字（1 十二点活字 = 12 磅）。将转换后的度量单位以 Single 类型返回。
PointsToPixels将度量单位从磅转换为像素。将转换后的度量单位以 Single 类型返回。
Repeat重复一次或多次最近的编辑操作。如果成功地重复了命令，则返回 True。

属性

 名称说明
ActiveDocument返回一个 Document 对象，该对象代表活动文档（焦点所在的文档）。只读。
ActivePrinter返回或设置活动打印机名称。可读/写 String 类型。
ActiveProtectedViewWindow返回一个 ProtectedViewWindow 对象，该对象代表活动的受保护的视图窗口（焦点所在的受保护的视图窗口）。只读。
ActiveWindow返回一个 Window 对象，该对象代表活动窗口（焦点所在的窗口）。只读。
AddIns返回一个 AddIns 集合，该集合代表所有可用的加载项，而不考虑当前是否已加载它们。只读。
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
AutoCaptions返回一个 AutoCaptions 集合，该集合代表在文档中插入项目（如表格和图片）时自动添加的标题。只读。
AutoCorrect返回一个 AutoCorrect 对象，该对象包含当前“自动更正”的选项、词条和例外项。只读。
AutoCorrectEmail返回一个 AutoCorrect 对象，该对象代表对电子邮件进行的自动更正。
CaptionLabels返回一个 CaptionLabels 集合，该集合代表所有可用的题注标签。只读。
CommandBars返回一个 CommandBars 集合，该集合代表 Microsoft Word 中的菜单栏以及所有工具栏。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
CustomDictionaries返回一个 Dictionaries 对象，该对象代表活动自定义词典的集合。只读。
CustomizationContext返回或设置一个 Template 或 Document 对象，该对象代表存储菜单栏、工具栏和键绑定的更改的模板或文档。可读写。
Dialogs返回一个 Dialogs 集合，该集合代表 Word 中的所有内置对话框。只读。
Documents返回一个 Documents 集合，该集合代表所有打开的文档。只读。
FileConverters返回一个 FileConverters 集合，该集合代表 Microsoft Word 可以使用的所有文件转换器。只读。
FindKey返回一个 KeyBinding 对象，该对象代表指定的组合键。只读。
FontNames返回一个 FontNames 对象，该对象包含所有可用字体的名称。只读。
HangulHanjaDictionaries返回一个 HangulHanjaConversionDictionaries 集合，该集合代表所有活动的自定义转换词典。
IsObjectValid 如果引用某个对象的指定变量有效，则该属性值为 True。只读 Boolean 类型。
IsSandboxed如果应用程序窗口是“受保护的视图”窗口，则该属性值为 True。只读。
KeyBindings返回一个 KeyBindings 集合，该集合代表自定义的按键指定方案，包含了键代码、键类别和命令。
KeysBoundTo返回一个 KeysBoundTo 对象，该对象代表分配给指定项的所有组合键。
LandscapeFontNames返回一个 FontNames 对象，该对象包含所有可用的横向字体的名称。
Languages返回一个 Languages 集合，该集合代表在“语言”对话框中列出的校对语言。
LanguageSettings返回一个 LanguageSettings 对象，包含 Microsoft Word 中语言设置的信息。
ListGalleries返回一个 ListGalleries 集合，该集合代表三个列表模板库（“项目符号”、“编号”和“多级符号”）。
MacroContainer返回 Document 或 Template 对象，该对象代表模板或文档，在该模板或文档中存储的模块包含正在运行的过程。
Name返回指定对象的名称。String 类型，只读。
NormalTemplate返回一个 Template 对象，该对象代表 Normal 模板。
Options返回一个 Options 对象，该对象代表 Microsoft Word 中的应用程序设置。
Parent返回一个 Object 类型的值，该值代表指定 Global 对象的父对象。
PortraitFontNames返回一个 FontNames 对象，该对象包括所有可用纵向字体的名称。
PrintPreview 如果当前视图是打印预览，则该属性值为 True。可读写 Boolean 类型。
ProtectedViewWindows返回一个 ProtectedViewWindows 对象，该对象代表打开的受保护的视图窗口。只读。
RecentFiles返回一个 RecentFiles 集合，该集合代表最近访问过的文件。
Selection返回一个 Selection 对象，该对象代表一个选定的范围或插入点。只读。
ShowVisualBasicEditor 如果显示“Visual Basic 编辑器”窗口，则该属性值为 True。可读写 Boolean 类型。
StatusBarMicrosoft Word Visual Basic for Applications 不再支持此属性。
SynonymInfo返回一个 SynonymInfo 对象，该对象包含同义词库中有关指定单词或短语的同义词、反义词或相关单词和表达方式的信息。
System返回一个 System 对象，该对象可用于返回与系统相关的信息，并执行与系统相关的任务。
Tasks返回一个 Tasks 集合，该集合代表所有正在运行的应用程序。
Templates返回一个 Templates 集合，该集合代表所有可用的模板，包括共用模板和附加到打开文档的模板。
VBE返回一个 VBE 对象，该对象代表“Visual Basic 编辑器”。
Windows返回一个 Windows 集合，该集合代表所有打开的文档窗口。只读。
WordBasic返回一个自动化对象 (Word.Basic)，其中包含适用于 Word 6.0 和 Word for Windows 95 中所有 WordBasic 语句和函数的方法。只读。
