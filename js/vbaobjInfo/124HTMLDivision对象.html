HTMLDivision 对象成员

代表 Web 文档中的单个 HTML DIV 元素。HTMLDivision 对象是 HTMLDivisions 集合的一个成员。

方法

 名称说明
Delete删除指定的 HTML 划分。
HTMLDivisionParent返回一个 HTMLDivision 对象，该对象代表当前 HTML 划分的父划分。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Borders返回一个 Borders 集合，该集合代表指定对象的所有边框。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
HTMLDivisions返回一个 HTMLDivisions 对象，该对象代表 Web 文档中的 HTML 划分。
LeftIndent返回或设置一个 Single 类型的值，代表指定的 HTML 划分的左缩进值（以磅为单位）。可读写。
Parent返回一个 Object 类型的值，该值代表指定 HTMLDivision 对象的父对象。
Range返回一个 Range 对象，该对象代表指定对象中包含的文档部分。
RightIndent返回或设置指定段落的右缩进量（以磅为单位）。可读写 Single 类型。
SpaceAfter返回或设置指定 HTML DIV 元素后面的间距（以磅为单位）。可读/写 Single 类型。
SpaceBefore返回或设置指定 HTML DIV 元素前面的间距（以磅为单位）。可读/写 Single 类型。
