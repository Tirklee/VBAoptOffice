Windows 对象成员

Window 对象的集合，这些对象代表所有可用的窗口。Application 对象的 Windows 集合包含应用程序中的所有窗口，而 Document 对象的 Windows 集合只包含显示指定文档的窗口。

方法

 名称说明
Add返回一个 Window 对象，该对象代表文档的新窗口。
Arrange在应用程序工作区排列所有打开的文档窗口。
BreakSideBySide如果两个窗口以并排方式排列，则终止该模式。返回一个代表该方法是否成功的 Boolean 类型的值。
CompareSideBySideWith打开两个窗口并以并排方式排列。返回一个 Boolean 类型的值。
Item返回集合中的单个 Window 对象。
ResetPositionsSideBySide重置“并排比较”查看模式中的两个文档窗口。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Count返回一个 Long 类型的值，该值代表集合中窗口的数量。只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Parent返回一个 Object 类型的值，该值代表指定 Windows 对象的父对象。
SyncScrollingSideBySide如果为 True，则启用同时滚动窗口内容。Boolean 类型，可读写。
