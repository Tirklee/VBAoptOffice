TextColumns 对象成员

TextColumn 对象的集合，这些对象代表文档或文档的节中的所有文本栏。

方法

 名称说明
Add返回一个 TextColumn 对象，该对象代表添加到节中或文档中的新文本栏。
Item返回集合中的单个 TextColumn 对象。
SetCount将文字编排为指定数量的文本栏。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Count返回一个 Long 类型的值，该值代表集合中文本列的数量。只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
EvenlySpaced 如果文本栏是等宽的，则该属性值为 True。Long 类型，可读写。
FlowDirection返回或设置从一个文本栏到下一个文本栏的文字排列方向。WdFlowDirection 类型，可读写。
LineBetween 如果 TextColumns 集合中的所有栏之间显示垂直分隔线，则该属性值为 True。Long 类型，可读写。
Parent返回一个 Object 类型的值，该值代表指定 TextColumns 对象的父对象。
Spacing返回或设置列间距（以磅为单位）。Single 类型，可读写。
Width返回或设置艺术字文字效果的宽度（以磅为单位）。Long 类型，可读写。
