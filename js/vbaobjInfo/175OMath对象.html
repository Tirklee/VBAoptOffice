OMath 对象成员

代表一个公式。OMath 对象是 OMaths 集合的成员。

方法

 名称说明
BuildUp 将公式转换为专业格式。
ConvertToLiteralText将公式转换为文字文本。
ConvertToMathText 将公式转换为数学文本。
ConvertToNormalText 将公式转换为普通文本。
Linearize 将公式转换为线性格式。
Remove从文档、区域或所选内容中的公式集合内删除一个公式。

属性

 名称说明
AlignPoint返回或设置一个 Long 类型的值，该值代表公式中对齐点的字符位置。可读写。
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
ArgIndex返回一个 Long 类型的值，该值代表此组件相对于包含数学对象的参数索引。只读。
ArgSize返回或设置一个 Long 类型的值，该值代表参数（如文本、脚本或脚本-脚本）的脚本大小。可读写。
Breaks返回一个 OMathBreaks 集合，该集合代表公式中的换行。只读。
Creator返回一个 32 位的整数，该整数表示在其中创建加载项的应用程序。只读 Long 类型。
Functions返回一个 OMathFunctions 集合，该集合代表公式中包含的函数。只读。
Justification返回或设置一个 WdOMathJc，它代表公式的对齐方式。可读写。
NestingLevel返回一个 Long 类型的值，该值代表 OMath 对象的嵌套层。只读。
Parent返回一个 Object 类型的值，该值代表指定 OMath 对象的父对象。
ParentArg返回一个 OMath 对象，该对象代表父参数或包含参数。只读。
ParentCol返回一个 OMathMatCol 对象，该对象代表矩阵中的父列。只读。
ParentFunction返回一个 OMathFunction 对象，该对象代表父函数（即包含函数）。只读。
ParentOMath返回一个代表父元素的 OMath 对象。只读。
ParentRow返回一个 OMathMatRow 对象，该对象代表矩阵中的父行。只读。
Range返回一个 Range 对象，该对象代表指定对象中包含的文档部分。只读。
Type返回或设置一个 WdOMathType 常量，该常量表示是以内嵌方式显示公式（使文本围绕在公式的周围），还是在单独的行上显示公式。可读写。
