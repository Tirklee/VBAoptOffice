MailMergeDataSource 对象成员
全部显示
全部隐藏

代表邮件合并操作中的邮件合并数据源。

方法

 名称说明
Close关闭指定的邮件合并数据源。
FindRecord在特定域的文字中搜索指定的邮件合并数据源的内容。如果找到了搜索文字，则该方法返回 True。Boolean 类型。
SetAllErrorFlags将邮件合并数据源中的所有记录标记为地址域中包含无效数据。
SetAllIncludedFlags包括或排除邮件合并中某个数据源的标记记录。

属性

 名称说明
ActiveRecord返回或设置活动邮件合并记录。可以是查询结果中的有效记录号，也可以是下列 WdMailMergeActiveRecord 常量之一。
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
ConnectString返回指定邮件合并数据源的连接字符串。String 类型，只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
DataFields返回一个 MailMergeDataFields 集合，该集合代表指定邮件合并数据源中的域。只读。
FieldNames返回一个 MailMergeFieldNames 集合，该集合代表指定邮件合并数据源中所有域的名称。只读。
FirstRecord返回或设置邮件合并操作中合并的第一个记录的编号。Long 类型，可读写。
HeaderSourceName返回附加到指定的邮件合并主文档的域名源的路径和文件名。String 类型，只读。
HeaderSourceType返回一个值，该值表明为邮件合并操作提供域名源的方式。WdMailMergeDataSource 类型，只读。
Included 如果邮件合并中包含记录，则该属性值为 True。Boolean 类型，可读写。
InvalidAddress 当该属性值为 True 时，如果邮件合并数据源中记录的地址域中包含无效数据，则 Microsoft Word 标记该记录。Boolean 类型，可读写。
InvalidComments如果 InvalidAddress 属性设置为 True，则返回或设置一个 String 值，该值描述无效的地址错误。可读写。
LastRecord返回或设置邮件合并操作中要合并的最后一个记录的编号。Long 类型，可读写。
MappedDataFields返回一个 MappedDataFields 集合，该集合代表 Microsoft Word 中可用的映射数据字段 （映射数据字段：代表常用信息的字段，例如，“First Name”。如果数据源中包含“First Name”字段或其变体（例如，“FName”），该数据源字段自动映射到相应的映射数据字段。）。
Name返回指定对象的名称。String 类型，只读。
Parent返回一个 Object 类型的值，该值代表指定 MailMergeDataSource 对象的父对象。
QueryString返回或设置查询字符串（SQL 语句），该语句用于从邮件合并数据源中检索数据子集。String 类型，可读写。
RecordCount返回一个 Long 对象，该对象代表数据源中的记录数。只读。
TableName返回一个带有 SQL 查询的 String 类型的值，用于从邮件合并文档附加的数据源文件检索记录。
Type返回邮件合并数据源的类型。只读 WdMailMergeDataSource 类型。
