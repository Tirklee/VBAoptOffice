MailMergeField 对象成员

代表文档中的单个邮件合并域。MailMergeDataField 对象是 MailMergeDataFields 集合的一个成员。MailMergeDataFields 集合包含文档中的所有邮件合并相关域。

方法

 名称说明
Copy将指定的邮件合并域复制到剪贴板上。
Cut从文档中删除指定的邮件合并域，并将其移动到剪贴板上。
Delete删除指定的邮件合并域。
Select选择指定的邮件合并域。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Code返回一个代表域代码的 Range 对象。可读写。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Locked 如果指定域已被锁定，则该属性值为 True。可读写 Boolean 类型。
Next返回邮件合并域集合中的下一个邮件合并域。只读。
Parent返回一个 Object 类型的值，该值代表指定 MailMergeField 对象的父对象。
Previous返回邮件合并域集合中的上一个邮件合并域。只读。
Type返回域的类型。只读 WdFieldType 类型。
