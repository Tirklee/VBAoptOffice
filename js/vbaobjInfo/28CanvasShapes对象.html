CanvasShapes 对象成员

使用 Shape 或 ShapeRange 对象的 CanvasItems 属性返回 CanvasShapes 集合。

方法

 名称说明
AddCallout在绘图画布上添加无框线的标注。返回表示该标注的 Shape 对象。
AddConnector返回一个 Shape 对象，该对象代表绘图画布上两个形状间的连线。
AddCurve返回一个 Shape 对象，该对象代表绘图画布上的贝赛尔曲线。
AddLabel在绘图画布上添加一个文本标签。返回一个表示该文本标签的 Shapes 对象。
AddLine在绘图画布上添加一条直线。返回一个代表该直线的 Shape 对象，并将其添加到 CanvasShapes 集合。
AddPicture在绘图画布上添加一幅图片。返回一个 Shape 对象，该对象代表图片并将其添加至 CanvasShapes 集合。
AddPolyline在绘图画布上添加一个开放或封闭的多边形。返回一个代表该多边形的 Shape 对象。
AddShape在绘图画布上添加一个自选图形。返回一个代表该自选图形的 Shape 对象。
AddTextbox在绘图画布上添加一个文本框。返回一个表示该文本框的 Shape 对象。
AddTextEffect在绘图画布上添加一个艺术字形状。返回一个表示该艺术字的 Shape 对象。
BuildFreeform建立任意多边形对象。返回一个 FreeformBuilder 对象，该对象表示正在建立的任意多边形。
Item返回集合中的单个 Shape 对象。
Range返回一个 ShapeRange 对象。
SelectAll选择画布中的所有形状。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Count返回一个 Long 类型的值，该值代表指定集合中画布图形的数量。只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Parent返回一个 Object 类型的值，该值代表指定 CanvasShapes 对象的父对象。
