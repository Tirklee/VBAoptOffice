Section 对象成员

代表所选内容、范围或文档中的一节。Section 对象是 Sections 集合的一个成员。Sections 集合包括所选内容、范围或文档中的所有节。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Borders返回一个 Borders 集合，该集合代表节中的所有边框。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Footers返回一个 HeadersFooters 集合，该集合代表指定节的页脚。只读。
Headers返回一个 HeadersFooters 集合，该集合代表指定节的页眉。只读。
Index返回 Long 值，该值表示项目在集合中的位置。只读。
PageSetup返回一个 PageSetup 对象，该对象与指定的节相关联。
Parent返回一个 Object 类型的值，该值代表指定 Section 对象的父对象。
ProtectedForForms 如果保护指定节中的窗体，则该属性值为 True。Boolean 类型，可读写。
Range返回一个 Range 对象，该对象代表指定对象中包含的文档部分。
