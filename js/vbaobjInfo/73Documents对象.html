Documents 对象成员

当前在 Word 中打开的所有 Document 对象的集合。

方法

 名称说明
Add返回一个 Document 对象，该对象代表添加到打开文档集合中的新建空文档。
AddBlogDocument 返回一个 Document 对象，该对象表示 Microsoft Word 向前三个参数所描述的帐户发布的新博客文档。
CanCheckOut 如果该属性值为 True，则 Microsoft Word 可将指定的文档从服务器签出。可读/写 Boolean 类型。
CheckOut将指定的文档从服务器复制到本地计算机以进行编辑。
Close关闭指定的文档。
Item返回集合中的单个 Document 对象。
Open打开指定的文档并将其添加到 Documents 集合。返回一个 Document 对象。
OpenNoRepairDialog打开指定的文档并将其添加到 Documents 集合。
Save保存 Documents 集合中的所有文档。

属性

 名称说明
Application返回一个代表 Microsoft Word 应用程序的 Application 对象。
Count返回一个 Long 类型的值，该值代表集合中的文档数。只读。
Creator返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Parent返回一个 Object 类型的值，该值代表指定 Documents 对象的父对象。
