SmartTagTypes 对象成员

一个代表 SmartTagType 对象的集合。

方法

  名称 说明
Item 返回集合中的单个 SmartTagType 对象。
ReloadAll 重新加载安装在用户计算机上的所有智能标记操作和识别器。

属性

  名称 说明
Application 返回一个代表 Microsoft Word 应用程序的 Application 对象。
Count 返回一个 Long 类型的值，该值代表集合中智能标记类型的数量。只读。
Creator 返回表示用于创建指定对象的应用程序的 32 位整数。只读 Long 类型。
Parent 返回一个 Object 类型值，该值代表指定 SmartTagTypes 对象的父对象。
